사용자가 등록한 IP 주소를 기준으로 와탭 서비스 접속 허용 여부를 설정할 수 있습니다. 

* **IP를 허용으로 등록 시**, 등록하지 않은 모든 IP는 차단됩니다. 
* **IP를 비허용으로 등록 시**, 추가된 IP만 차단됩니다.

### IP 주소 등록/수정

1. 화면 오른쪽 위의 **프로필 아이콘**을 클릭한 후, **계정 관리**로 이동하세요. 

1. **접근 제어**에서 **IP 접근 제어** 목록을 확인합니다. 

1. **+ IP 추가** 버튼을 클릭해, 신규 IP를 정보(IP 주소, 설명, 접근 허용 여부)를 모두 입력하세요. 

    a. 등록된 IP 주소를 수정하려면, 해당 IP 주소의 ![수정 아이콘](/img/ic-edit.svg) 아이콘을 클릭해 정보를 수정하세요.

    :::caution

    **IP 허용 등록 시(화이트 리스트 방식) 유의 사항**
    
    허용 목록이 등록되면 화이트 리스트 방식이 적용되어, 등록된 IP 외에 모든 접근이 차단됩니다. 작업 중단을 방지하기 위해 현재 작업자의 IP를 먼저 허용 목록에 등록해 주세요.

    :::

1. **확인** 버튼을 클릭해 IP를 등록하세요. 


### IP 주소 삭제 

IP 주소 삭제 시 접속 권한이 변경됩니다. 삭제할 IP 주소의 ![삭제 아이콘](/img/ic-delete-critical.svg) 아이콘을 클릭하세요. 

* **허용 상태인 IP 삭제 시**: 기존에 접속이 차단되었던 IP에서 접속할 수 있게 됩니다.

* **비허용 상태인 IP 삭제 시**: 해당 IP에서 접속할 수 있게 됩니다.
