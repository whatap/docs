홈 화면 > 프로젝트 선택 > ***분석*** > ***로그 파일 뷰어*** 또는 ***로그 파일 뷰어(cloud)***

에이전트를 설치한 데이터베이스 서버 또는 클라우드 데이터베이스 서버의 로그 파일 내용을 확인할 수 있습니다. 한번에 하나의 데이터베이스 로그를 확인할 수 있습니다.

![로그 파일 뷰어](/img/analysis-log-viewer.png)

* ![지시선 1](/img/number-01.png) ***인스턴스***

  프로젝트에 포함된 인스턴스를 선택할 수 있습니다. 인스턴스 항목을 클릭해 개별 인스턴스를 선택하세요.

* ![지시선 2](/img/number-02.png) ***로그 파일 목록***

  각 인스턴스별로 확인할 수 있는 로그 파일 목록입니다. 파일이 많아 찾기 어렵다면 검색을 통해 원하는 파일을 찾을 수 있습니다.

* ![지시선 3](/img/number-03.png) ***로그 파일 뷰어*** 

  2000 바이트(byte) 단위로 로그 내용을 확인할 수 있습니다. ![이전 페이지 아이콘](/img/left-arrow.svg) 또는 ![이후 페이지 아이콘](/img/right-arrow.svg) 버튼을 선택해 2000 바이트(byte) 단위로 페이지를 이동할 수 있습니다. 처음과 끝으로 이동하려면 ![첫 페이지 아이콘](/img/ico-first.svg) 또는 ![마지막 페이지 아이콘](/img/ico-last.svg) 버튼을 선택하세요.

  사용자가 원하는 임의로 위치로 이동하려면 원하는 위치를 입력한 다음 ***이동*** 버튼을 선택하세요.
