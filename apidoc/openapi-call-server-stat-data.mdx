## 서버 통계 상세

<span class='apitype'>GET</span> https://service.whatap.io/open/json/stat/sm_stat?<code>stitme</code>&<code>etime</code>&<code>filter</code><code>&oid</code>

```bash title='command'
$ curl -L -w "\n" -H "x-whatap-token: {{PROJECT API TOKEN}}" -H "x-whatap-pcode: {{projectCode}}" "https://service.whatap.io/open/json/stat/sm_stat?stime=시작UTC&etime=종료UTC&filter=m5&oid=0"
```

| 파라미터 | 설명 |
| -------- | ---- |
| stime | 시작 시간 UTC (UNIX epoch time, millisecond) |
| etime | 종료 시간 UTC (UNIX epoch time, millisecond) |
| filter | 기본 `m5` 자동 설정, 시간 범위 내에 5분 통계를 가지고 올때 `m5`를 사용합니다.<br/>5초 통계는 `s5`, 1시간 통계는 `h1`을 사용합니다. |
| oid | 특정 서버만 모니터링을 원할 경우 추가하세요. |

:::note

* 조회 시간(`stime`, `etiem`)은 한 시간 이내로 제한하며, ms(millisecond) 단위입니다.
* `oid` 외 다른 파라미터는 허용하지 않습니다.

:::
