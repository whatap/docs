* **watchlog_enabled**<span class='type'>Boolean</span><br/>
    기본값 `false`<br/>
    값이 `true`이면 로그 모니터링을 활성화합니다.

* **watchlog_check_interval**<span class='type'>Int</span><br/>
    기본값 `2000`<br/>
    로그를 감시하는 간격입니다. 매 간격으로 로그 파일에 기록을 추가했는지 확인합니다.

* **watchlog_read_count**<span class='type'>Int</span><br/>
    기본값 `8`<br/>
    로그를 읽는 최대 횟수입니다. 한 번에 `watchlog_buffer_size` 크기 만큼 읽습니다.

* **watchlog_buffer_size**<span class='type'>Int</span><br/>
    기본값 `131072(128k)`<br/>
    로그를 한 번에 읽는 로그 사이즈입니다.

* **watchlog.LOGKEY**<br/>
    하나의 로그 파일을 감시하기 위한 설정입니다. `file` 이름을 입력하고 감시한 `word` 여러 개를 콤마(,)로 구분해 설정합니다. `check_interval` 간격으로 로그를 감시하다가 키워드를 발견하면 경고합니다. 한번 경고가 나가면 `silent` 만큼 경고를 멈춥니다.

    * **watchlog.LOGKEY.enabled**<span class='type'>Boolean</span> 기본값 `true`
    * **watchlog.LOGKEY.file**<span class='type'>String</span>
    * **watchlog.LOGKEY.words**<span class='type'>String</span>
    * **watchlog.LOGKEY.silent**<span class='type'>Int</span> 기본값 `10000`
    * **watchlog.LOGKEY.check_interval**<span class='type'>Int</span> 기본값 `1000`

* **log_root**<span class='type'>String</span><br/>
    기본값 `${WHATAP_HOME}`/logs<br/>
    에이전트 로그 경로를 설정합니다.

* **log_datasource_lookup_enabled**<span class='type'>Boolean</span><br/>
    기본값 `true`<br/>
    InitialContext Lookup 시 DataSource인 경우 로그를 기록하는 기능을 활성화합니다.

* **log_rotation_enabled**<span class='type'>Boolean</span><br/>
    기본값 `true`<br/>
    에이전트 로그 파일을 날짜 별로 저장하는 기능을 활성화합니다. 로그 파일명은 *whatap-`yyyyMMdd`.log* 형태로 저장합니다. 값이 `false`이면 로그 파일명은 *whatap.log* 형태로 저장합니다.

* **log_keep_days**<span class='type'>Int</span><br/>
    기본값 `7`<br/>
    로그 파일 보관 기간을 설정합니다. `log_rotation_enabled` 값이 `true`인 경우에만 동작합니다.

* **log_oname_enabled**<span class='type'>Boolean</span><br/>
    기본값 `false`<br/>
    값이 true이면 각 로그마다 에이전트 이름을 추가합니다. [예시](/img/java-agent-log-oname.png)
