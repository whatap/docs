* **trace_user_enabled(*)**<span class='type'>Boolean</span><br/>
    기본값 `true`<br/>
    실시간 사용자 추적 여부를 결정합니다. 이 값을 활성화하는 경우 기본값은 IP가 됩니다.

* **trace_user_using_ip(*)**<span class='type'>Boolean</span><br/>
    기본값 `true`<br/>
    실시간 사용자의 구분을 IP로 하고자 하는 경우 설정합니다. 이 데이터는 Real Time User에서 확인 가능합니다. `user_header_ticket` 옵션과 배타적으로 설정을 동시에 적용할 수 없습니다.
