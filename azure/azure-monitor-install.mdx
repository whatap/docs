## 모니터링 절차

1. 사용자 와탭 콘솔에서 인티그레이션 프로젝트를 Azure Monitor 사용 상태로 생성하세요.
1. Azure Registered App을 추가하세요.
1. Azure 인증 정보를 입력하세요.
1. Azure Monitor Namespace 및 수집 주기를 선택하세요.
1. ResourceGroup, Metric, Stats 등을 선택하세요.

:::note
설치 후 사용량에 따라 와탭 요금과는 별개로 클라우드에서 추가 비용이 발생할 수 있습니다. 설치 전 사용 중인 클라우드 서비스의 가격 정책을 [다음 문서](https://azure.microsoft.com/ko-kr/pricing/details/monitor/)에서 확인하세요.
:::

## Azure Registered App 추가

와탭 모니터링 용 [Registered App](https://portal.azure.com/#view/Microsoft_AAD_IAM/ActiveDirectoryMenuBlade/~/RegisteredApps)을 추가할 수 있습니다. Secret을 생성하고 Application ID, Tenant Id 와 Secret을 와탭에 입력합니다.

![Azure Active Directory](https://img.whatap.io/media/images/registeredapps.png)

:::note
와탭 모니터링을 위해서는 Monitor Reader, 독자, API Management 서비스 독자 권한이 필요합니다. Azure Portal에서 Subscript > 액세스 제어(IAM) 메뉴에서 추가 버튼을 통해 권한을 추가할 수 있습니다.
:::

## 인증 정보 입력

Azure 인증 정보인 Application ID, Tenant ID와 Secret을 와탭에 입력합니다.

![Azure Monitor - Authentification](https://img.whatap.io/media/images/whatap_azure_auth_dialog.png)
