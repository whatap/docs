대시보드 메뉴에 배치한 위젯을 사용자가 원하는 옵션으로 설정할 수 있습니다.

:::note

웹 브라우저의 화면 크기에 따라 위젯 오른쪽 위에 아이콘의 모양이 다를 수 있습니다. ![더보기 아이콘](/img/ico-more.svg) 버튼을 선택하면 추가 아이콘을 확인할 수 있습니다.

:::


## 그래프 차트 기준 데이터 변경하기

<ImgLang img='widget-sample.png' desc='위젯' />

위젯의 오른쪽 위에 ![지시선 1](/img/number-01.png) 메트릭스 버튼을 선택하세요. ***메트릭스*** 팝업창이 나타납니다. 원하는 항목을 선택해 그래프 차트의 데이터를 변경할 수 있습니다.

:::note

* ***메트릭스*** 팝업창에서 선택할 수 있는 항목은 위젯에 따라 다를 수 있습니다.
* 위젯의 데이터에 따라 메트릭스 버튼을 지원하지 않을 수 있습니다.

:::

## 전체 화면 보기

전체 화면으로 보고 싶은 위젯의 오른쪽 위에 ![상세 보기 아이콘](/img/ico-detail-mode.svg) 버튼을 선택하세요. 웹 브라우저 상에서 위젯의 그래프 차트를 전체 화면 크기로 확인할 수 있습니다.

<ImgLang img='widget-fullscreen-mode.png' desc='위젯 전체 화면' />

화면 오른쪽 위에 ![숫자 1](/img/number-01.png) 옵션 버튼들을 선택해 원하는 데이터를 다양한 방식으로 조회할 수 있습니다.

:::note

위젯의 데이터에 따라 선택할 수 있는 옵션이 다를 수 있습니다.

:::

### 데이터 조회 시간 변경하기

현재 진행 중인 데이터의 흐름을 실시간으로 확인할 수 있고, 지나간 시간의 데이터를 확인할 수도 있습니다. ![시간 선택 아이콘](/img/ico-time-set.svg)을 선택하세요. ***시간 선택*** 팝업창이 나타납니다.

<ImgLang img='widget-select-time.png' desc='시간 선택' />

* 그래프 차트의 데이터 조회 시간을 변경하려면 ***대시보드 설정을 따름***의 체크박스를 해제하세요.
* 그래프 차트를 실시간으로 모니터링하려면 ![실시간 아이콘](/img/ico-play-time.svg) 버튼을 선택하세요.
* 조회 시간은 ![일시 정지 아이콘](/img/ico-pause.svg) 버튼을 선택한 다음 변경할 수 있습니다.
* ![왼쪽 방향 아이콘](/img/left-arrow.svg) 또는 ![오른쪽 방향 아이콘](/img/right-arrow.svg) 버튼을 선택해 시간을 이동할 수 있습니다. 이동 간격은 오른쪽에 설정된 ![숫자 2](/img/number-02.png) 시간 간격입니다.
* ![숫자 2](/img/number-02.png) 시간 간격 버튼을 클릭해 시간 간격을 선택할 수 있습니다.
* 원하는 날짜와 시간을 직접 선택하려면 날짜 및 시간 영역을 클릭하세요.

### 이전 시간과 비교하기

지난간 시간의 데이터를 현재 시간과 비교해 볼 수 있습니다. ![이전 시간과 비교 아이콘](/img/ico-timing.svg) 버튼을 선택하세요. ***이전 시간과 비교*** 팝업창이 나타납니다.

<ImgLang img='widget-compare-time.png' desc='이전 시간과 비교' />

초, 분, 시간, 일 단위로 시간을 비교할 수 있고, 원하는 날짜를 선택해 비교할 수도 있습니다. 원하는 옵션을 선택한 다음 ***적용*** 버튼을 클릭해야 그래프 차트에 적용할 수 있습니다. 이전 시간 데이터와 비교를 취소하려면 ***미설정***을 클릭한 다음 적용 버튼을 선택하세요.

### 기준 데이터 변경하기{#change-data}

<ImgLang img='widget-metric.png' desc='메트릭스' />

1. 오른쪽 위에 위치한 옵션 중 위젯의 지표에 해당하는 텍스트를 선택하세요. (보통 세번째에 위치합니다.)

1. 메트릭스 팝업창이 나타납니다.

1. 원하는 항목을 선택하세요.

선택한 항목으로 그래프 차트의 데이터를 변경합니다.

### 상위 N 데이터 확인하기

* ***상위 10*** 버튼을 클릭하면 최대 상위 10 항목까지 선택할 수 있습니다. 선택한 수치만큼 높은 순으로 그래프 차트에 데이터를 적용할 수 있습니다.

  <ImgLang img='widget-n-data.png' desc='상위 N' />

* ![상위 N 아이콘](/img/ico-topn-graph.svg) 버튼을 선택해 상위 N 데이터 항목을 그래프 차트에 오버레이 형식으로 표시할 수 있습니다. 그래프 차트에서 상위 N 데이터 항목을 사라지게 하려면 ![그래프 아이콘](/img/ico-normal-graph.svg) 버튼을 선택하세요.

  <ImgLang img='widget-topn-chart.png' desc='Top N' />

### 전체 데이터 확인하기

<ImgLang img='widget-all-data.png' desc='All' />

전체 데이터를 비교하며 함께 확인하려면 ![전체 아이콘](/img/ic-area-chart.svg) 버튼을 선택하세요. 차트 위로 마우스를 오버하면 툴팁 형태로 각 데이터별 수치를 확인할 수 있습니다.

### MXQL로 데이터 조회하기

MXQL을 이용해 데이터를 조회하려면 ![메트릭스 편집 아이콘](/img/ico-edit-metric.svg) 버튼을 선택하세요. MXQL 편집창이 나타납니다.

:::note

MXQL에 대한 자세한 내용은 [다음 문서](../mxql/mxql-overview)를 참조하세요.

:::

### 테이블 컬럼 변경하기

데이터 테이블 위젯의 경우 전체 화면에서 컬럼을 변경할 수 있습니다. 오른쪽 위에 ![컬럼 아이콘](/img/ico-column.svg) 버튼을 선택하세요. ***컬럼 편집*** 창이 나타나면 테이블에 표시할 컬럼을 숨기거나 추가할 수 있습니다. 설정을 완료한 다음 ***저장*** 버튼을 선택해야 데이터 테이블 차트에 적용할 수 있습니다.

<ImgLang img='widget-column.png' desc='컬럼 편집' />

## 차트 설정하기

y축의 최대값을 설정하고 수평선을 추가해 임계치를 설정하려면 위젯의 오른쪽 위에 ![설정 아이콘](/img/ico-setting.svg) 버튼을 선택하세요.

<ImgLang img='widget-chart-set.png' desc='차트 설정' />

* ***y축 차트 최대값***: ***자동*** 또는 ***고정***을 선택해 y축의 최대값을 설정할 수 있습니다. ***고정***을 선택한 경우 수치를 입력할 수 있습니다. 수치를 입력했다면 ***적용*** 버튼을 선택하세요.

* ***수평선 설정***: 그래프 차트에 수평선을 추가해 임계치를 설정하고 효율적으로 모니터링할 수 있습니다.

  * ***+ 추가*** 버튼을 선택한 다음 ***라벨***과 ***임계치*** 항목에 수치를 입력하고, 색상을 선택하세요. ***임계치*** 항목에서는 ***범위***를 선택해 수평선 구역을 설정할 수 있습니다. ***적용*** 버튼을 선택해 설정을 완료하세요.

  * 수평선을 삭제하려면 ![삭제 아이콘](/img/ico-trash-red.svg) 버튼을 선택하세요.

* ***차트 최대값 표시***: 그래프 차트에 최대값을 표시할 수 있습니다. 가장 높은 수치의 값을 차트에 텍스트로 표시합니다.

* ***차트 범례 표시***: 차트 위젯 아래에 각 그래프 데이터의 이름을 표시합니다. 그래프의 색상과 범례를 비교해 차트를 이해하는데 도움이 됩니다.
