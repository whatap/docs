<Cmdname sid="side_ipLoadStatistics" className="uitext" /> 메뉴는 웹 사이트에 접속한 각 IP 주소의 로드 성능 정보를 제공합니다. 이를 통해 성능 저하가 발생한 IP 주소를 확인하고, 개선이 필요한 IP 주소를 식별할 수 있습니다. 평균값, 최댓값, 최솟값, 백분위수 기준의 통계 정보를 제공하며, Navigation Timing 데이터를 활용하여 각 IP 주소별 페이지 로드 과정의 세부 시간 정보를 필터링하고 분석할 수 있습니다.

**IP 로드 시간**은 특정 IP 주소에서 웹 페이지의 모든 리소스(CSS, JS, 이미지, 폰트 등)를 로드하고, 사용자의 화면에 모든 콘텐츠가 완전히 렌더링되기까지의 총 시간을 의미합니다. 브라우저 에이전트는 이 데이터를 수집하기 위해 브라우저에서 제공하는 Web API 중 하나인 **Navigation Timing API**를 활용합니다.

:::note

<Cmdname sid="side_ipLoadStatistics" className="uitext" /> 메뉴에서 수집하는 데이터에 대한 자세한 내용은 [다음 문서](collect-data#pageloadtime)를 참조하세요.

:::

## 기본 사용 안내

특정 IP 주소에서 접속한 각 페이지의 성능 지표를 제공합니다. IP 주소별로 **IP 로드 시간** 및 **DOM 로드 시간**, **네트워크 지연 시간**, **리소스 로드 시간** 등 다양한 성능 지표를 확인할 수 있습니다. 이를 통해 각 IP 주소의 성능 상태를 평가하고, 최적화 여부를 판단할 수 있습니다.

<ImgLang img="browser-ipload-statistics.png" desc="페이지 로드 통계" />

1. <Cmdname sid="time" className="uitext" />에서 성능 지표를 조회할 날짜와 시간을 설정하세요. 

1. <Cmdname sid="TTL08276" className="uitext" />에서 원하는 기준을 선택하세요.

1. 화면 오른쪽 상단의 ![검색 아이콘](/img/ic-search-on-press.svg) 버튼을 선택하세요.

조회한 시간 범위의 성능 지표를 각 섹션에서 그래프 차트와 목록을 통해서 확인할 수 있습니다. 각 섹션의 그래프 차트로 마우스를 오버하면 해당 시간대의 지표 정보를 확인할 수 있습니다.

:::note

### 통계 기준 선택하기 {#statcriteria}

<Cmdname sid="TTL08276" className="uitext" />은 사용자가 분석하려는 통계 지표를 다음 선택한 기준으로 확인할 수 있는 옵션입니다.

* <Cmdname sid="merge_avg" className="uitext" />: 각 IP 주소의 평균 로드 시간을 계산하여 통계를 제공합니다.

* <Cmdname sid="merge_max" className="uitext" />: 각 IP 주소의 최대 로드 시간을 기준으로 통계를 제공합니다. 즉, 페이지 로드 시간이 가장 비효율적인 IP 주소를 파악할 수 있습니다.

* <Cmdname sid="merge_min" className="uitext" />: 각 IP 주소의 최소 로드 시간을 기준으로 통계를 제공합니다. 이를 통해 가장 최적의 조건에서 페이지를 로드한 IP 주소를 확인할 수 있습니다. 또는 성능 최적화 목표를 설정할 때 유용합니다.

* <Cmdname sid="ITM08291" className="uitext" />: 각 IP 주소의 백분위수 기준 로드 시간에 대한 통계를 제공합니다.

  - <Cmdname sid="TTL08288" className="uitext" replace="95" />: 가장 느린 5%의 IP 로드 시간으로, IP 로드 시간이 가장 느렸던 상위 5% 사용자의 경험을 의미합니다.

  - <Cmdname sid="TTL08288" className="uitext" replace="90" />: 가장 느린 10%의 IP 로드 시간으로, 주로 사용자 그룹의 성능 문제를 파악할 때 유용합니다.

  - <Cmdname sid="TTL08288" className="uitext" replace="75" />: IP 로드 시간이 하위 75%보다 느린 상위 25%의 로드 시간을 보여줍니다.

  - <Cmdname sid="TTL08288" className="uitext" replace="50" />: 중간값으로, 전체 IP 주소 중 절반은 해당 시간보다 빠르게 페이지를 로드하고, 절반은 더 느리게 로드한 것을 의미합니다.
