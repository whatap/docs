## rum_page_load_each_page {#b798b7654dc22dee3a29d413a85dad14}

페이지 로드 시간에 관한 지표는 **Performance API** 중 **Navigation Timing**의 값을 참조합니다. 다음 그림과 같이 페이지 로드 탐색이 진행됩니다.

![](https://w3c.github.io/navigation-timing/timestamp-diagram.svg)

페이지 로드 응답 시간은 **loadEventEnd** 시점까지를 의미합니다.

* 수집 간격 : 5초

* 통계 데이터 : 5분, 1시간

| ---------- | ----------- | ---- |
| `page_group` | 페이지 그룹 | -    |

| `page_load_event_time` | 밀리 세컨드 | 브라우저가 측정하는 load 이벤트가 완료되는 평균 시간 | 시작부터 **loadEventEnd**까지 소요된 시간 |
