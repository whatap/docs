## 페이지 그룹 필터

페이지 그룹별 필터링 옵션을 적용할 수 있습니다. 대시보드 화면 왼쪽 위에 <Cmdname sid="page_group" className="uitext" /> 필터 버튼을 선택해 필터 옵션을 설정할 수 있습니다. 

* <Cmdname sid="equal" className="uitext" />: 선택한 페이지 그룹과 일치하는 데이터를 조회할 수 있습니다.

  <ImgLang img='browser-dashboard-filter.png' desc='Page group filter' />

- <Cmdname sid="contain" className="uitext" />: 특정 문자열을 포함하는 페이지 그룹의 데이터를 조회할 수 있습니다.

  <ImgLang img='browser-dashboard-filter-include.png' desc='Page group filter' />

- <Cmdname sid="exclude" className="uitext" />: 특정 문자열을 포함하는 페이지 그룹을 제외해 데이터를 조회할 수 있습니다.

  <ImgLang img='browser-dashboard-filter-exclude.png' desc='Page group filter' />

- <Cmdname sid="add_filter" className="uitext" />: 여러 개의 필터를 적용하려면 <Cmdname sid="add_filter" className="uitext" /> 버튼을 선택한 다음 원하는 조건을 입력하세요. **AND** 조건으로 필터를 추가할 수 있습니다.

  <ImgLang img='browser-dashboard-filter-and.png' desc='Page group filter' />
