메트릭스는 모니터링 대상의 성능 정보를 시계열 데이터 형태로 표현한 지표입니다. 프로젝트를 생성하고 아이템을 설치하면 메트릭스 데이터가 자동으로 수집되며, 수집된 데이터를 다양한 방식으로 분석할 수 있습니다.

:::note

메트릭스에 대한 자세한 내용은 [용어 알아보기 &gt; 메트릭스](/metrics/common-metrics-intro)를 참고하시기 바랍니다.

:::
