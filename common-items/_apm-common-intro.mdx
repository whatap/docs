### 주요 특장점

* **실시간 트랜잭션 모니터링**

  와탭은 현재 실행 중인 트랜잭션의 상태를 실시간으로 보여줍니다. 트랜잭션 수가 급증할 경우, 수행 중인 SQL, 외부 호출, 메서드를 즉시 파악해 문제를 빠르게 진단할 수 있습니다.

* **트랜잭션 성능 Deep Dive**

  와탭은 모든 트랜잭션의 수행 이력을 수집하고, SQL, Http 호출, 메서드 수준까지 추적해 성능을 분석합니다.
  
  <InDoc product='java,python,dotnet'>

  특히 **액티브 스택** 기능을 사용하면 번거로운 설정 없이 메서드 레벨 분석이 가능합니다. **액티브 스택**은 와탭의 특허 기술로, 기존 제품이 놓친 숨겨진 문제 원인을 정확히 찾아 개선 방향을 제시합니다.

  :::note

  액티브 스택 기능은 Java, Python, .NET에서만 지원합니다.

  :::

  </InDoc>

* **MSA 호출 패턴 분석**

  MSA 구조의 복잡한 애플리케이션간의 호출 관계를 트랜잭션 URL 기준으로 통계적으로 시각화하여 제공합니다. 시점별로 변화하는 호출 관계와 호출 비중의 변화를 통계적으로 파악할 수 있습니다.

</Xclude>

* **AI 기반 응답 분포 패턴 분석**

  와탭은 머신러닝을 활용해 특정 패턴을 자동 인식하고, 운영자가 인지하지 못한 이슈도 감지해 경고 알림을 제공합니다. <br/>

  :::

* **멀티 프로젝트 트랜잭션 연계 분석**

  MSA 환경에서 트랜잭션 연계 추적 기능을 제공합니다. 전체 트랜잭션을 추적하여 지연이 발생하는 애플리케이션을 식별할 수 있습니다.

</Xclude>

* **스택 통계 분석**

  통계 분석으로 실시간 모니터링이나 트레이스 정보에서 발견하기 힘든 튜닝 포인트를 찾아낼 수 있습니다. 거시적인 문제의 추이 또한 그래프와 표를 통하여 개선 대상을 선별하거나 특정 시간대의 응답 지연 현상을 확인할 수 있습니다.

</InDoc>

* **사후 분석**(**Cube**)

  응답 시간, 에러 수, 처리량 등의 주요 지표를 시간 흐름에 따라 시각화합니다. 문제가 발생한 시간대를 빠르게 식별하고, 특정 시점의 서비스 상태를 정확히 파악할 수 있습니다. 

<InDoc product='java'>

* **서비스 폭주 시 부하량 제어**

  시스템이 감당할 수 없는 과도한 요청이 들어오는 경우, 일부 요청을 거절하거나 잘못된 서비스 요청은 차단하여 안정적인 서비스 운영을 지원합니다.

* **비동기 트랜잭션 트레이스**

  와탭만의 고유한 BCI(Byte Code Instrumentation) 기술로 비동기 트랜잭션의 성능을 효과적으로 추적합니다. WebFlux, Reactor, Hystrix 등을 사용하는 애플리케이션의 트랜잭션 트레이스 정보를 수집합니다.
