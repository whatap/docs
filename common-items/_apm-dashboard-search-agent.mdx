### 에이전트 검색하기 {#searchagents}

프로젝트에 연결된 에이전트의 목록이 많아 찾기 어렵다면 에이전트 검색 기능을 이용하세요. 화면 오른쪽 위에 ![검색 아이콘](/img/ico-search.svg) <Cmdname sid="search" className="uitext" /> 버튼을 선택하면 문자열을 입력할 수 있는 상자가 나타납니다.

<ImgLang img="apm-dashboard-search-agent.png" desc="에이전트 검색" />

원하는 문자열을 입력하면 에이전트 목록에 입력한 문자열과 일치하는 에이전트만 표시됩니다. 검색한 에이전트를 기준으로 대시보드의 데이터를 필터링하려면 문자 입력 상자 오른쪽에 <Cmdname sid="select" className="uitext" /> 버튼을 선택하세요.

:::note

선택한 에이전트 항목을 초기화하려면 에이전트 선택 옵션 오른쪽에 <UI>Total</UI> 버튼을 선택하세요.

<ImgLang img="apm-dashboard-reset-agent.png" desc="Total" />

:::
