### 조회 시간 설정하기

<Cmdname sid="time" className="uitext" />에서 성능 지표를 조회할 날짜와 시간을 설정하세요.

* 가장 오른쪽의 녹색 버튼을 선택하면 다양한 시간 범위를 선택할 수 있습니다. 

  <InDoc pages="pageload-stat"><ImgLang img="browser-statistics-set-time-select-options.png" className="width-600" desc="시간 설정" /></InDoc>

  <InDoc pages="ipload-stat"><ImgLang img="browser-ip-statistics-set-time-select-options.png" className="width-600" desc="시간 설정" /></InDoc>

* ![왼쪽 방향 아이콘](/img/left-arrow.svg) 또는 ![오른쪽 방향 아이콘](/img/right-arrow.svg) 버튼을 선택해 시간을 이동할 수 있습니다. 이동 간격은 오른쪽에 설정된 시간 간격입니다.

* 원하는 날짜와 시간을 직접 선택하려면 날짜 및 시간 영역을 클릭하세요.

  <InDoc pages="pageload-stat"><ImgLang img="browser-statistics-set-custom-time.png" desc="시간 설정" /></InDoc>

  <InDoc pages="ipload-stat"><ImgLang img="browser-ip-statistics-set-custom-time.png" desc="시간 설정" /></InDoc>

:::note

* 최대 선택할 수 있는 시간은 3일이며, 최소 선택할 수 있는 시간은 5분입니다.

* 2시간 이하 범위를 선택하면 5분 단위로 데이터를 조회하고, 2시간을 초과하면 1시간 단위로 데이터를 조회합니다.

:::
