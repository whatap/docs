### 분류별 에이전트 모니터링

에이전트 설정에서 분류한 단위로 여러 에이전트를 쉽게 모니터링할 수 있습니다.

* <Cmdname sid="ITM06005" className="uitext" />: 개별 에이전트

* <Cmdname sid="ITM06006" className="uitext" />: 에이전트의 종류(예: 웹서버, 데이터베이스 등)로 분류된 그룹, <code>whatap.okind</code>로 설정

* <Cmdname sid="ITM06007" className="uitext" />: 서버(노드)별로 분류된 그룹, <code>whatap.onode</code>로 설정

* <Cmdname sid="ITM06008" className="uitext" />: <code>whatap.okind</code>로 분류된 에이전트 목록

* <Cmdname sid="ITM06009" className="uitext" />: <code>whatap.onode</code>로 분류된 에이전트 목록

:::note

에이전트 설정에 대한 자세한 내용은 [다음 문서](agent-name)를 참조하세요.

:::
