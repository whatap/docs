### 사용자 IP 주소 추가하기

<UI>TOP 30</UI> 섹션의 <Cmdname sid="ip" className="uitext" /> 기준으로 트랜잭션 수행 건수를 조회할 때 특정 IP 주소를 추가해 상시 확인할 수 있는 기능을 제공합니다.

1. <UI>TOP 30</UI> 섹션에서 <Cmdname sid="ip" className="uitext" />를 선택하세요.

1. 목록의 가장 위에 입력란에 특정 IP 주소를 입력하세요.

1. ![추가 아이콘](/img/ico-add.svg) 버튼을 클릭하세요.

    <ImgLang img='txmap-filter-ip-added.png' desc='IP' />

목록의 가장 위에 입력한 IP 주소가 고정되어 표시됩니다.

:::note

추가한 IP 주소를 삭제하려면 ![삭제 아이콘](/img/ico-trash.svg) 버튼을 클릭하세요.

:::
