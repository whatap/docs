## 클러스터 구성 {#cluster-set}

* **cluster_name** <span class='type'>String</span>

  클러스터 세트를 정의하는 옵션으로 동일한 `cluster_name` 옵션 값을 갖는 데이터베이스들은 <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_instanceList" className="uitext" /> 메뉴에서 트리 구조로 표시됩니다. <Cmdname sid="side_multiInstance" className="uitext" /> 메뉴의 에이전트 선택 옵션에서 <Cmdname sid="ITM07760" className="uitext" /> 항목을 선택해 클러스터를 구성하는 인스턴스들끼리 대시보드에 표현되어 한눈에 모니터링할 수 있습니다. 일부 위젯은 클러스터 단위의 정보를 제공합니다. 자세한 내용은 [다음 문서](multi-instance-monitoring#select-agent-category)를 참조하세요.
