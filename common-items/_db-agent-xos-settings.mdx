데이터베이스 서버의 프로세스 사용량을 모니터링하려면 데이터베이스 서버에 별도의 XOS 에이전트 프로세스를 띄워 데이터를 수집할 수 있습니다. 진행 중인 세션의 cpu usage 값을 함께 확인할 수 있습니다.

:::note

XOS 에이전트를 이용해 데이터베이스 프로세스 모니터링 설정 방법은 [다음 문서](agent-manage#xos)를 참조하세요.

:::

* **xos** <span class='type'>String</span>

  기본값 `0`

  XOS 에이전트 사용 여부를 설정합니다. 값이 `0`인 경우 xos 에이전트 수집 처리를 하지 않습니다. 값이 `1` 또는 xos 에이전트의 IP 주소를 입력하세요. IP 주소를 설정하면 해당 XOS 에이전트로부터만 데이터를 수집합니다.

* **xos_port** <span class='type'>String</span>

  기본값 `3002`

  XOS 에이전트와 통신할 포트를 설정합니다. XOS 에이전트와는 UDP로 통신합니다. 해당 포트에 방화벽이 설정되어 있다면 UDP 통신이 될 수 있도록 방화벽을 해제하세요.

* **xos_cpu_iowait** <span class='type'>Int</span>

  기본값 `1`

  XOS 에이전트로부터 수집한 CPU 사용률에 iowait의 값 포함 여부를 설정합니다. 값을 `1`로 설정할 경우 iowait를 포함하고 `0`으로 설정하면 iowait를 포함하지 않습니다.

* **xos_mem_buffcache** <span class='type'>Int</span>

  기본값 `0`

  XOS 에이전트로부터 수집한 메모리 사용률에 buffcache의 값 포함 여부를 설정합니다. 값을 `1`로 설정할 경우 buffcache를 포함하고 `0`으로 설정하면 buffcache를 포함하지 않습니다.
