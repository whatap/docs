## 인스턴스 성능 지표 비교하기

2개 이상의 인스턴스를 선택해 성능 지표를 비교할 수 있습니다. 성능 지표를 비교할 인스턴스를 목록에서 선택하면 화면 오른쪽 위에 <Cmdname sid="multi_view" className="uitext" /> 버튼이 활성화됩니다. <Cmdname sid="multi_view" className="uitext" /> 버튼을 선택하면 <Cmdname sid="side_multiInstance" className="uitext" /> 메뉴로 이동합니다. 주요 성능 지표의 데이터를 비교한 그래프 차트와 액티브 세션과 락 트리에 대한 상세 정보를 확인할 수 있습니다.

:::note

<Cmdname sid="side_multiInstance" className="uitext" />에 대한 자세한 내용은 <Link to="multi-instance-monitoring">다음 문서</Link>를 참조하세요.

:::
