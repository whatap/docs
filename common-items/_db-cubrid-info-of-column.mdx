## 컬럼 정보 안내{#column-info}

| 항목 | 설명 |
| -- | -- |
| tranindex | 트랜잭션 인덱스 |
| user | 데이터베이스 사용자 이름 |
| host | 트랜잭션을 수행하는 CAS의 호스트 이름 |
| pid | 브로커의 프로세스 ID |
| program | 클라이언트 프로그램 이름 |
| query_time | 수행 중인 쿼리의 총 수행 시간(단위: 초) |
| tran_time | 트랜잭션의 총 수행 시간(단위: 초) |
| transtatus | 트랜잭션의 상태 |
| lock_holder | 트랜잭션이 락 대기 중이면 해당 락을 소유하고 있는 트랜잭션의 목록 |
| sql_id | SQL Text에 대한 ID |
| sql_text | 수행 중인 쿼리문(최대 30자) |
| sql_param | 쿼리에서 분리한 리터럴 값 |
| sql_hash | 쿼리의 해시값(와탭 내부 관리 값) |
