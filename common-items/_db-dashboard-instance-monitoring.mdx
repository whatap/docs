## 인스턴스 모니터링 {#side_dbMonitoring}

* 하나의 화면에서 동시에 여러 인스턴스를 그룹화하여 모니터링합니다. 관심이 있는 중요 지표들을 선택하여 전반적인 시스템 상황을 모니터링 가능합니다.

* 실시간 모니터링은 5초 간격으로 성능 지표들을 모니터링합니다. 현재 진행 중인 <Cmdname sid="active_sessions" className="uitext" />, <Cmdname sid="lock_tree" className="uitext" />, <Cmdname sid="process_info" className="uitext" />를 확인할 수 있습니다. 

자세한 내용은 [다음 문서](instance-monitoring)를 참조하세요.
