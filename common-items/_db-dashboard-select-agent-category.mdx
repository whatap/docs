### 분류별 에이전트 모니터링 {#select-agent-category}

<ImgLang img='db-dashboard-select-agent-by-category.png' desc='분류별 에이전트 선택하기' />

에이전트 설정을 통해 분류한 그룹 단위로 에이전트를 선택해 모니터링할 수 있습니다.

* <Cmdname sid="ITM06005" className="uitext" />: 개별 에이전트를 선택하거나 모두 선택할 수 있습니다.

* <Cmdname sid="ITM07760" className="uitext" />: 클러스터로 구성된 데이터베이스의 경우 클러스터별 단위로 에이전트를 에이전트를 모니터링할 수 있습니다.

:::note

<Cmdname sid="ITM07760" className="uitext" /> 항목을 이용하려면 DBX 에이전트(*whatap.conf*)의 다음 옵션을 설정해야 합니다. 자세한 내용은 [다음 문서](agent-dbx-settings#cluster-set)를 참조하세요.

```ini title="whatap.conf"
cluster_name={cluster_name}
```

:::
