### 조회 목록 다운로드하기 {#downloadcsv}

현재 조회 중 인 ***Session Table*** 목록을 CSV 파일 형식으로 다운로드할 수 있습니다. ***Session Table*** 섹션의 오른쪽 위에 ![다운로드 아이콘](/img/ico-download.svg) 버튼을 선택하세요.

다운로드한 파일의 이름 형식은 다음과 같습니다.

* <Cmdname sid="active_sessions" className="uitext" />: <em>activeSession_HH_MM_SS.csv</em>

* <Cmdname sid="lock_tree" className="uitext" />: <em>lockTree_HH_MM_SS.csv</em>
