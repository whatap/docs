테이블의 컬럼 항목을 표시하거나 숨길 수 있고, 순서를 변경할 수 있습니다.

1. 테이블의 오른쪽 위에 ![컬럼 아이콘](/img/ico-column.svg) 버튼을 선택하세요. 

1. ***컬럼 편집*** 창이 나타나면 테이블의 컬럼에 표시할 항목을 숨기거나 추가, 순서를 변경하세요.

    * 컬럼 항목을 선택한 다음 ![왼쪽 방향 아이콘](/img/left-arrow.svg) 또는 ![오른쪽 방향 아이콘](/img/right-arrow.svg), ![추가 아이콘](/img/ico-add.svg) 또는 ![빼기 아이콘](/img/ico-minus.svg) 버튼을 선택하세요. ***표시*** 또는 ***숨김***으로 이동합니다. 
    * 컬럼 항목을 선택한 상태에서 드래그하면 위치를 이동할 수 있습니다.

1. 설정을 완료한 다음 ***저장*** 버튼을 선택하세요.

:::note

설정한 내용은 브라우저 쿠키값으로 저장되어 페이지 새로 고침 후에도 상태를 유지합니다. 쿠키 삭제 또는 기타 이유로 설정 간 오류가 발생하면 초기화합니다.

:::
