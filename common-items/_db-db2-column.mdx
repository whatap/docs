<TabItem value="activesession" label="Active session">

| 항목 | 설명 |
| -- | -- |
| `instance`            | DB2 인스턴스(에이전트) 이름입니다. |
| `dbname`              | 데이터베이스 이름입니다.  |
| `application_handle`  | 특정 애플리케이션을 식별하는 고유 핸들 값입니다. |
| `username`            | 현재 세션의 사용자 이름입니다. |
| `state`               | 세션의 현재 상태입니다. |
| `activity_type`       | 세션에서 실행 중인 작업 유형입니다. |
| `query`               | 세션에서 실행 중인 SQL 쿼리의 내용입니다. |
| `application_id`      | 애플리케이션의 고유 ID로,세션을 구분하는 데 사용합니다. |
| `application_name`    | 세션이 실행 중인 애플리케이션의 이름입니다.            |
| `client_hostname`     | 세션을 시작한 클라이언트의 호스트 이름 또는 IP 주소입니다. |
| `last_reference_time` | 세션에서 마지막으로 작업을 실행한 시간입니다. |
| `local_start_time`    | 세션을 시작한 로컬 시간입니다. |
| `query_hash`          | 실행 중인 쿼리의 고유 해시 값으로, 쿼리 내용을 식별하기 위한 값입니다. |
| `runtime`             | 현재 세션이 실행된 시간입니다. |
| `time`                | 세션이 작업에 소모한 총 시간입니다. |

:::note

와탭은 클라이언트와 관련한 정보를 기본 저장합니다.

:::

</TabItem>

<TabItem value="locktree" label="Lock tree">

| 항목 | 설명 |
| -- | -- |
| `instance`            | DB2 인스턴스(에이전트) 이름입니다. |
| `application_handle`  | 특정 애플리케이션을 식별하는 고유 핸들 값입니다. |
| `dbname`              | 데이터베이스 이름입니다.  |
| `username`            | 현재 세션의 사용자 이름입니다. |
| `runtime`             | 현재 세션이 실행된 시간입니다. |
| `state`               | 세션의 현재 상태입니다. |
| `activity_type`       | 세션에서 실행 중인 작업 유형입니다. |
| `query`               | 세션에서 실행 중인 SQL 쿼리의 내용입니다. |
| `lock_name`              | 락의 고유 식별자입니다. |
| `lock_object_type`       | 락이 적용된 객체 유형입니다. |
| `table_name`             | 락이 적용된 테이블의 이름입니다. |
| `holder_application_name`| 락을 보유 중인 애플리케이션의 이름입니다. |
| `holder_member`          | 락을 보유 중인 멤버의 ID입니다. |
| `waiter_application_name`| 락 대기 중인 애플리케이션의 이름입니다. |
| `waiter_member`          | 락 대기 중인 멤버의 ID입니다. |
| `wait_time`              | 락 대기 시간입니다. |

</TabItem>

<TabItem value="process-info" label="Process information">

다음 항목은 XOS 에이전트를 설치한 경우 수집되는 지표입니다.

| 항목 | 설명 |
| -- | -- |
| `pid` | 데이터베이스 프로세스의 PID입니다. |
| `cputime` | CPU 사용 시간입니다. |
| `cpuusage` | CPU 사용률입니다. |
| `elapse` | CPU 사용 경과 시간입니다. |
| `vsize` | 가상 메모리 사이즈(Kb)입니다. |
| `rss` | Resident Set Size(RSS), 프로세스와 관련한 물리적 페이지 수입니다. |
| `state` | 프로세스 상태입니다. |
| `ioread` | 블록을 읽는 데 소요된 실제 시간(milliseconds)입니다. |
| `iowrite` | 블록을 쓰는 데 소요된 실제 시간(milliseconds)입니다. |
| `pss` | 프로세스 고유 메모리 사용량 + 하나의 프로세스가 차지하는 공유 메모리 비율 |

</TabItem>
