## DBX 에이전트 다운로드

DBX 에이전트 파일을 다운로드하세요. 다음 두 가지 방법을 이용하세요.

* `wget` 명령어를 이용해 다운로드할 수 있습니다.

  ```bash title="BASH"
  wget -O whatap.agent.database.tar.gz https://api.whatap.io/download/agent/whata.agent.database.tar.gz
  ```

* 명령어로 다운로드할 수 없다면 와탭 모니터링 서비스 화면에서 <Cmdname sid="download" className="uitext" /> 버튼을 선택하세요.

DBX 에이전트 파일을 다운로드한 다음 압축을 해제하세요.

```bash title="BASH"
tar -zxvf whatap.agent.database.tar.gz
```
