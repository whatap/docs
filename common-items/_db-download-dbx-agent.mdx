## 에이전트 다운로드

DBX 에이전트 파일을 다운로드하세요. 다음 두 가지 방법 중 하나를 이용할 수 있습니다.

### wget 명령어를 이용한 다운로드

모니터링할 데이터베이스 종류에 따라 `type` 파라미터에 해당 값을 입력하세요.  
아래 예시를 참고해 데이터베이스 이름을 지정하면 됩니다.

```bash title="BASH"
# PostgreSQL
wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=postgresql&format=tar.gz"

# Oracle
wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=oracle&format=tar.gz"

# MySQL
wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=mysql&format=tar.gz"

# Altibase
wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=altibase&format=tar.gz"

# CUBRID
wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=cubrid&format=tar.gz"

# MSSQL
wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=mssql&format=tar.gz"

# DB2
wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=db2&format=tar.gz"

# SAP ASE
wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=sap_ase&format=tar.gz"
```

### 서비스 화면에서 직접 다운로드

명령어로 다운로드할 수 없는 경우,
와탭 모니터링 서비스 화면에서 <Cmdname sid="download" className="uitext" /> 버튼을 선택하세요.

DBX 에이전트 파일을 다운로드한 다음 압축을 해제하세요.

```bash title="BASH"
tar -zxvf whatap.agent.database.tar.gz
```

:::note

보안 설정으로 인해 tar 형식의 파일을 다운로드할 수 없는 사용자를 위해 ZIP 형식의 파일도 함께 제공합니다. 설치 화면에서 <UI>.zip</UI> <Cmdname sid="download" className="uitext" /> 버튼을 선택하세요.

:::
