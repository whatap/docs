<InDoc product='postgresql'>

홈 화면 > 프로젝트 선택 > ***사이트맵*** > ***경고 알림*** > ***파일 로그 알림***

</InDoc>

![파일 로그 알림](/img/db-file-log-notice.png)

* ***시간***: 조회 시간을 선택하거나 선택할 수 있습니다.

* ***인스턴스***: 조회하길 원하는 인스턴스를 선택하세요.

* ***로그 파일 목록***: 인스턴스에 해당하는 로그 파일 목록을 확인할 수 있습니다.

* ***로그 발생 기록***: 설정한 파일 이름과 설정한 키워드를 통해 로그가 발생한 시간을 확인할 수 있습니다.
