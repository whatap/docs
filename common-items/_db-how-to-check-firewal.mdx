### 방화벽 확인 방법

`telnet` 명령어를 수행해 다음과 같은 접속 관련 정보가 표시되어야 정상입니다. (`telnet [서버 IP] [서버 포트]`)

```bash title='예시'
$ telnet 52.193.60.176 6600
Trying 52.193.60.176...
Connected to 52.193.60.176.
Escape character is '^]'.
```

:::note

와탭 수집 서버 정보는 ***프로젝트 메뉴*** > ***관리*** > ***에이전트 설치*** 메뉴에서 확인할 수 있습니다.

:::
