## 기본 화면 안내

<ProdImg img='db-instance-list.png' desc='인스턴트 목록' />

* ![숫자 1](/img/number-01.png) 영역은 프로젝트에 설정된 인스턴스들의 상태를 표시합니다. 표시된 현황 정보는 전체 인스턴스의 개수와 일치하고, 테이블의 <Cmdname sid="status" className="uitext" /> 컬럼 정보와 일치합니다.

  <InDoc product="postgresql,mysql,oracle-pro">

  * <span class='snormal'><Cmdname sid="sms_alert_normal" /></span>: 정상 상태의 에이전트 수입니다.

  * <span class='slow'><Cmdname sid="sms_alert_warn" /></span>: <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventNoti" className="uitext" /> 메뉴에서 설정한 이벤트의 임곗값이 <span class='slow'>경고</span> 수준을 넘은 에이전트 수입니다.

  * <span class='vslow'><Cmdname sid="sms_alert_fatal" /></span>: <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventNoti" className="uitext" /> 메뉴에서 설정한 이벤트의 임곗값이 <span class='vslow'>위험</span> 수준을 넘은 에이전트 수입니다.

  :::note

  <Cmdname sid="side_event" className="uitext" /> 메뉴에 대한 자세한 내용은 <Link to="set-notice">다음 문서</Link>를 참조하세요.

  :::

  </InDoc>

* <Cmdname sid="status" className="uitext" /> 컬럼에서 정상이 아닌 상태 아이콘을 클릭하면 <Cmdname sid="event" className="uitext" /> 창이 나타납니다. 해당 인스턴스에서 발생한 이벤트 내역을 확인할 수 있습니다.

* 각 인스턴스를 선택하면 <Cmdname sid="side_dbMonitoring" className="uitext" /> 메뉴로 이동합니다. 선택한 인스턴스에 대한 상세 정보를 실시간으로 모니터링할 수 있습니다. 

  :::note

  <Cmdname sid="side_dbMonitoring" className="uitext" />에 대한 자세한 내용은 <Link to="instance-monitoring">다음 문서</Link>를 참조하세요.

  :::

* 주요 성능 지표를 그래프 차트 형태로 제공합니다. 가장 높은 수치를 차트에 표시합니다. 차트에 마우스를 오버하면 해당 위치의 수치를 확인할 수 있습니다. 성능 지표에 대한 자세한 내용은 [다음 문서](metrics-data-list)를 참조하세요.

<Xclude product="oracle,oracle-pro,oracle-v1,altibase,mssql">

* <Cmdname sid="instance_list" className="uitext" />에서 <Cmdname sid="server_name" className="uitext" /> 컬럼에 표시된 <span class='mark master'>M</span>, <span class='mark instance'>S</span>, <span class='mark master'>C</span>는 각각 Master, Slave, Cluster를 의미합니다. Replication으로 구성한 DB일 경우 표시합니다.

  Replication 인스턴스가 비정상 상태인 경우 <LinkImage img='ic-warinig-db.svg' desc='경고' className='ico' /> 아이콘이 표시됩니다.

  <ImgLang img='db-instance-list-replication-warning.png' />

</Xclude>

* 화면 오른쪽 위에 ![알림 아이콘](/img/ico-noti.svg) 버튼을 선택하면 실시간 알림을 확인할 수 있습니다. 알림 설정에 대한 자세한 내용은 [다음 문서](set-notice)를 참조하세요.
