### 연계 프로젝트 확인하기

<Cmdname sid="side_correlatedProjectManagement" className="uitext" /> 메뉴에서 다른 애플리케이션 프로젝트를 추가하면, 액티브 세션 항목에 ![연계 아이콘](/img/ic-correlations.svg) 버튼이 추가됩니다.

<ImgLang img="db-instance-monitoring-linked-project.png" desc="연계 프로젝트" />

![연계 아이콘](/img/ic-correlations.svg) 버튼을 선택하면 나타나는 <Cmdname sid="TTL07046" className="uitext" /> 창을 통해서 사용자는 액티브 세션과 연동된 애플리케이션의 액티브 트랜잭션 정보를 실시간으로 확인할 수 있습니다. 트랜잭션 URL 및 데이터베이스 연결 정보, DB 타입, HTTP 메서드, 실행 시간 등의 기본 정보와 함께 트랜잭션 ID, 스레드 ID, 클라이언트 IP, CPU 사용 시간과 같은 구체적인 지표를 확인할 수 있습니다. 또한, SQL 쿼리 실행 시간, 호출된 SQL 개수, DB 연결 시간 등도 모니터링할 수 있어 시스템의 성능을 종합적으로 분석할 수 있습니다.

특히, 스택 트레이스를 통한 콜 스택(Call stack) 정보와 SQL 쿼리를 함께 제공하여 문제 발생 시 상세한 원인을 추적하고 해결하는 데 도움이 됩니다. 이를 통해 애플리케이션과 데이터베이스 간의 상호작용을 심층적으로 분석하고 성능 병목 현상을 신속히 진단할 수 있습니다.

:::note

연계 프로젝트 추가 및 활용에 대한 자세한 내용은 [다음 문서](linked-projects)를 참조하세요.

:::
