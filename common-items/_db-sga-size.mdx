## SGA 사이즈

홈 화면 > 프로젝트 선택 > ***통계/보고서*** > ***SGA 사이즈***

오라클 서버 메모리 영역인 SGA 요약 정보를 매 시간 저장하여 시간별 변화량을 모니터링할 수 있습니다. 

<ProdImg img='db-stat-sga-size.png' alt='SGA 사이즈' />

1. ***시간***에서 조회하고 싶은 시간을 설정하세요.

2. 인스턴스에서 추이 정보를 조회하려는 인스턴스 이름을 선택하세요.

3. ![검색 아이콘](/img/ico-btn-search.png) 버튼을 선택하세요.

:::note

조회 시간은 최대 3주까지 선택할 수 있습니다. 임의의 날짜와 시간을 조회하려면 시계 표시 부분을 선택하세요. 날짜와 시간 텍스트 영역을 클릭하면 날짜와 시간을 선택할 수 있는 옵션이 나타납니다.

:::
