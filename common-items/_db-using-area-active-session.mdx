## 액티브 세션 영역 활용하기

화면 아래에 위치한 ***액티브 세션*** 영역을 통해 실시간 수행 중인 세션 정보를 확인할 수 있습니다. 다음은 테이블 오른쪽 위에 버튼에 대한 기능 안내입니다.

<Xclude product='postgresql'>

* ![정지 아이콘](/img/ico-pause.svg): 실시간으로 수집하는 데이터의 변화를 멈출 수 있습니다. 다시 시작하려면 ![시작 아이콘](/img/ico-play-grey.svg) 버튼을 선택하세요.

</Xclude>

* ![새로 고침 아이콘](/img/ico-refresh.svg): 테이블의 목록을 새로 고침할 수 있습니다.

* ![컬럼 아이콘](/img/ico-column.svg): 테이블의 컬럼 헤더 항목을 표시하거나 숨길 수 있습니다.

<InDoc product='postgresql'>

* ![필터 아이콘](/img/ico-filter.svg): 테이블의 컬럼 헤더 항목을 기준으로 목록을 필터링할 수 있습니다.

</InDoc>

<Xclude product='postgresql'>

* ![필터 아이콘](/img/ico-filter.svg): 테이블의 컬럼 헤더 항목을 기준으로 목록을 필터링할 수 있습니다. 버튼을 선택하면 각 헤더 컬럼에서 ***포함***, ***미포함***, ***일치***, ***불일치*** 조건을 설정할 수 있습니다.

  :::note

  필터 조건 옵션을 입력하는 텍스트 상자가 보이지 않을 경우 컬럼 너비를 조절하세요.

  :::

* ![다운로드 아이콘](/img/ico-download.svg): 테이블의 내용을 CSV 파일로 다운로드할 수 있습니다.

</Xclude>
