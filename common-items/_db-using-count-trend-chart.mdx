## 카운트 추이 차트 활용하기

카운트 추이 관련 메뉴에서 다음의 기능을 활용할 수 있습니다.

<ImgLang img="analysis-count-24.png" desc="카운트 추이 24시간" />

화면 오른쪽 위에 ![레이아웃 아이콘](/img/ico-layout-set.svg) 버튼을 선택하세요. **레이아웃 설정** 창에서 **1 x 3**으로 설정한 다음 **저장** 버튼을 선택하세요. **시간**은 **최근 24시간**으로 설정하세요. 다음과 같이 하루 동안의 데이터베이스 핵심 지표를 자세히 확인할 수 있습니다.

<ImgLang img="analysis-count-usage.png" desc="카운트 추이 시간별 확인" />

* 지표 차트의 특정 시점을 클릭하면(![지시선 1](/img/number-01.png)) 해당 지점을 붉은 선으로 표시하며 당시에 수집된 액티브 세션을 확인할 수 있습니다. 

* 액티브 세션 데이터는 5초 단위로 수집하고 있습니다. 화면 아래에 ![지시선 2](/img/number-02.png) 초 단위 셀을 선택해 5초 단위로 데이터를 확인할 수 있습니다.

* 1분 단위로 시간을 이동하려면 ![지시선 3](/img/number-03.png) ![이전 시간 아이콘](/img/ico-left-time.svg) 또는 ![이후 시간 아이콘](/img/ico-right-time.svg) 버튼을 선택하세요.

* 상세 정보 데이터 셀에 표시하는 항목을 더하거나 빼려면 ![지시선 3](/img/number-03.png) ![컬럼 아이콘](/img/ico-column-set.svg) 버튼을 선택하세요. **컬럼 편집** 창이 나타면 원하는 항목을 더하거나 뺄 수 있습니다.

* ![지시선 3](/img/number-03.png) ![필터 아이콘](/img/ico-filter.svg) 버튼을 선택하면 데이터 셀 헤더에 필터를 적용해 원하는 데이터만 정렬할 수 있습니다.

* 화면 아래에 출력된 데이터를 csv 파일로 다운로드할 수 있습니다. ![지시선 3](/img/number-03.png) ![다운로드 아이콘](/img/ico-download.svg) 버튼을 선택하세요.
