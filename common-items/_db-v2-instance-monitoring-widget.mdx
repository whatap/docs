### ![](/img/number-02.png) 그래프 차트 섹션

![숫자 3](/img/number-03.png) ***Default*** 프리셋으로 저장한 16가지 지표를 그래프 차트 형식의 위젯으로 표시합니다. 가로축은 시간, 세로축은 각 지표의 수치입니다. 각 지표의 의미를 알고 싶다면 이름 옆에 ![정보 아이콘](/img/ic-info-circle.svg) 버튼을 선택하거나 [다음 문서](metrics-data-list)를 참조하세요.

* 다른 위젯과 위치를 변경하려면 위젯의 윗 부분을 선택한 상태에서 드래그하세요. 단 위젯의 크기는 변경할 수 없습니다.

  ![위젯 이동하기](/img/db-dashboard-move-widget.png)

* 위젯에서 차트의 특정 시간을 선택하면 **실시간 조회** 모드를 중지하고 선택한 시간에 수행한 세션 정보를 ![숫자 4](/img/number-04.png) <Cmdname sid="active_sessions" className="uitext" /> 테이블에 표시합니다. 이후 <Cmdname sid="active_sessions" className="uitext" /> 테이블 위에는 5초 단위로 데이터를 조회할 수 있는 버튼이 나타납니다.

  <ProdImg img='db-instance-monitoring-seltime.png' />

  :::note

  * 선택한 시간은 <Cmdname sid="active_sessions" className="uitext" /> 테이블 오른쪽 위에서 확인할 수 있습니다.

  * <Cmdname sid="active_sessions" className="uitext" />  테이블 목록에 글자 색상은 검정색 → <span class='slow'>주황색</span> → <span class='vslow'>빨간색</span> 순으로 세션의 수행 속도가 느린 것을 의미합니다.

  :::

<InDoc product="oracle,oracle-pro">

* <UI>Top Stat</UI>: 위젯에 표시된 버튼을 선택하면 사용자 커밋에 대한 상위 통계를 제공합니다. 모니터링하는 인스턴스에서 트랜잭션 커밋이 발생한 세션 정보와 성능 관련 주요 지표를 보여줍니다. 

  * 특정 **sid** 또는 **module**에서 트랜잭션이 집중되는지를 확인할 수 있습니다.

  * **program** 컬럼을 통해 특정 작업이 어떤 프로세스를 통해 수행되었는지 파악할 수 있습니다.

* <UI>Top Event</UI>: 위젯에 표시된 버튼을 선택하면 해당 지표와 관련한 주요 이벤트 대기에 대한 정보를 제공합니다. 이 데이터를 통해 성능 저하의 원인을 확인하고 문제가 발생하는 세션을 진단할 수 있습니다.

  * **time waited** 값이 높은 세션을 확인하여 병목 구간을 찾아낼 수 있습니다.

  * 특정 **module** 또는 **program**에서 대기 이벤트가 집중되는지 확인할 수 있습니다.

:::note

<UI>Top Stat</UI> 또는 <UI>Top Event</UI> 버튼은 제공하는 지표에 따라 제공하지 않을 수 있습니다.

:::
