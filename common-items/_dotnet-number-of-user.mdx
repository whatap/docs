* **trace_user_cookie_keys** <span class='type'>String</span>

  설정한 쿠키의 값을 고유한 사용자 정보로 선택합니다. 여러 개를 등록할 때는 쉼표(,)를 구분자로 이용하세요. 설정한 순서대로 검색합니다.

* **trace_user_using_ip** <span class='type'>Boolean</span>

  기본값 `false`

  실시간 사용자를 IP 주소 기반으로 집계합니다. IP 주소가 아닌 쿠키를 기반으로 사용자를 구분하고 싶으면 값을 `false`로 변경하세요.
