:::caution

플러그인은 충분히 이해한 경우에만 사용하길 권장합니다.

:::

*WHATAP_HOME* 경로 하위에 *plugin* 폴더를 생성하고, *plugin*로 이동하세요. `vi` 명령어로 *AppServiceStart.x* 파일을 생성하세요.

![플러그인 사용 예시](https://img.whatap.io/media/agent_python/data/app_start_plug.png)

`println("test");`라고 타이핑하고 저장하면 화면에 "test"라는 문자열이 출력되는 것을 확인할 수 있습니다. 확인 후 파라미터에서 정보를 추출하세요. 

다음 예시에서는 `HashMap`이 인수로 전달되고, 이 `HashMap`에는 'url'이라는 키가 포함되어 있습니다.

```javascript title=JAVA
Object url =((java.util.HashMap)$point.getArgs()[0]).get("url");
$ctx.service((String)url);
//println("url="+url);
```

이렇게 플러그인을 만들면 트랜잭션 이름을 변경할 수 있습니다.
