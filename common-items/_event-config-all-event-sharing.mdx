## 모든 이벤트 설정 공유하기 {#share-event-setting}

기본 이벤트를 포함한 모든 이벤트 설정을 **JSON 또는 Excel 형식으로 다운로드**할 수 있습니다. 같은 유형의 상품 설정을 공유하면, 반복되는 이벤트 설정 작업을 줄일 수 있습니다.

### 내보내기 {#export-event-setting}

모든 이벤트 설정을 포함한 JSON 파일을 다운로드할 수 있습니다. 다운로드한 파일은 JSON 일괄 수정 기능을 통해 편집할 수 있습니다. 화면 상단의 ![다운로드 아이콘](/img/ico-download.svg) <Cmdname sid="BTN07762" className="uitext" /> 버튼을 클릭하세요.

* JSON 파일 이름은 *integrated-event-`YYYY`-`MM`-`DD`.json* 형식입니다.

* JSON 데이터의 구조와 `events` 속성의 하위 속성은 상품(`platform`) 또는 상품 유형(`productType`)에 따라 다를 수 있습니다.

:::caution

JSON 데이터의 속성 중 `productType` 또는 `platform` 속성의 값을 임의로 수정하면 다른 프로젝트에서 이벤트가 동작하지 않을 수 있습니다.

:::

**JSON 데이터 구조**

```json title="integrated-event"
{
  "productType": "{productType}",
  "platform": "{platform}",
  "events": {
    "HITMAP": {
      ...
    },
    "LOG_REALTIME": [
      {...}
    ],
    "METRICS": [
      {...}
    ],
    "COMPOSITE_LOG": [
      {...}
    ],
    "BASIC": [
      {...}
    ],
    "COMPOSITE_METRICS": [
      {...}
    ],
    "ANOMALY": [
      {...}
    ]
  }
}
```

### 불러오기 {#import-event-setting}

1. 화면 상단의 ![편집 아이콘](/img/ico-edit.svg) <Cmdname sid="BTN07761" className="uitext" /> 버튼을 클릭하세요.

1. JSON 편집창이 열리면 화면 아래에 ***가져오기*** 버튼을 클릭하세요.

1. ![다운로드 아이콘](/img/ico-download.svg) <Cmdname sid="BTN07762" className="uitext" /> 기능을 통해 다운로드 받은 JSON 파일을 선택하세요.

1. <Cmdname sid="overwrite" className="uitext" /> 버튼을 선택하세요.

1. 확인 메시지가 나타나면 <Cmdname sid="overwrite" className="uitext" /> 버튼을 클릭하세요.

:::caution

덮어쓰기 후 이전의 이벤트 설정으로 복구할 수 없습니다.

- `productType` 또는 `platform` 속성의 값을 임의로 수정하면 다른 프로젝트에서 이벤트가 정상 동작하지 않을 수 있습니다.

- JSON 편집창은 JSON 검증(Validation) 기능이 내장되어 있습니다. 형식이나 값이 올바르지 않은 경우, 편집창 아래에 오류 메시지가 표시되며, <Cmdname sid="overwrite" className="uitext" /> 버튼은 비활성화됩니다.

  <ImgLang img='event-config-v2-all-events-json-validate.png' />

:::
