## Flex 보드 복사하기

사용자가 생성한 Flex 보드를 복사해 다른 사용자에게 전달할 수 있습니다.

***Flex 보드*** 메뉴의 ***보드*** 목록에서 ![복사 아이콘](/img/ico-copy.svg) 버튼을 선택하세요. ***나에게 복사*** 또는 ***다른 사람에게 복사*** 옵션을 선택하세요.

* ***나에게 복사*** 옵션을 선택하면 ***보드*** 목록에 '_copied' 접두어가 붙은 항목이 추가됩니다.

* ***다른 사람에게 복사*** 옵션을 선택하면 ***+ 계정***을 클릭하세요. 다른 사용자 계정의 이메일을 입력한 다음 ***대시보드 복사하기*** 버튼을 선택하세요. ***읽기 전용*** 옵션을 선택하면 공유 받은 사용자가 대시보드를 수정할 수 없습니다.

:::note

***프로젝트 메뉴*** > ***Flex 보드*** 메뉴에서 복사 기능은 프로젝트 멤버와 대시보드를 공유합니다.

:::

## Flex 보드 내보내기, 가져오기

json 파일로 저장한 Flex 보드 설정을 다른 사용자 계정에서 가져와 복사할 수 있습니다. 대시보드 관리자 권한의 기능입니다.

1. ***Flex 보드*** 메뉴의 ***보드*** 목록에서 ![내보내기 아이콘](/img/ico-export.svg) 버튼을 선택하세요. 

1. 웹 브라우저에서 Flex 보드 설정을 json 파일로 다운로드합니다.

1. 화면 오른쪽 위에 ![가져오기 아이콘](/img/ico-import.svg) ***가져오기*** 버튼을 선택하세요.

1. json 파일을 선택하세요.

:::note

이 기능은 **프로젝트 수정 권한**을 소유한 사용자만 이용할 수 있습니다.

:::
