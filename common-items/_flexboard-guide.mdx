## Flex 보드 만들기 {#createflexboard}

Flex 보드를 생성하고 위젯을 배치해 자신만의 대시보드를 만들 수 있습니다.

1. <Cmdname sid="side_flexboard" className="uitext" />에서 오른쪽 위의 <Cmdname sid="create_dashboard" className="uitext" /> 버튼을 클릭하세요.

1. <Cmdname sid="create_dashboard" className="uitext" /> 창이 나타나면 입력 상자에 대시보드 <Cmdname sid="name" className="uitext" />을 입력하세요.

1. 위젯 배치 방식을 선택한 후, <Cmdname sid="create_dashboard" className="uitext" /> 버튼을 클릭하세요.

    * <Cmdname sid="fixed_flexboard" className="uitext" />: 픽셀 기반으로 위젯 자유 배치

    * <Cmdname sid="flexible_flexboard" className="uitext" />: 브라우저 크기 기준 그리드 반응형 레이아웃  
    (그리드 설정한 뒤 <Cmdname sid="create_dashboard" className="uitext" /> 버튼 클릭)

1. <Cmdname sid="widget_template" className="uitext" /> 선택 화면에서 대시보드에 배치할 위젯을 선택하세요. 

    * 메트릭스 위젯을 추가하려면 <Cmdname sid="widget_template" className="uitext" />의 오른쪽 **모든 메트릭스** 버튼을 클릭하세요.

    :::note

    * <Cmdname sid="widget_template" className="uitext" />: 일반적인 모니터링 상황에서 중요하게 다뤄지는 지표를 간추려 사전 정의된 위젯 목록을 선택할 수 있습니다.
    * <Cmdname sid="all_metrics" className="uitext" />: 사용자의 프로젝트에서 수집 중인 모든 메트릭스 데이터를 기준으로 위젯을 생성합니다. 메트릭스 위젯에 대한 자세한 설명은 [다음 문서](flexboard-metric-widget)를 참고하세요.
    
    :::

1. 배치한 위젯의 위치를 이동하거나 크기를 조절하세요.

    * 위젯의 위쪽에 커서를 올려 십자 커서가 보이면, 위젯을 클릭한 채 드래그하여 위치를 이동합니다. 

    * 위젯 오른쪽 아래에 커서를 올려 화살표 커서가 보이면, 위젯을 클릭한 채 드래그하여 크기를 조절합니다. 

1. 추가할 위젯 배치를 완료하면 <Cmdname sid="view_mode" className="uitext" />를 선택해 레이아웃을 확인하세요. 레이아웃을 다시 수정하려면 <Cmdname sid="edit_mode" className="uitext" />를 선택해 위젯의 배치를 변경하세요.

1. 모든 과정을 완료했다면 오른쪽 위 ![목록으로 가기 아이콘](/img/ico-exit.svg) (<Cmdname sid="go_to_list" className="uitext" />) 아이콘을 클릭하세요. 

    * <Cmdname sid="side_flexboard" className="uitext" />의 <Cmdname sid="dashboard_list" className="uitext" />에서 생성한 보드를 확인할 수 있습니다. 생성한 보드를 선택해 새로 생성한 Flex 보드를 확인할 수 있습니다.

:::note

고객의 의견을 반영해 대시보드 템플릿 또는 위젯을 추가하려고 합니다. 필요한 대시보드 템플릿 또는 위젯이 있다면 [support@whatap.io](mailto:support@whatap.io)로 문의해 주세요.

:::

      :::caution

      <Cmdname sid="project" className="uitext" /> 옵션은 **홈 화면** &gt; <Cmdname sid="side_integratedFlexboard" className="uitext" /> 메뉴에서만 설정할 수 있습니다.  
      <Cmdname sid="side_integratedFlexboard" className="uitext" /> 메뉴로 진입한 경우, 위젯마다 데이터를 가져올 프로젝트를 선택할 수 있습니다. 
      
      1. 위젯 오른쪽 위 ![추가 아이콘](/img/ico-add.svg) 버튼을 클릭하고 원하는 프로젝트를 검색해 하나 또는 다중 선택하세요.
      
      1. 프로젝트를 선택한 후 <Cmdname sid="apply" className="uitext" /> 버튼을 클릭하세요.
      
      :::

    * <Cmdname sid="layout" className="uitext" />

      * 위젯을 드래그하여 위치를 이동합니다.

      * 위젯을 삭제하려면 위젯 오른쪽 위에 ![삭제 아이콘](/img/ico-trash.svg) 아이콘을 클릭하세요.

    * <Cmdname sid="widget" className="uitext" /><UI>.json</UI>: 대시보드의 위젯 설정을 JSON 형식으로 불러올 수 있습니다. ![복사 아이콘](/img/ico-copy.svg) 아이콘을 클릭하면 JSON 내용을 복사합니다.

    * <Cmdname sid="option" className="uitext" /><UI>.json</UI>: 대시보드에 설정된 옵션을 JSON 형식으로 불러올 수 있습니다. ![복사 아이콘](/img/ico-copy.svg) 아이콘을 클릭하면 JSON 내용을 복사합니다.

* 대시보드를 공유받은 사용자가 대시보드를 삭제하면, 해당 사용자의 <Cmdname sid="dashboard_list" className="uitext" />에서만 삭제됩니다. 원본 대시보드는 유지됩니다.

:::

## Flex 보드 공유하기 {#shareflexboard}

:::caution

Flex 보드 공유 기능은 **홈 화면 > <Cmdname sid="integratedFlexboard" className="uitext" />** 메뉴에서만 사용할 수 있습니다.  
프로젝트의 <Cmdname sid="flexboard" className="uitext" /> 메뉴에서는 제공되지 않습니다.

:::

:::info

 **공유와 복사의 차이**

![공유와 복사의 차이](/img/flexboard-compare-sharing-copy.png)

* 공유: 공유한 멤버는 모두 같은 대시보드를 볼 수 있습니다.

* 복사: 대시보드를 다른 멤버 또는 나에게 복사하면, 복사된 별도의 대시보드를 볼 수 있습니다.

* <Cmdname sid="read_only" className="uitext" />로 공유/복사받은 대시보드는 수정할 수 없습니다. 단, **수정 모드**로 공유/복사된 경우는 수정할 수 있습니다.

* <Cmdname sid="read_only" className="uitext" />로 공유받은 멤버도 해당 대시보드를 **읽기 전용으로 재공유**하거나 **복사**할 수 있습니다.

* 대시보드에 포함된 프로젝트 중 **최소 1개** 프로젝트에 초대된 멤버에게만 공유할 수 있습니다. 공유 또는 복사 받을 멤버의 프로젝트 소속 여부를 확인하세요.

    <ImgLang img='flexboard-check-projects.png' desc='대시보드 목록' />

:::

<Cmdname sid="integratedFlexboard" className="uitext" />에서 생성한 Flex 보드를 다른 멤버와 공유할 수 있습니다. 소유자가 공유된 대시보드를 수정하면, 공유받은 모든 멤버에게 동일하게 반영됩니다.

1. <Cmdname sid="integratedFlexboard" className="uitext" />의 <Cmdname sid="dashboard_list" className="uitext" />에서 공유할 대시보드의 <LinkImage img="ico-flexboard-share.svg" className="ico" /> 아이콘을 클릭하세요.

1. <Cmdname sid="share_dashboard" className="uitext" /> 창이 나타나면 입력창에 공유할 멤버를 한 명 또는 여러명을 선택하거나 계정을 직접 입력하세요. 

    * <Cmdname sid="read_only" className="uitext" />를 선택하면 공유받은 멤버는 대시보드를 수정할 수 없습니다.

    * 공유 대상은 대시보드에 포함된 프로젝트 중 최소 1개 프로젝트에 소속되어 있어야 합니다.

1. <Cmdname sid="BTN09441" className="uitext" /> 버튼을 클릭하세요.

    * 공유된 항목은 <Cmdname sid="shared" className="uitext" /> 태그가 표시됩니다. <Cmdname sid="shared" className="uitext" /> 태그에 마우스를 올리면 공유한 멤버 정보를 확인할 수 있습니다. 
    
    * 읽기 전용으로 공유한 항목은 <Cmdname sid="read_only" className="uitext" /> 태그가 표시됩니다.

    | 대시보드를 공유받은 경우 | 대시보드를 공유한 경우(대시보드 소유자) |
    | :----------------: | :---------------------------: |
    | <ImgLang img="flexboard-share-04.png" desc="Flex 보드 공유" /> | <ImgLang img="flexboard-share-05.png" desc="Flex 보드 공유" /> |

    :::tip

    **대시보드 공유 조건**

    * 대시보드 공유 기능은 **홈 화면** &gt; <Cmdname sid="integratedFlexboard" className="uitext" />에서만 제공합니다.

    * 누구나 자신의 대시보드를 공유할 수 있으며, 공유받은 대시보드를 다시 공유하거나 복사할 수 있습니다.

    * <Cmdname sid="read_only" className="uitext" />로 공유받은 대시보드는 수정할 수 없지만, **수정 모드**로 공유받은 대시보드는 수정할 수 있습니다.

    * **소유자**가 대시보드를 삭제하면, 공유받은 모든 계정에서 해당 대시보드가 삭제됩니다.

    * **공유받은 사용자**가 삭제하면, 해당 사용자의 <Cmdname sid="dashboard_list" className="uitext" />에서만 삭제되며 원본은 유지됩니다.

    * 공유받은 대시보드의 프로젝트 중 <Cmdname sid="view_and_analysis" className="b600" /> 권한이 없는 프로젝트는 데이터를 조회할 수 없습니다.

    :::

        c. 해당 멤버의 <Cmdname sid="dashboard_list" className="uitext" />에 `_copied` 접미어가 붙은 항목이 추가됩니다.

### JSON 파일로 공유하기 {#share-flexboard-as-json}

Flex 보드 설정을 JSON 파일로 다른 멤버에게 전달하거나 다른 멤버의 설정을 가져올 수 있습니다.

#### 내보내기

1. <Cmdname sid="integratedFlexboard" className="uitext" /> 또는 프로젝트의 <Cmdname sid="flexboard" className="uitext" /> 메뉴로 이동하세요.

1. <Cmdname sid="dashboard_list" className="uitext" />에서 <LinkImage img="ico-export.svg" desc="내보내기 아이콘" className="ico" /> 버튼을 클릭하세요.

1. 내보내기 옵션을 선택하세요.

    * <Cmdname sid="BTN07940" className="uitext" />: 대시보드에 설정된 프로젝트 정보를 JSON 파일에 포함해 공유

    * <Cmdname sid="BTN07941" className="uitext" />: 대시보드에 설정된 프로젝트 정보를 JSON 파일에 포함하지 않고 공유

1. JSON 파일을 다운로드하고 공유할 멤버에게 전달하세요.

#### 가져오기

1. <Cmdname sid="integratedFlexboard" className="uitext" /> 또는 프로젝트의 <Cmdname sid="flexboard" className="uitext" /> 메뉴로 이동하세요.

1. 화면 오른쪽 위 ![가져오기 아이콘](/img/ico-import.svg) <Cmdname sid="import" className="uitext" /> 버튼을 클릭하세요.

1. 다운로드한 JSON 파일을 선택하세요.

:::note

이 기능은 프로젝트의 **수정** 권한을 소유한 멤버만 이용할 수 있습니다.

:::

### 모드 종류 

<Cmdname sid="side_flexboard" className="uitext" />는 화면 모드에 따라 설정할 수 있는 기능이 다릅니다.

* <Cmdname sid="edit_mode" className="uitext" />: 위젯 템플릿 목록에서 Flex 보드의 레이아웃에 위젯을 배치하거나 위젯을 수정, 삭제, 위치 조정할 수 있습니다.

* <Cmdname sid="admin" className="uitext" />: 위젯의 세부 속성을 json 형태로 관리할 수 있습니다. 레이아웃에 배치한 위젯의 오른쪽 위에 위치한 ![코드 아이콘](/img/ico-custom-code.svg) 버튼을 선택하세요.

* <Cmdname sid="view_mode" className="uitext" />: 위젯 배치, 크기 조절, 세부 옵션 등의 설정을 완료한 다음 대시보드와 같이 각 위젯의 데이터를 실시간으로 모니터링할 수 있습니다.

#### 데이터 필터링 {#Filtering-data}

데이터 필터링을 통해 모니터링 대상을 간추려 효율적인 Flex 보드를 구성할 수 있습니다.

<ImgLang img='flexboard-filter.png' desc='Flex 보드 - 필터' />

1. 화면 위 ![필터 아이콘](/img/ico-filter.svg) <Cmdname sid="filter" className="uitext" /> 탭을 클릭하세요. 

1. 화면 왼쪽에 나타나는 <Cmdname sid="filter" className="uitext" /> 창에서 필터링 대상을 선택하세요. 필터링 대상을 모두 선택하려면 <Cmdname sid="select_all" className="uitext" />을 선택하세요.

    * <Cmdname sid="agent" className="uitext" /> 옵션: 에이전트 종류 또는 애플리케이션 목록 선택, <Cmdname sid="search" className="uitext" /> 기능 제공

    * <Cmdname sid="metrics" className="uitext" /> 옵션: 프로젝트 또는 에이전트 유형 선택, <UI>+ And</UI> 조건 추가하거나 <Cmdname sid="contain" className="uitext" />, <Cmdname sid="exclude" className="uitext" />로 문자열 기반 자동 선택 가능 

1. 필터링 대상을 선택했다면 <Cmdname sid="apply" className="uitext" /> 버튼을 클릭하세요. 

1. 자주 사용하는 조건은 <Cmdname sid="save_filter" className="uitext" /> 버튼으로 저장하세요. 
    
    a. 필터 이름을 입력하고 <Cmdname sid="save" className="uitext" />을 선택하세요.  <Cmdname sid="saved_filter" className="uitext" />는 <Cmdname sid="filter" className="uitext" /> 창 하단에 나타납니다.

##### 필터링 조건 {#filtering conditions}

필터링 조건은 에이전트 타입과 메트릭스 타입으로 구분합니다.  
에이전트는 기본 모니터링 대상만 선택 옵션으로 제공하며, 메트릭스는 프로젝트에서 수집한 모든 메트릭스 데이터의 태그를 선택 옵션으로 제공합니다. 메트릭스 타입은 에이전트가 아닌 형태로 데이터를 수집한 후 필터링을 적용할 때 유용합니다. 메트릭스에 대한 자세한 내용은 [다음 문서](metrics-intro)를 참조하세요.

| 타입 | 에이전트 | 메트릭스 |
| :---: | -------- | -------- |
| 선택 옵션 | 와탭 프로젝트 모니터링 대상<br/>(애플리케이션, 서버, 데이터베이스,<br/>컨테이너, 쿠버네티스 POD와 NODE) | 에이전트를 포함한 모든 메트릭스 태그 |
| 적용 범위 | 모든 위젯 | 메트릭스 데이터를 사용하는 모든 위젯 |
| 필터링 조건 | 체크박스 | <Cmdname sid="equal" className="uitext" />, <Cmdname sid="contain" className="uitext" />, <Cmdname sid="exclude" className="uitext" /> 방식 |

#### 인쇄하기 {#printing}

Flex 보드 화면을 PDF로 저장해 인쇄할 수 있습니다. 

1. Flex 보드 화면의 오른쪽 위 ![인쇄 아이콘](/img/ico-print.svg) 아이콘을 클릭해 인쇄 모드로 전환합니다. 

1. 옵션을 설정합니다.

    * <Cmdname sid="filter_visible_msg" className="uitext" />: 화면에 필터 옵션 켜거나 끄기 
    * <Cmdname sid="change_landscape_mode_msg" className="uitext" replace="horizontal" /> / <Cmdname sid="change_landscape_mode_msg" className="uitext" replace="vertical" />: 화면을 가로 또는 세로로 변환

1. 완료 후 <UI>PDF</UI><Cmdname sid="download" className="uitext" />를 선택하세요.

1. 인쇄 모드를 종료하려면 오른쪽 위 ![종료 아이콘](/img/ico-close.svg) <Cmdname sid="close_print_mode" className="uitext" />를 클릭하세요.

#### Flex 보드 관리 {#flex-board-management}

Flex 보드의 레이아웃을 변경하거나 위젯의 위치 변경, 크기 조절, 이름 변경 등의 관리 기능을 제공합니다.
<Cmdname sid="flexboard_management" className="uitext" /> 기능은 <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_flexboard" className="uitext" /> 메뉴의 <Cmdname sid="dashboard_list" className="uitext" />에서 ![편집 아이콘](/img/ico-edit.svg) 아이콘을 클릭해 진입할 수도 있습니다.

1. Flex 보드 화면의 오른쪽 위 ![설정 아이콘](/img/ico-setting.svg) 아이콘을 클릭하면 <Cmdname sid="flexboard_management" className="uitext" /> 창이 열립니다. 

    * <Cmdname sid="rename" className="uitext" />: Flext 보드의 이름 변경

    * <Cmdname sid="layout" className="uitext" />: 위젯의 위치 및 크기조절 및 삭제

    * <Cmdname sid="dont_allow_users_to_modify" className="uitext" />: 수정 제한 옵션

        * 수정 권한이 있는 멤버는 수정, 삭제, 편집할 수 있으며 옵션을 활성화하면 <Cmdname sid="dont_allow_users_to_modify" className="uitext" /><UI>: On</UI>으로 변경됨

    * <Cmdname sid="widget" className="uitext" /><UI>.json</UI>: Flex 보드에 포함된 위젯을 JSON 형태로 편집 및 관리

    * <Cmdname sid="option" className="uitext" /><UI>.json</UI>: Flex 보드의 설정을 JSON 형태로 편집 및 관리(프로젝트 정보, Flex 보드 필터, 시간 선택 등 설정값 편집)

        :::caution

        <Cmdname sid="widget" className="uitext" /><UI>.json</UI>, <Cmdname sid="option" className="uitext" /><UI>.json</UI> 기능은 대시보드 관리자 권한의 사용자에게만 제공합니다.

        :::

1. 모든 설정을 완료하고 <Cmdname sid="save" className="uitext" /> 버튼을 클릭하세요.
