## 화면 모드

<Cmdname sid="side_flexboard" className="uitext" />는 다음과 같은 화면 모드를 제공합니다. 화면 모드에 따라 설정할 수 있는 기능이 다릅니다.

* <Cmdname sid="edit_mode" className="uitext" />: 위젯 템플릿 목록에서 Flex 보드의 레이아웃에 위젯을 배치하거나 위젯을 수정, 삭제, 위치 조정할 수 있습니다.

* <Cmdname sid="admin" className="uitext" />: 위젯의 세부 속성을 json 형태로 관리할 수 있습니다. 레이아웃에 배치한 위젯의 오른쪽 위에 위치한 ![코드 아이콘](/img/ico-custom-code.svg) 버튼을 선택하세요.

* <Cmdname sid="view_mode" className="uitext" />: 위젯 배치, 크기 조절, 세부 옵션 등의 설정을 완료한 다음 대시보드와 같이 각 위젯의 데이터를 실시간으로 모니터링할 수 있습니다.

<Cmdname sid="side_flexboard" className="uitext" />에서는 다음의 공통 기능을 사용할 수 있습니다.

* 데이터 필터링
* 실시간 데이터 조회 범위 설정
* 인쇄 모드
* Flex 보드 관리
* 데이터 병합 옵션

## 데이터 필터링

Flex 보드는 수백 가지의 종류의 데이터를 다룹니다. 데이터 필터링 기능을 이용해 모니터링하려는 대상을 간추려 Flex 보드를 구성하세요.

<ImgLang img='flexboard-filter.png' desc='Flex 보드 - 필터' />

1. 화면 위에 ![지시선 1](/img/number-01.png) ![필터 아이콘](/img/ico-filter.svg) <Cmdname sid="filter" className="uitext" />를 선택하세요. 

1. 화면 왼쪽으로 ![지시선 2](/img/number-02.png) <Cmdname sid="filter" className="uitext" /> 창이 나타납니다.

    * <Cmdname sid="agent" className="uitext" /> 옵션을 선택했다면 에이전트 종류 또는 애플리케이션 목록 기준으로 필터링 대상을 선택할 수 있습니다. 필터링 대상을 모두 선택하려면 <Cmdname sid="select_all" className="uitext" />을 선택하세요. 필요한 경우 <Cmdname sid="search" className="uitext" /> 기능을 이용해 선택할 수 있습니다.

    * <Cmdname sid="metrics" className="uitext" /> 옵션을 선택했다면 <Cmdname sid="category" className="uitext" /> 옵션에서 프로젝트, 에이전트 유형에 따라 필터링 대상을 선택할 수 있습니다. <Cmdname sid="category" className="uitext" /> 옵션은 <UI>+ And</UI>를 선택해 추가할 수 있습니다. <Cmdname sid="contain" className="uitext" />, <Cmdname sid="exclude" className="uitext" /> 옵션을 선택하면 문자열로 필터링 대상을 자동 선택합니다.

1. 필터링 대상 선택을 완료했다면 <Cmdname sid="apply" className="uitext" /> 버튼을 선택하세요. 

사용자가 설정한 필터링 설정을 저장해 이용하려면 <Cmdname sid="save_filter" className="uitext" /> 버튼을 선택하세요. 필터 이름을 입력하고 <Cmdname sid="save" className="uitext" />을 선택하세요. ![지시선 3](/img/number-03.png) <Cmdname sid="saved_filter" className="uitext" />는 <Cmdname sid="filter" className="uitext" /> 창 하단에 나타납니다.

필터링 조건은 에이전트 타입과 메트립스 타입으로 구분합니다.

| 타입 | 에이전트 | 메트릭스 |
| ---- | -------- | -------- |
| 선택 옵션 | 와탭 프로젝트 모니터링 대상<br/>(애플리케이션, 서버, 데이터베이스,<br/>컨테이너 및 쿠버네티스 POD와 NODE) | 에이전트를 포함한 모든 메트릭스 태그 |
| 적용 범위 | 모든 위젯 | 메트릭스 데이터를 사용하는 모든 위젯 |
| 필터링 조건 | 체크박스 | <Cmdname sid="equal" className="uitext" />, <Cmdname sid="contain" className="uitext" />, <Cmdname sid="exclude" className="uitext" /> 방식 |

:::note

* 에이전트 타입은 기본적인 모니터링 대상만을 선택 옵션으로 제공합니다.

* 메트릭스 타입은 프로젝트에서 수집 중인 모든 메트릭스 데이터의 태그를 선택 옵션으로 제공합니다. 메트릭스 타입은 클라우드 모니터링과 같이 에이전트가 아닌 형태로 데이터를 수집한 후 필터링을 적용할 때 유용합니다.

* 메트릭스에 대한 자세한 내용은 [다음 문서](metrics-intro)를 참조하세요.

:::

## 인쇄하기

Flex 보드 화면을 PDF로 저장해 인쇄할 수 있습니다. Flex 보드 화면의 오른쪽 위에 ![인쇄 아이콘](/img/ico-print.svg) 버튼을 선택하세요. 화면을 인쇄 모드로 전환합니다. 다음의 옵션을 설정한 다음 <UI>PDF</UI><Cmdname sid="download" className="uitext" />를 선택하세요.

* <Cmdname sid="filter_visible_msg" className="uitext" />: 화면에 필터 옵션을 켜거나 끌 수 있습니다.
* <Cmdname sid="change_landscape_mode_msg" className="uitext" replace="horizontal" /> / <Cmdname sid="change_landscape_mode_msg" className="uitext" replace="vertical" />: 화면을 가로 또는 세로로 변환할 수 있습니다.

인쇄 모드를 종료하려면 오른쪽 위에 ![종료 아이콘](/img/ico-close.svg) <Cmdname sid="close_print_mode" className="uitext" />를 선택하세요.

## Flex 보드 관리

Flex 보드의 레이아웃을 변경하거나 위젯의 위치 변경, 크기 조절, 이름 변경 등의 관리 기능을 제공합니다. Flex 보드 화면의 오른쪽 위에 ![설정 아이콘](/img/ico-setting.svg) 버튼을 선택하세요. <Cmdname sid="flexboard_management" className="uitext" /> 창이 나타납니다. 

* <Cmdname sid="rename" className="uitext" />: Flext 보드의 이름을 변경할 수 있습니다. 변경하려는 이름을 입력하세요.

* <Cmdname sid="layout" className="uitext" />: 위젯의 배치를 변경하거나 위치 및 크기를 조절할 수 있습니다. 배치한 위젯을 삭제하려면 ![삭제 아이콘](/img/ico-trash.svg) 버튼을 클릭하세요.

* <Cmdname sid="dont_allow_users_to_modify" className="uitext" />: 다른 사용자가 해당 Flex 보드의 수정을 할 수 없도록 설정합니다. 수정 권한이 있는 멤버만 수정 또는 삭제, 위젯을 편집할 수 있습니다. 버튼을 클릭하면 <Cmdname sid="dont_allow_users_to_modify" className="uitext" /><UI>: On</UI>으로 변경됩니다.

* <Cmdname sid="widget" className="uitext" /><UI>.json</UI>: Flex 보드에 포함된 위젯을 json 형태로 편집해 관리할 수 있습니다.

* <Cmdname sid="option" className="uitext" /><UI>.json</UI>: Flex 보드의 설정을 json 형태로 편집해 관리할 수 있습니다. 프로젝트 정보, Flex 보드 필터, 시간 선택 등의 설정값을 편집할 수 있습니다.

모든 설정을 완료한 다음에는 화면 오른쪽 아래에 <Cmdname sid="save" className="uitext" /> 버튼을 클릭하세요.

:::note

* <Cmdname sid="widget" className="uitext" /><UI>.json</UI>, <Cmdname sid="option" className="uitext" /><UI>.json</UI> 기능은 대시보드 관리자 권한의 사용자에게만 제공합니다.

* <Cmdname sid="flexboard_management" className="uitext" /> 기능은 <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_flexboard" className="uitext" /> 메뉴의 <Cmdname sid="dashboard_list" className="uitext" />에서 ![편집 아이콘](/img/ico-edit.svg) 버튼을 선택해 진입할 수도 있습니다.

:::
