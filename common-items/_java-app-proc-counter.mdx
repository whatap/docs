### app_proc_counter
----

애플리케이션 프로세스 지표입니다.

* 수집 간격 : 5초
* 통계 데이터 : 5분, 1시간

|----|----|----|

|containerKey|컨테이너 키 값|고유값|
|host_ip|Host IP|고유값|
|okindName|애플리케이션 종류명|-|
|oname|애플리케이션 이름|고유값|

|type|애플리케이션 유형|언어 이름|
