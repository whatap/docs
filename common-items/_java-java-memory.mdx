## java_memory {#javamemory}

애플리케이션이 실행되는 동안 JVM의 메모리 사용 상태에 대한 지표를 수집한 데이터의 카테고리입니다. 

* 수집 간격: 5초
* 통계 데이터: 5분, 1시간

### Tags {#javamemory-tags}

<div className="c3-30-60-10">

|태그명|설명|비고|
|----|----|----|
|`host_ip`|호스트 IP 주소|고유값|
|`okindName`|`whatap.okind`로 분류된 그룹 이름|-|
|`oname`|에이전트 이름|고유값|
|`onodeName`|`whatap.onode`로 분류된 그룹 이름|-|
|`pid`|특정프로세스의 고유 식별자(ID)|-|

</div>

### Fields {#javamemory-fields}
