## 로그 태그 옵션 {#log-tag-options}

로그 태그 선택 시 다음과 같이 드롭다운 메뉴가 나타납니다. <Cmdname sid="search" className="uitext" />, <Cmdname sid="exclusion_search" className="uitext" />, <Cmdname sid="TTL06582" className="uitext" /> 옵션을 확인할 수 있습니다. 

<ImgLang img='log-search-tag.png' desc='로그 태그 드롭다운 메뉴'/> 

* **검색**

  <Cmdname sid="search" className="uitext" /> 옵션을 선택하면 <Cmdname sid="filter" className="uitext" />에 해당 태그가 포함('==') 조건으로 입력됩니다.

* **제외 검색**

  <Cmdname sid="exclusion_search" className="uitext" /> 옵션을 선택하면 <Cmdname sid="filter" className="uitext" />에 해당 태그가 제외('!=') 조건으로 입력됩니다.

* **인접 로그**

  <Cmdname sid="TTL06582" className="uitext" /> 옵션을 선택하면 <Cmdname sid="TTL06582" className="uitext" /> 상세 창이 나타납니다. 선택한 로그의 서버를 대상으로 선택한 로그와 인접한 시간대의 로그를 조회합니다. <Cmdname sid="time_selector" className="uitext" /> 버튼을 클릭해 인접한 시간대의 로그를 조회할 수 있습니다. 기준 로그는 파란색 바탕으로 표시됩니다.

  <ImgLang img='log-search-adjacent-log.png' desc='로그 태그 인접 로그'/>
