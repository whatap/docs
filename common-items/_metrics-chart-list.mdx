:::note 

**데이터 병합**

데이터 병합은 ***시간 병합***과 ***오브젝트 병합***을 제공합니다. 

* ***시간 병합***은 원본 데이터에서 필드 값이 같은 데이터끼리 일정한 간격으로 데이터를 병합합니다. 

* ***오브젝트 병합***은 서로 다른 필드 값을 가진 데이터들 중에서 태그가 일치하는 경우 해당 데이터를 병합합니다.

:::
