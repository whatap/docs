| <Cmdname sid="monthly_active_users" className="b600" report/> (**MAU**, Monthly Active User) | 한달간 애플리케이션을 한 번 이상 사용한 고유 사용자 수입니다. |
| <Cmdname sid="_realtime_user" className="b600" report/> | 중복을 제거한 동시 접속 사용자 수입니다. |

| <Cmdname sid="daily_hitmap" className="b600" report/> | 하루 동안 시간대별 트랜잭션 분포와 에러 발생 상황을 한눈에 파악할 수 있는 분포도 차트입니다. |
| <Cmdname sid="error_ratio" className="b600" report/> | 전체 트랜잭션 건수 중 에러 비율입니다. |
| <Cmdname sid="event_count" className="b600" report/> | 메트릭스 이벤트 중 <span class="vslow">Critical</span> 레벨인 이벤트의 수입니다. |
| <Cmdname sid="error_cnt" className="b600" report/> | 트랜잭션 처리 중 발생한 오류로 인해 실패한 트랜잭션 수입니다. |
| <Cmdname sid="application_count" className="b600" report/> | 조회 기간 모니터링한 애플리케이션 수(에이전트 수)입니다. |
| <Cmdname sid="total_core" className="b600" report/> | 조회 기간 사용된 CPU 코어 수입니다. |
| <b>APDEX\*100**</b> (Application Performance Index) | 조회 기간 동안 애플리케이션에 대한 고객 성능 만족도입니다. |
| <Cmdname sid="tps" className="b600" report/> (**TPS**, Transaction Per Second) | 초당 처리된 트랜잭션의 수입니다. |
| <Cmdname sid="response_avg" className="b600" report/> / <Cmdname sid="response_time" className="b600" report/> | 트랜잭션에 대한 평균 응답 시간입니다. |
| <Cmdname sid="cpu_core" className="b600" report/> | 사용된 CPU 코어 수입니다. |
| <Cmdname sid="visitor_per_hour" className="b600" report/> | 조회 기간의 시간당 방문자 수입니다. |
| <Cmdname sid="cpu_trend" className="b600" report/> | 서버의 CPU 사용률입니다. |
| <Cmdname sid="mem_sys" className="b600" report/> | 서버의 메모리 사용률입니다. |
| <Cmdname sid="active_cnt" className="b600" report/> / <Cmdname sid="active_tx_cnt" className="b600" report/> | 진행 중인 트랜잭션 건수입니다. |
| <Cmdname sid="_cpu" className="b600" report/> | CPU 사용률입니다. |
| <Cmdname sid="heap_min" className="b600" report/> | 최소 힙 메모리 사용량입니다. |
| <Cmdname sid="operating_rate" className="b600" report/> | 애플리케이션이 주어진 시간 동안 정상적으로 동작한 시간을 백분율로 나타낸 것입니다. |
