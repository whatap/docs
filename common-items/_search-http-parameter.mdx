### HTTP 파라미터 조회 {#view-http-parameter}

<Cmdname sid="TTL06126" className="uitext" /> 탭에서 해당 트랜잭션의 HTTP 파라미터를 조회할 수 있습니다. 

1. 페이지 아래로 스크롤해 트랜잭션 수행의 가장 마지막 단계로 이동하세요. 

1. ![잠금 아이콘](/img/ic-lock.svg) <Cmdname sid="TTL06520" className="uitext" /> 항목을 선택하세요.

1. <UI>HTTP-PARAMETERS</UI> 창이 나타나면 <Cmdname sid="BTN06764" className="uitext" /> 버튼을 선택하세요.

1. 설정한 Param Key를 입력하세요.

가려진 매개 변수를 확인할 수 있습니다.

<Xclude product='java,kubernetes,php'>

:::note

* HTTP 파라미터와 관련한 에이전트 설정은 [다음 문서](agent-transaction#profile_http_parameter)를 참조하세요.

* <Cmdname sid="BTN06764" className="uitext" />: 복호화된 파라미터 값을 확인할 수 있습니다. 비밀번호는 <em><code>WHATAP_HOME</code>/paramkey.txt</em> 파일 내 6자리 문자열입니다. 다른 문자열로 변경 가능합니다. <em>paramkey.txt</em> 내 키는 SQL 변수 조회, HTTP 쿼리 조회, Thread 중지에 필요합니다.

:::
