<Title hashid='setting-table-contents' level={props.level}>테이블 콘텐츠 표시 설정</Title>

<InDoc product="browser">

브라우저 에러 목록에 표시하는 콘텐츠의 내용을 설정할 수 있습니다. 너무 많은 정보가 표시되어 보기 불편하거나 필요한 내용을 추가하고 싶은 경우 이용할 수 있습니다.

</InDoc>

1. 테이블 오른쪽 위에 ![설정 아이콘](/img/ico-setting.svg) 버튼을 선택하세요.

1. <Cmdname sid="ITM08349" className="uitext" /> 창이 나타나면 원하는 옵션을 설정하세요.

    * <UI>content</UI> 또는 <Cmdname sid="tag" className="uitext" /> 옵션을 선택해 로그의 콘텐츠 표시 여부를 선택하세요.

    * <Cmdname sid="tag_management" className="uitext" />: 태그는 추가한 순서대로 로그에 표시됩니다. 태그의 순서는 드래그로 변경할 수 있습니다. 비활성화된 태그는 로그에서 보이지 않습니다.

1. 모든 설정을 완료한 다음 ![닫기 아이콘](/img/ico-close.svg) 버튼을 선택하세요.

설정한 내용이 테이블에 적용됩니다.

:::note

<Cmdname sid="ITM08349" className="uitext" />에 대한 자세한 내용은 [다음 문서](../log/log-search#set-table)를 참조하세요.

:::
