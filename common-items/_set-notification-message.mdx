홈 화면 &gt; 프로젝트 선택 &gt; <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventSetting" className="uitext" /> &gt; <Cmdname sid="metrics" className="uitext" /> 탭 선택

프로젝트 멤버에게 메일 또는 서드 파티 플러그인을 통해 전달하는 경고 알림 메시지를 수정할 수 있는 사용자 정의 기능을 제공합니다. 화면 오른쪽 위에 <Cmdname sid="goto_alertTemplate" className="uitext" /> 버튼을 선택하세요. 수신할 메시지에서 불필요하다고 생각되는 항목을 감추거나 사용자가 원하는 특별한 내용을 추가할 수 있습니다.

<ImgLang img='notification-set-message.png' desc='알림 메시지 설정' />

:::note

* 이 기능은 <Cmdname sid="metrics" className="b600" />, <Cmdname sid="composite_metrics" className="b600" /> 경고 알림에만 적용할 수 있습니다.

* 이 기능은 프로젝트 선택 &gt; <Cmdname sid="side_lab" className="uitext" /> &gt; <Cmdname sid="side_alertTemplate" className="uitext" /> 메뉴 경로로도 진입할 수 있습니다.

* 이 기능은 <Cmdname sid="sms_alert_settings" className="b600" /> 권한을 가진 멤버가 설정할 수 있습니다. 자세한 내용은 [다음 문서](../project/project-structure#project-auth)를 참조하세요.

:::


## 기본 화면 안내

<ImgLang img='notification-set-message-main.png' desc='알림 메시지 설정' />

* 알림 메시지의 종류는 <Cmdname sid="OCCURRED" className="occur set-noti" />과 <Cmdname sid="SOLVED" className="resolve set-noti" />로 분류할 수 있습니다. <Cmdname sid="OCCURRED" className="occur set-noti" />은 이벤트가 발생했을 때 전달되는 알림이며, <Cmdname sid="SOLVED" className="resolve set-noti" />은 이벤트가 해소되었을 때 전달되는 알림입니다.

* 수신한 메시지의 상세 항목을 감추려면 ![숫자 1](/img/number-01.png) 토글 버튼을 선택해 비활성화하세요.

* 수신한 메시지의 내용을 수정하려면 ![숫자 2](/img/number-02.png) ![편집 아이콘](/img/ico-edit.svg) <Cmdname sid="edit" className="uitext" /> 버튼을 선택하세요.

* ![숫자 3](/img/number-03.png) 영역은 수신할 메시지의 상세 항목입니다. ![숫자 4](/img/number-04.png) 영역은 수신할 메시지의 상세 항목에 대한 내용입니다. ![숫자 5](/img/number-05.png) ![미리보기 아이콘](/img/ico-noti.svg) <Cmdname sid="preview" className="uitext" /> 버튼을 선택하면 수신할 메시지의 내용을 미리 확인할 수 있습니다.


## 수신 메시지 상세 항목 감추기

수신 메시지의 내용 중 불필요하다고 생각되는 항목을 감출 수 있습니다. 각 상세 항목의 가장 오른쪽의 토글 버튼을 선택해 해당 항목을 수신 메시지에서 감추거나 보이도록 설정할 수 있습니다. ![미리보기 아이콘](/img/ico-noti.svg) <Cmdname sid="preview" className="uitext" /> 버튼을 선택해 수신할 메시지의 내용을 미리 확인할 수 있습니다.

* 상세 항목 활성화 예시

  <ImgLang img='notification-set-message-preview.png' desc='활성화 미리보기' />

* 상세 항목 비활성화 예시

  <ImgLang img='notification-set-message-preview-disabled.png' desc='비활성화 미리보기' />

변경 사항을 반영하려면 ![저장 아이콘](/img/ic-s-check.svg) <Cmdname sid="save" className="uitext" /> 버튼을 선택하세요. 다른 카테고리의 변경 사항을 한번에 모두 반영하려면 화면 오른쪽 위에 <Cmdname sid="save_all" className="uitext" /> 버튼을 선택하세요.

:::note

* 감춘 상세 항목을 수신 메시지에 다시 추가하려면 해당 항목의 토글 버튼을 선택해 활성화하세요.

* 모든 상세 항목을 수신 메시지에 추가하려면 ![활성화 아이콘](/img/ic-power.svg) <Cmdname sid="active_all" className="uitext" /> 버튼을 선택하세요.

:::


## 상세 항목 내용 수정하기

상세 항목의 내용에 사용자가 원하는 특별한 내용을 추가할 수 있습니다. 

1. 각 상세 항목 가장 오른쪽에 ![편집 아이콘](/img/ico-edit.svg) <Cmdname sid="edit" className="uitext" /> 버튼을 선택하세요.

1. <Cmdname sid="edit_message" className="uitext" /> 창이 나타나면 추가할 내용을 입력하세요.

    <ImgLang img='notification-set-message-edit.png' desc='메시지 편집' />

1. <Cmdname sid="confirm" className="uitext" /> 버튼을 선택하세요.

1. ![미리보기 아이콘](/img/ico-noti.svg) <Cmdname sid="preview" className="uitext" /> 버튼을 선택해 수신할 메시지의 내용을 미리 확인하세요.

    <ImgLang img='notification-set-message-edit-preview.png' desc='메시지 편집' />

변경 사항을 반영하려면 ![저장 아이콘](/img/ic-s-check.svg) <Cmdname sid="save" className="uitext" /> 버튼을 선택하세요. 다른 카테고리의 변경 사항을 한번에 모두 반영하려면 화면 오른쪽 위에 <Cmdname sid="save_all" className="uitext" /> 버튼을 선택하세요.

:::note

<Cmdname sid="edit_message" className="uitext" /> 창에서 텍스트 블록은 알림 발생 상황에 따라 변경되어 출력되는 변수입니다. 텍스트 블록을 드래그해서 위치를 변경할 수 있습니다. 텍스트 블록을 삭제했다면 창 아래에 텍스트 블록 버튼을 선택해 다시 추가할 수 있습니다.

<ImgLang img='notification-set-message-edit-block.png' desc='메시지 편집' />

:::

:::caution

사용자에 의해 내용을 수정하기 전의 메시지는 사용자의 언어 상태에 따라 번역되어 전송되나, 내용 수정 이후에는 언어 상태와 관계 없이 사용자 정의된 포맷으로만 알림을 전송합니다.

:::


## 다른 프로젝트에 설정 복사하기

사용자가 설정한 내용을 다른 프로젝트에 복사할 수 있습니다. 

1. 화면 오른쪽 위에 <Cmdname sid="apply_to_other_project" className="uitext" /> 버튼을 선택하세요.

1. <Cmdname sid="apply_project" className="uitext" /> 창이 나타면 설정 내용을 복사할 프로젝트를 선택하세요. 모든 프로젝트를 선택하려면 <Cmdname sid="select_all" className="uitext" />을 선택하세요.

1. <Cmdname sid="apply" className="uitext" /> 버튼을 선택하세요.

:::info

현재 프로젝트와 동일한 제품 타입만 선택할 수 있습니다.

:::


## 초기화하기

사용자가 설정한 모든 변경 사항을 초기화하려면 ![초기화 아이콘](/img/ico-reset.svg) <Cmdname sid="initialized" className="uitext" /> 버튼을 선택하세요. 모든 카테고리의 변경 사항을 초기화하려면 화면 오른쪽 위에 <Cmdname sid="alerttemplate_reset_all" className="uitext" /> 버튼을 선택하세요. 확인 메시지 창이 나타나면 <Cmdname sid="confirm" className="uitext" /> 버튼을 선택하세요.

:::caution

초기화된 설정은 복구할 수 없습니다.

:::
