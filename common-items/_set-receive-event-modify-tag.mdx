### 이벤트 수신 태그 수정 및 삭제하기

1. ***사용자별 이벤트 수신 설정*** 섹션의 사용자 목록에서 ***+ 태그 추가*** 또는 ***+***를 선택하세요.

1. ***이벤트 수신 태그*** 팝업 창이 나타나면 ***태그 목록***에서 수정 또는 삭제할 항목의 ![편집 아이콘](/img/ico-edit.svg) 버튼을 선택하세요.

1. 태그 이름을 수정하거나 색상을 변경한 다음 ***적용*** 버튼을 선택하세요.
  
  태그를 삭제하려면 ![삭제 아이콘](/img/ico-trash-red.svg) ***태그 삭제*** 버튼을 선택하세요.

:::note

이벤트에 적용 중인 ***이벤트 수신 태그*** 항목은 삭제할 수 없습니다.

:::
