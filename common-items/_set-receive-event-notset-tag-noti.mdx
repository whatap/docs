<Cmdname sid="event_tag" className="uitext" />가 설정되지 않은 경고 알림을 받으려면 <Cmdname sid="ITM05991" className="uitext" /> 옵션을 선택하세요. <Cmdname sid="event_tag" className="uitext" />가 설정된 경고 알림만 받고 싶다면 선택을 해제하세요.

:::
