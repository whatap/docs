## SpringBoot 2.2 버전 이상, Tomcat JMX가 off 상태

SpringBoot 2.2 버전 이상에서 Embedded Tomcat을 사용하는 경우, 기본값으로 JMX 관련 기능이 동작하지 않습니다. JMX 기능을 활성화하세요. [관련 문서](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.2-Release-Notes#jmx-now-disabled-by-default)를 참고하세요.

```properties title='application.properties'
spring.jmx.enabled=true
server.tomcat.mbeanregistry.enabled=true # tomcat embedded인 경우 #
```

```yml title='application.yml'
spring:
  jmx:
    enabled: true
server:
  tomcat:
    mbeanregistry:
      enabled: true #tomcat embedded인 경우
```
