토폴로지 메뉴에서 이용할 수 있는 부가 기능에 대한 설명을 제공합니다.

## 하이라이트 필터

특정 노드를 선택해 연관 관계를 가진 노드를 강조하여 보여주기 위한 기능을 제공합니다. 대량 노드 간의 복잡한 연관 관계를 토폴로지로 용이하게 확인할 수 있습니다.

<ImgLang img='topology-highlight.png' desc='하이라이트 필터' />

* 특정 노드에 마우스 오버하여 해당 노드 및 연관 노드, 링크 정보를 하이라이트로 표시합니다.
* 키워드 입력을 통해 원하는 노드의 선택을 유지할 수 있습니다. 토폴로지 갱신 시에도 하이라이트 상태는 유지합니다.

## 노드 선택 기능 이용하기

<Cmdname sid="node_selection" className="uitext" /> 버튼을 선택하세요. 화면 오른쪽에 프로젝트, DB 호출, 외부 호출 노드를 그룹핑하여 다이어그램으로 제공합니다. 

<ImgLang img='topology-select-node.png' desc='노드 선택' />

오른쪽 다이어그램에서 하위 노드를 선택하면 해당 노드를 왼쪽 토폴로지 화면에서 하이라이트 필터 적용합니다.

<ImgLang img='topology-select-node-agent.png' desc='노드 선택' />

오른쪽 화면에서는 다음 기능을 이용할 수 있습니다.

* 줌인/줌아웃
* 드래그
* 하위 노드 클릭 시 하이라이트 필터 적용
* 상위 노드 클릭 시 하위 노드 숨김 또는 표시

하이라이트 필터 기능을 취소하려면 키워드 입력란에 에이전트 이름을 삭제하세요.

:::note

<Cmdname sid="side_groupStatTopology" className="uitext" /> 메뉴에서는 화면 오른쪽 아래에 <Cmdname sid="node_selection" className="uitext" /><span class='uitext'>(</span><Cmdname sid="double_click" className="uitext" /><span class='uitext'>)</span> 버튼으로 해당 기능을 제공합니다.

:::

:::

## 노드 위치 저장 및 복원

토폴로지 화면에서 차트의 노드 위치는 마우스 왼쪽 버튼을 클릭한 상태에서 드래그해 변경할 수 있습니다. 변경한 노드의 위치는 다른 메뉴로 진입한 후 토폴로지 화면으로 다시 돌아와도 유지됩니다.

* <Cmdname sid="save_position" className="uitext" />: 사용자가 변경한 노드의 위치 정보를 저장할 수 있습니다.
* <Cmdname sid="load_position" className="uitext" />: 사용자가 위치 저장을 실행한 시점으로 노드의 위치를 복원합니다.
* <Cmdname sid="clear_position" className="uitext" />: 노드의 위치를 초기화합니다.

:::note

* 노드 위치 메뉴는 사용자가 진입한 토폴로지 메뉴에 따라 지원하지 않을 수 있습니다.
* 변경한 노드의 위치 정보는 웹 브라우저의 로컬 스토리지에 저장됩니다. 다른 사용자의 화면에서는 변경된 노드 위치가 반영되지 않습니다.

:::

## 별칭 부여 및 제거

1. <Cmdname sid="node_selection" className="uitext" /> 버튼을 선택하세요.

1. 화면 오른쪽에서 별칭을 설정할 노드로 이동해 마우스 오른쪽 버튼을 클릭하세요.

    ![별칭 입력](/img/node-custom-name.png)

1. 별칭 입력 창이 나타나면 별칭을 입력 후 엔터를 입력하세요.

별칭 정보를 저장합니다. 차트의 데이터를 새로 조회하는 시점에서 저장한 별칭을 토폴로지 화면의 노드에 적용합니다.

:::note

노드 별칭 정보는 웹 브라우저의 로컬 스토리지에 저장됩니다.

:::
