* **profile_http_header_enabled** <span class='type'>Boolean</span>

  기본값 `false`

  트레이스 내역에 http 헤더 정보를 기록하려면 `true`로 설정하세요.

* **profile_http_parameter_enabled** <span class='type'>Boolean</span><ScrollToLink anchor='profile_http_parameter' />

  기본값 `false`

  트레이스 내역에 http 파라미터 정보를 기록하려면 `true`로 설정하세요. 파라미터는 별도 보안키를 입력해야 조회할 수 있습니다.

  <InDoc product='nodejs'>

  :::note

  * **[Node.js 에이전트 0.4.95](../release-notes/nodejs/nodejs-0_4_95) 버전 미만**: 보안 키는 WAS 서버 *`${WHATAP_AGENT_HOME}`/paramkey.txt* 파일 내에 6자리로 작성합니다. *paramkey.txt* 파일이 존재하지 않는 경우 랜덤 값으로 자동 생성합니다.

  * **[Node.js 에이전트 0.4.95](../release-notes/nodejs/nodejs-0_4_95) 버전 이상**: 보안 키는 WAS 서버 *`${WHATAP_AGENT_HOME}`/security.conf* 파일 내에 `paramkey` 키값을 확인하세요. *security.conf* 파일이 존재하지 않을 경우 `paramkey` 키값을 **WHATAP**으로 자동 생성합니다.

  :::

  </InDoc>

  <InDoc product='java'>

  :::note

  * **Java 에이전트 2.2.2 버전 미만**: 보안 키는 WAS 서버 *`${WHATAP_AGENT_HOME}`/paramkey.txt* 파일 내에 6자리로 작성합니다. *paramkey.txt* 파일이 존재하지 않는 경우 랜덤 값으로 자동 생성합니다.
  
  * **Java 에이전트 2.2.2 버전 이상**: 보안 키는 WAS 서버 *`${WHATAP_AGENT_HOME}`/security.conf* 파일 내에 `paramkey` 키값을 확인하세요. *security.conf* 파일이 존재하지 않을 경우 `paramkey` 키값을 **WHATAP**으로 자동 생성합니다.

  * 보안키 설정 파일에 대한 자세한 내용은 [다음 문서](install-agent#security)를 참조하세요.

  :::
  
  </InDoc>

* **profile_http_header_url_prefix** <span class='type'>String</span>

  트레이스 내역에 http 헤더 정보를 기록할 대상 URL의 prefix를 정의할 때 사용합니다.

* **profile_http_parameter_url_prefix** <span class='type'>String</span>

  트레이스 내역에 http 파라미터 정보를 기록할 대상 URL의 prefix를 정의할 때 사용합니다.
