데이터베이스의 사용 현황을 모니터링하고 관리할 수 있는 메뉴입니다. 이 메뉴는 데이터베이스의 전체 크기와 사용량을 시각적으로 제공하며, 세부적인 사용 내역을 확인할 수 있습니다.

* 데이터베이스의 크기와 사용량 변화를 시간대별로 시각적으로 확인할 수 있습니다.

* 각 데이터베이스의 전체 크기, 사용 크기, 사용 비율을 상세히 제공합니다.

데이터베이스의 사용 현황을 실시간으로 모니터링함으로써 사용량을 확인하고 적절한 용량 관리를 할 수 있습니다. 사용량이 높은 데이터베이스를 추적해 용량 관리와 성능 최적화를 위한 조치를 취하세요. 또한 데이터베이스의 과도한 사용을 사전에 감지하고 대응할 수 있습니다.

## 기본 사용 안내

<Cmdname sid="side_dbSize" className="uitext" /> 메뉴에서는 데이터베이스의 공간 사용량과 파일 시스템에서 사용되는 실제 파일의 크기와 사용량을 확인할 수 있습니다.

1. <Cmdname sid="time" className="uitext" />에서 분석할 기간을 설정하세요.

2. <Cmdname sid="instance" className="uitext" />에서 분석할 데이터베이스 인스턴스를 선택하세요.

3. ![검색 아이콘](/img/ico-btn-search.png) 버튼을 선택하세요.

:::note

* 조회 시간은 최대 3주까지 설정할 수 있습니다. 3일 이상 조회할 경우 그래프에는 1일 평균으로 표시됩니다

* <Cmdname sid="time" className="uitext" /> 옵션 이용 방법에 대한 자세한 내용은 [다음 문서](usage-ui#timeselector)를 참조하세요.

:::

### SPACE

데이터베이스의 공간 사용량을 시간별로 시각화하여 표시합니다. PERMANENT, TEMPORARY, Active_log, Archive_log와 같은 주요 데이터베이스의 총 공간과 사용 공간, 그리고 사용 비율을 확인할 수 있습니다.

<ImgLang img='db-stat-table-space-cubrid.png' desc='Database size' />

* ***PERMANENT***: 데이터베이스에서 영구적으로 저장되는 데이터가 차지하는 공간입니다.

* ***TEMPORARY***: 임시 데이터를 저장하는 공간입니다.

* ***Active_log***: 현재 진행 중인 트랜잭션 로그를 저장하는 공간입니다.

* ***Archive_log***: 이전에 완료된 트랜잭션 로그를 저장하는 공간입니다.

### FILE

데이터베이스 파일 시스템에서 사용되는 실제 파일의 크기와 사용량을 확인할 수 있습다. INDEX, HEAP, SYSTEM, TEMP 등으로 구분된 데이터베이스 파일 시스템을 분석할 수 있습니다.

<ImgLang img='db-stat-table-space-file-cubrid.png' desc='Database size' />

* ***INDEX***: 데이터베이스 인덱스가 저장되는 파일입니다.

* ***HEAP***: 데이터베이스의 실제 테이블 데이터를 저장하는 파일입니다.니다.

* ***SYSTEM***: 데이터베이스의 시스템 관련 정보를 저장하는 파일입니다.

* ***TEMP***: 임시 데이터를 저장하는 파일입니다.

<Cmdname sid="TTL06025" className="uitext" /> 섹션에서는 시간대별로 데이터베이스의 전체 크기와 사용량 변화를 시각적으로 확인할 수 있습니다.

<ImgLang img="db-stat-table-space-check-trend-cubrid.png" desc="Database size" />

차트에서 특정 시간대의 데이터를 확인하려면 마우스를 차트 위로 오버한 상태에서 이동하세요. 해당 시간대의 상세 정보를 툴팁으로 확인할 수 있습니다.

## 사이즈 상세 확인하기 {#databasesizedetail}

<Cmdname sid="TTL06026" className="uitext" /> 섹션에서는 각 데이터베이스의 상세 정보를 제공합니다.

<ImgLang img="db-stat-table-space-size-details-cubrid.png" desc="Database size" />

최초 조회한 시간과 현재 시간을 비교하여 각 데이터베이스의 사용량 변화를 확인할 수 있습니다. 데이터베이스의 세부 내역을 확인하려면 데이터베이스 이름 옆의 ![화살표 아이콘](/img/ic-caret-right.svg) 버튼을 선택하세요.

* <UI>total</UI>: 데이터베이스 전체 크기의 합계입니다

* <UI>used</UI>: 사용 중인 데이터베이스의 크기 합계입니다.

* <UI>usage</UI>: (`used` / `total`) 🞨 100으로 계산된 사용 비율입니다.

* <UI>changed amount</UI>: 최초 조회한 시간 대비 현재 시간의 사용량 변화입니다.
