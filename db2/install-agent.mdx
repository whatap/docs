### JDBC 다운로드

DB 서버의 */`db2_instance_path`/sqllib/java* 경로에서 *db2jcc4.jar* 파일을 복사해서 */`agent_installation_path`/jdbc* 폴더에 넣어주세요.

```txt
/압축을 해제한 폴더/jdbc
```

:::note

IBM DB2의 JDBC 드라이버 구성에 대한 자세한 내용은 [다음 링크](https://www.ibm.com/docs/baw/23.x?topic=installation-configuring-jdbc-drivers)를 참조하세요.

:::
