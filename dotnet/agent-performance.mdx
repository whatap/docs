.NET 에이전트 성능 관련 옵션입니다.

:::note

* 에이전트 성능 옵션은 설정 후 whatap .NET 윈도우 서비스를 다시 시작해야 적용됩니다.
* 에이전트 성능을 확대할 경우 CPU, Memory 사용률이 증가할 수 있습니다.

:::

* **tx_max_count** <span class='type'>Int</span>

    기본값 `5000`

    트랜잭션을 최대 저장하는 개수입니다. 동시에 처리하는 트랜잭션이 많을 경우 해당 크기를 더 늘려주세요.

* **tx_default_capacity** <span class='type'>Int</span>

    기본값 `101`

    트랜잭션을 저장하는 버퍼의 시작 크기입니다.

* **tx_load_factor** <span class='type'>String(Float)</span>

    기본값 `0.75`

    트랜잭션을 저장하는 버퍼의 크기를 점차 증가시키기 위한 기준입니다. 101개에서 75% 이상 저장하면 버퍼의 크기를 두 배로 증가시킵니다.

* **queue_log_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    queue 관련 로그를 출력합니다.

* **queue_tcp_enabled** <span class='type'>Boolean</span>

    기본값 `true`

    TCP 데이터 송신 처리에 사용하는 Channel을 대신해서 별도의 Queue를 이용합니다. 데이터 우선 순위 Queue를 사용합니다.

* **queue_tcp_sender_thread_count** <span class='type'>Int</span>

    기본값 `2`

    TCP 데이터 송신을 처리하는 스레드의 개수를 설정합니다. 개수를 늘리면 데이터 송신을 좀 더 빠르게 처리할 수 있습니다. 

* **net_send_buffer_size** <span class='type'>Int</span>

    기본값 `1024`

    TCP 데이터 처리를 위한 Channel 버퍼를 설정합니다.

* **queue_udp_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    UDP 데이터 수신 처리에 Channel을 대신해서 별도의 Queue를 사용합니다.

* **queue_udp_size** <span class='type'>Int</span>

    기본값 `2048`

    UDP 데이터 수신 버퍼를 설정합니다. Channel, Queue 버퍼를 설정합니다. 

* **queue_udp_read_thread_count** <span class='type'>Int</span>

    기본값 `3`

    UDP 데이터 수신을 처리하는 스레드의 개수를 설정합니다. 개수를 늘리면 UDP 버퍼에서 좀 더 빠르게 데이터를 읽어드릴 수 있습니다. 

* **queue_udp_overflowed_size** <span class='type'>Int</span>

    기본값 `4096`

    UDP 데이터 수신에 Queue를 사용하는 경우 overflowed Queue 버퍼 크기를 설정합니다.

* **queue_udp_process_thread_count** <span class='type'>Int</span>

    기본값 `3`

    UDP 데이터를 처리하는 스레드의 개수를 설정합니다. 개수를 늘리면 UDP에서 읽은 데이터를 좀 더 빠르게 처리할 수 있습니다.

* **queue_profile_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    트레이스 정보 처리에 Channel을 대신해 별도의 Queue 를 사용합니다.

* **queue_profile_size** <span class='type'>Int</span>

    기본값 `512`

    트레이스 정보 처리를 위한 버퍼를 설정합니다.

* **queue_profile_process_thread_count** <span class='type'>Int</span>

    기본값 `1`

    트레이스 정보를 처리하는 스레드의 개수를 설정합니다. 개수를 늘리면 좀 더 빠르게 처리할 수 있습니다.

* **queue_text_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    텍스트 데이터 처리를 위해 Channel 대신에 Queue 를 사용합니다.

* **queue_text_size** <span class='type'>Int</span>

    기본값 `512`

    텍스트 데이터 처리를 위한 버퍼를 설정합니다.
