와탭은 .NET Framework 4.6.1을 지원하며, .NET 기반 웹 애플리케이션 서버를 실시간 모니터링할 수 있습니다.

IIS, 콘솔, 서비스, 윈도우 응용프로그램을 포함한  .NET 애플리케이션을 모니터링할 수 있으며, DB 쿼리와 외부 호출을 추적해 지연 구간을 파악할 수 있습니다. 장애를 재현하지 않아도 실시간으로 애플리케이션 상태를 확인할 수 있습니다.

## 동작 방식

.NET 프로그램은 컴파일 후 중간언어(Microsoft Intermediate Language, MSIL) 형태 *.exe* 또는 *.dll* 파일로 변환됩니다. 공용 언어 런타임(Common Language Runtime, CLR)의 JIT(Just-In-Time) 컴파일러가 네이티브 코드로 번역해 동작합니다. 

와탭은 .NET 프레임워크의 트레이싱 API를 통해 실행 중인 애플리케이션을 모니터링합니다.

![트레이싱 아키텍쳐](https://img.whatap.io/media/agent_dotnet/install/profiling-architecture.png)

.NET 모니터링 서비스를 사용하려면 모니터링 대상 애플리케이션에 와탭 모니터링 에이전트를 설치해야 합니다. 에이전트는 윈도우에 설치 가능한 인스톨러로 제공되며, 설치 후 자동으로 와탭 에이전트가 윈도우 서비스에 등록됩니다.
