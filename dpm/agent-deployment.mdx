## 방화벽 확인 {#firewall}

DBX 에이전트가 와탭 서버와 통신할 수 있도록 방화벽을 설정해야 합니다.

- **Outbound**: DBX 에이전트 → 와탭 서버 (TCP 6600, 6100 등)  
- **Inbound**: 필요 시 DB 서버 포트(PostgreSQL 5432, MySQL 3306 등)  

:::note
프로젝트 환경에 따라 방화벽 정책이 다를 수 있습니다.  
보안팀과 협의해 네트워크 연결을 반드시 확인하세요.
:::
