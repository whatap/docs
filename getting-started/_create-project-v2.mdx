## 프로젝트 생성하기{#create-project}

에이전트를 설치하기 전에 먼저 프로젝트를 생성하세요.

1. [와탭 모니터링 서비스](https://service.whatap.io)로 이동한 다음 로그인하세요.

1. 프로젝트를 생성하려면 화면 왼쪽 메뉴에서 ***+*** <cmdname sid="BTN06696" className="uitext" /> 버튼을 선택하세요.

1. <cmdname sid="platform_catalog" className="uitext" /> 화면에서 프로젝트에 설치할 상품을 선택하세요.

1. <cmdname sid="project_name" className="uitext" />, <cmdname sid="data_server_region" className="uitext" />, <cmdname sid="project_time_zone" className="uitext" /> 등의 항목을 차례로 설정하세요.

    <Xclude product='browser'>
    <ImgLang img='create-project-v2.png' desc=''/>
    </Xclude>

    <InDoc product='browser'>
    <ImgLang img='create-project-v2-browser.png' desc=''/>
    </InDoc>

1. 모든 설정을 완료한 다음에는 <cmdname sid="creating_a_project" className="uitext" /> 버튼을 선택하세요.

:::note

* <cmdname sid="data_server_region" className="uitext" />은 리전(클라우드 서비스를 제공하기 위해 설치한 데이터 센터의 묶음)을 의미합니다. 특정 리전을 선택하면 해당 리전에 속한 데이터 센터에 사용자의 데이터를 저장합니다.

* <cmdname sid="project_time_zone" className="uitext" />은 알림, 보고서를 생성하는 기준 시간입니다.

* 여러 개의 프로젝트를 그룹으로 묶어 관리하려면 <cmdname sid="project_groups" className="uitext" />에서 그룹을 선택하거나 그룹을 추가하세요. 그룹에 대한 자세한 설명은 [다음 문서](../project/group)를 참조하세요.

* 조직을 선택한 상태에서 프로젝트를 추가할 경우 <cmdname sid="group_of_organization" className="uitext" />을 필수로 설정해야 합니다.

:::
