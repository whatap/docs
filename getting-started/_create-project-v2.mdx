## 프로젝트 생성하기{#create-project}

</Xclude>

에이전트를 설치하기 전에 먼저 프로젝트를 생성하세요.

1. [와탭 모니터링 서비스](https://service.whatap.io)에 로그인합니다.

1. 왼쪽 사이드 메뉴에서 <Cmdname sid="BTN07332" className="uitext" /> &gt; + <Cmdname sid='BTN06696' className='uitext'/> 버튼을 클릭합니다.

1. <Cmdname sid="TTL07073" className="uitext" /> 화면에서 설치할 상품을 선택합니다.

1. 아래 항목을 입력하거나 선택합니다.

    * <Cmdname sid="project_name" className="uitext" />: 프로젝트의 이름을 입력합니다.

    * <Cmdname sid="data_server_region" className="uitext" />: 데이터 서버가 위치한 리전을 선택합니다. 리전은 클라우드 서비스를 제공하는 데이터 센터의 묶음입니다. 선택한 리전에 사용자의 데이터가 저장됩니다.

    * <Cmdname sid="project_time_zone" className="uitext" />: 알림 및 보고서 생성 시 기준이 되는 시간을 설정합니다.

    * <Cmdname sid="TTL07727" className="uitext" />: 경고 알림 메시지의 언어를 설정합니다. (한글, 영어 지원)

    * <Cmdname sid="project_groups" className="uitext" />: 여러 프로젝트를 그룹으로 묶어 관리할 수 있습니다. 소속될 그룹이 있으면 선택하세요.

    * <Cmdname sid="project_description" className="uitext" />: 프로젝트에 대한 추가 설명이나 세부 정보를 입력합니다.

1. 모든 설정을 완료하면 <Cmdname sid="creating_a_project" className="uitext" /> 버튼을 클릭합니다.

:::note

조직을 선택한 상태에서 프로젝트를 추가할 경우 <Cmdname sid="group_of_organization" className="uitext" />을 필수로 설정해야 합니다.

그룹에 대한 자세한 설명은 [다음 문서](/project/group)를 참고하세요.

:::
