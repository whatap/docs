와탭의 모니터링 서비스에 대한 빠른 이해와 효율적인 이용을 위해 학습 자료를 제공합니다. 다음의 학습 자료는 ***애플리케이션 대시보드***, ***히트맵 트랜잭션***, ***리소스 보드***를 다루고 있습니다. 애플리케이션 모니터링과 서버 모니터링 제품의 활용에 관심이 있는 사용자를 대상으로 기본 안내 문서를 보완한 것입니다. 

![학습하기 1.100.0 sc](/img/learning.png)

### 애플리케이션 모니터링

* [애플리케이션 대시보드 살펴보기](../best-practice-guides/about-apm-dashboard)
  와탭의 ***애플리케이션 대시보드*** 메뉴의 차트형 위젯에 대한 분석 방법을 다음과 같은 순서로 안내합니다.

  * 애플리케이션 연결 상태 확인하기
  * 트랜잭션 분석하기
  * 사용자 및 서비스, 리소스 분석

* [히트맵 트랜잭션 살펴보기](../best-practice-guides/about-apm-hitmap-class)  
  와탭의 ***히트맵 트랜잭션*** 메뉴를 활용해 서비스의 장애 현황을 파악하는 방법을 다음과 같은 순서로 안내합니다.

  * 패턴 분석
  * 구간 및 스택 분석
  * 호출 관계 분석
  * 트랜잭션 로그 분석


### 서버 모니터링

* [리소스 보드 살펴보기](../best-practice-guides/about-server-dashboard)  
  와탭의 ***리소스 보드*** 메뉴의 차트형 위젯에 대한 분석 방법을 다음과 같은 순서로 안내합니다.

  * 서버 모니터링 핵심 지표
  * 서버 상태 요약
  * 시스템 운영 통계
  * 조기 경보 및 알림
