Go 에이전트 서비스를 재시작하거나 일시 중지, 삭제하는 방법을 제공합니다.

## 에이전트 서비스 재시작

와탭 에이전트(whata-agent) 서비스가 실행 중이거나 오류가 발생한 경우 재시작하세요.

<Tabs groupId="golang">
<TabItem value="rc" label="Red Hat/CentOS">

```bash title='SH'
$ service whatap-agent restart
```

</TabItem>
<TabItem value="du" label="Debian/Ubuntu">

```bash title='SH'
$ sudo service whatap-agent restart
```

</TabItem>
<TabItem value="al" label="Amazon Linux">

```bash title='SH'
$ sudo service whatap-agent restart
```

</TabItem>
<TabItem value="apl" label="Alpine Linux">

```bash title='SH'
$ /usr/whatap/agent/whatap-agent stop
$ /usr/whatap/agent/whatap-agent start
```

</TabItem>
</Tabs>

## 에이전트 삭제

Go 애플리케이션의 소스 코드에서 Whatap 라이브러리를 삭제하세요.

<Tabs groupId="golang">
<TabItem value="rc" label="Red Hat/CentOS">

```bash title='SH'
$ sudo yum remove whatap-agent
```

</TabItem>
<TabItem value="du" label="Debian/Ubuntu">

```bash title='SH'
$ sudo apt-get purge whatap-agent
```

</TabItem>
<TabItem value="al" label="Amazon Linux">

```bash title='SH'
$ sudo yum remove whatap-agent
```

</TabItem>
<TabItem value="apl" label="Alpine Linux">

```bash title='SH'
$ /usr/whatap/agent/whatap-agent stop

# /usr/whatap/agent 디렉터리 삭제
$ sudo rm -r /usr/whatap/agent
```

</TabItem>
</Tabs>
