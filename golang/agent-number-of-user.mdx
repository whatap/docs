description: WAS에 연결한 실시간 사용자 수를 집계합니다.
tags: [ Golang ]
---

Golang 웹 애플리케이션 서버에 연결한 사용자의 수를 집계합니다.

* **trace_user_header_ticket** <span class='type'>String</span>

  HTTP Header의 특정 값으로 사용자 수를 집계하고자 하는 경우 해당 Key 값을 설정합니다.

* **trace_user_cookie_keys** <span class='type'>String</span>

  설정한 쿠키의 값을 고유한 사용자 정보로 선택합니다. 여러 개를 등록할 때는 쉼표(,)를 구분자로 이용하세요. 설정한 순서대로 검색합니다.
