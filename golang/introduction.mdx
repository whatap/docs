와탭 Go 애플리케이션 모니터링은 Go 애플리케이션에 대한 모니터링 서비스를 제공합니다.

- 웹 프레임워크를 지원합니다. 웹 트랜잭션의 URL, 응답시간, TPS 정보, 오류 메세지 정보 등을 수집합니다.
- Go Runtime 패키지의 정보를 지속적으로 수집합니다. 메모리, goroutine, gc 관련 정보를 수집합니다.
- MSA 환경에서 여러개의 API/RPC 서비스 호출하는 경우 연계 추적을 통해 호출 관계를 수집합니다.
