## Go 에이전트 지원 환경

와탭 Go 모니터링 에이전트를 설치하기 전에 지원 환경을 확인하세요.

:::info
최종 업데이트: 2025-11-18
:::

### 지원 버전

| 언어 | 지원 버전 |
| --- | --- |
| Go | 1.17 이상 |

### 운영체제

| OS | 최소 버전 | 아키텍처 |
| --- | --- | :---: |
| CentOS / Red Hat | 6.x | 64bit |
| Debian / Ubuntu | 12.04 | 64bit |
| Amazon Linux | - | 64bit |
| Alpine Linux | - | 64bit |
| Linux | - | ARM64 (aarch64) |

### 라이브러리

| 카테고리 | 라이브러리 |
| --- | --- |
| **웹 프레임워크** | net/http, gin, chi, fiber, gorilla, echo, fasthttp |
| **데이터베이스** | database/sql, gorm, redigo |
| **기타** | grpc, sarama, k8s.io/client-go |

:::note

프레임워크 또는 라이브러리 적용 방법 및 예제는 [GitHub 사이트](https://github.com/whatap/go-api-example)를 참고하세요.

:::
