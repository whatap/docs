Resin을 사용하는 애플리케이션 서버 환경에서 JVM 옵션을 추가하는 방법을 확인하세요. 

1. Resin 설정 파일에서 ```jvm-arg```를 추가하여 ```-javaagent``` 옵션을 설정하세요.

    ```xml title='Resin 4.x: $RESIN_HOME/conf/resin.properties'
    ...
    <resin xmlns="http://caucho.com/ns/resin">
    <cluster id="web-tier">
        <server-default>
            <jvm-arg>-Xmx1024m -XX:MaxPermSize=128m -javaagent:/whatap/whatap.agent-X.Y.Z.jar</jvm-arg>
        </server-default>
        ...
    </cluster>
    </resin>
    ...
    ```

1. 애플리케이션 서버를 다시 시작하세요.

1. 애플리케이션 서버 로그 파일과 에이전트 로그 파일을 통해 에이전트가 정상 작동하는지, 에러가 발생하지 않았는지 확인하세요. 로그 파일의 위치는 다음을 확인하세요.
    
    * 에이전트: *$WHATAP_HOME/logs/whatap-{SERVER_NAME}-{DATE}.log*
    * RESIN 4.x: *$RESIN_HOME/log/jvm-app-#.log*
