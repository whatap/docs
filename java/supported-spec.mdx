## Java 에이전트 지원 환경

에이전트를 설치하기 전에 JVM 애플리케이션 서버가 다음의 사양을 충족하는지 확인하세요.

Web Application Server (WAS), 데몬, 배치 애플리케이션 등 JVM 기반 모든 애플리케이션에 적용할 수 있습니다. 

:::info
최종 업데이트: 2025-11-18
:::

:::note

Java 1.6 미만은 support@whatap.io로 문의하거나 [다음 문서](add-jvm-opt/java-under-ver)를 참고하세요.

:::

  | OS 계열     | 지원 버전                       | 아키텍처      |
  | ----------- | ------------------------------- | ------------- |
  | **Unix**    | HP-UX 11.x                      | 32bit, 64bit  |
  | ^          | Oracle Solaris 2.8, 2.9, 10, 11 | 32bit, 64bit  |
  | **Linux**   | Red Hat                         | Itanium 64bit |
  | ^            | x86 Intel Linux                 | 32bit         |
  | **Windows** | Windows Server                  | 2000 이상     |
  | ^            | Windows                         | XP 이상       |
  | **기타**    | Itanium                         | 64bit         |

  :::note

  JDK 6를 설치할 수 있는 모든 운영체제를 지원합니다.
  
  :::
