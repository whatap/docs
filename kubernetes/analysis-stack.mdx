와탭은 10초(기본값) 간격으로 수집한 스레드 스택을 활용하여 메소드 레벨의 성능 지연 구간을 분석합니다. 이 기능을 통해 각 컨테이너의 ***탑 스택***, ***유니크 스택***, ***액티브 스택***을 확인할 수 있습니다. 

:::note

***탑 스택***, ***유니크 스택***, ***액티브 스택***에 대한 자세한 내용은 다음 문서를 참조하세요. 

* [Java](../java/analysis-apm)

* [Python](../python/analysis-apm)

:::

스택 트레이스 상의 각 Step 기준으로, Step과 Step 간의 호출 비율을 백분율로 분석한 정보

유니크 스택
Stack Trace 전체의 Hash 값 기준의 산출 결과로 전체 Step이 동일한 호출 비율을 백분율로 분석한 정보
