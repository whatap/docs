### 컨테이너 요약 분석과 Pod 요약 분석{#kube-object-summary-analysis}

***컨테이너 요약 분석***과 ***Pod 요약 분석***은 쿠버네티스 환경에서 각 컨테이너와 Pod의 성능과 자원에 대한 분석 보고서를 제공합니다. 클러스터와 네임스페이스의 컨테이너 및 Pod의 자원 할당 및 사용 현황을 실시간으로 모니터링하고 보고서 내 제안을 참고하여 효과적으로 대응할 수 있습니다. 

:::note 

현재 각 요약 분석 기능은 국문으로만 제공하고 있습니다. 

:::

#### 컨테이너 요약 분석{#container-summary-analysis}

* 기본 정보: 컨테이너의 기동 시간, 재기동 횟수, 속한 파드 및 네임스페이스 정보를 제공합니다.

* 컨테이너 이미지: 현재 사용 중인 컨테이너 이미지와 동일한 이미지를 사용하는 다른 컨테이너의 수 및 전체 리소스 사용량을 확인할 수 있습니다.

* 노드 정보: 컨테이너가 실행되는 노드의 운영체제, IP 주소, CPU 코어 수, 메모리 크기 등 하드웨어 정보를 제공합니다.

* 리소스 사용 현황: 컨테이너의 CPU 및 메모리 리소스 사용량, 설정된 한도(Limit) 및 요청(Request) 값과 비교한 사용률을 제공합니다.

#### Pod 요약 분석{#pod-summary-analysis}

* 기본 정보: Pod의 클러스터 이름, 네임스페이스, 기동 시간, 현재 상태(Phase) 등을 확인할 수 있습니다.

* 쿠버네티스 이벤트: 최근 파드와 관련된 이벤트 로그를 통해 문제가 발생했는지 확인할 수 있습니다.

* 실행 주체: 파드를 생성한 컨트롤러(DaemonSet, Deployment 등)를 식별합니다.

* 노드 정보: 파드가 실행 중인 노드의 운영체제, 커널 버전, CPU, 메모리 등의 상세 사양을 제공합니다.

* 리소스 사용 현황: 파드 내 모든 컨테이너의 CPU와 메모리 사용량, 설정된 리소스 한도(Limit)와 비교한 사용률을 보여줍니다.
