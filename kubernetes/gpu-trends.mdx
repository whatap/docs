# GPU 트렌드

**GPU 트렌드**는 GPU 장비별 사용량을 장기 통계 데이터로 시각화하여, 특정 시점에 어떤 장비가 비효율적으로 사용되었는지 빠르게 식별할 수 있습니다. 

## 기본 화면

GPU 장기 사용량 통계 및 패턴을 확인할 수 있습니다. 조회 단위는 **일간, 주간, 월간**을 지원합니다.
장비별 GPU 사용률이 블록 형태로 표시되며, 색상 진하기에 따라 사용량 수준을 구분할 수 있습니다.

* **50% 미만**: 연한 파랑
* **80% 미만**: 파랑
* **80% 이상**: 진한 파랑

## 상세 정보 확인

각 블록에 마우스를 올리면 해당 시점과 장비의 상세 정보를 툴팁으로 확인할 수 있습니다.

* **Label**: 장비명 및 GPU 모델
* **UUID**: GPU 고유 식별자
* **Time**: 기준 시각
* **GPU Utilization**: 해당 시점의 GPU 사용률(%)
