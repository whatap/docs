홈 화면 > 프로젝트 선택 > ***관리*** > ***네임스페이스 관리***

클러스터 프로젝트에서 ***네임스페이스 관리*** 메뉴를 통해 네임스페이스 프로젝트를 관리할 수 있습니다. 네임스페이스 프로젝트에서는 해당 메뉴로 진입할 수 없습니다.

:::note

클러스터 프로젝트의 ***대시보드*** > ***컨테이너 맵*** 메뉴 화면 위에는 하위 네임스페이스로 이동할 수 있는 리스트 박스를 제공합다.

![네이스페이 이동](https://img.whatap.io/media/images/whatap_kubernetes_container_map_namespace_project_list.png)

:::

### 네임스페이스 프로젝트 생성

***네임 스페이스 목록***에서 프로젝트를 생성할 항목의 ***+ 프로젝트 추가*** 버튼을 선택하세요. ***프로젝트 추가*** 창이 나타나면 ***확인*** 버튼을 선택하세요.

### 네임스페이스 프로젝트 삭제

1. ***네임 스페이스 목록***에서 삭제하려는 프로젝트 항목의 가장 오른쪽에 ![설정 아이콘](/img/ico-setting.svg) 버튼을 선택하세요.

1. ***프로젝트 관리*** 메뉴로 이동합니다.

1. 화면 오른쪽 위에 ***프로젝트 삭제*** 버튼을 선택하세요.

:::caution

네임스페이스 프로젝트를 삭제할 경우 사용자 목록 정보와 그동안 수집한 데이터는 모두 삭제합니다. 삭제한 데이터는 다시 복구할 수 없기 때문에 주의하세요.

:::
