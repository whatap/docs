<Cmdname sid="openmetrics" className="uitext" />를 처음 사용하는 사용자가 복잡한 설정 과정 없이 빠르게 메트릭을 수집하고 시각화할 수 있도록, 기본 템플릿과 사전 정의된 템플릿를 제공합니다. 이를 통해 자주 사용되는 오픈소스 메트릭을 손쉽게 적용할 수 있으며, 초기 도입 장벽을 낮추고 운영자가 확장 가능한 모니터링 환경을 구축할 수 있도록 지원합니다.

## 템플릿 목록

<Cmdname sid="openmetrics_template_list" className="uitext" />은 기본 오픈소스 템플릿 카드를 제공합니다. 각 카드에 간단한 설명, 템플릿 이동, 수집 안내가 포함되어 있습니다.

* ![오른쪽 화살표 아이콘](/img/ic-arrow-right.svg) **템플릿**: 클릭하면 해당 템플릿 화면으로 바로 이동합니다.
* ![정보 아이콘](/img/ic-info-circle.svg) **수집 안내**: 클릭하면 데이터 수집 설정 방법을 제공합니다.

### Nginx 템플릿 {#nginx-dashboard}

Apache Nginx 모니터링을 통해 웹 서버의 성능을 실시간으로 추적하고 분석할 수 있습니다.

### Istio 템플릿 {#istio-dashboard}

Istio 서비스 메시 환경의 서비스 간 관계와 트래픽 흐름을 시각적으로 표현하며, WhaTap APM 데이터와 연계하여 통합 모니터링 환경을 템플릿로 제공합니다. 화면 오른쪽 상단의 버튼을 클릭해 템플릿를 확대하거나 축소할 수 있습니다.

#### 범례 {#legend}

화면 오른쪽 상단의 ![정보 아이콘](/img/ic-info-circle.svg) 버튼을 클릭하면 범례를 확인할 수 있습니다. 

##### 노드 모양 {#nod-shapes}

* 삼각형: 쿠버네티스 서비스

* 원: 워크로드

* 사각형: 애플리케이션

* 작은 원: Pod(파드)

##### 노드 상태 {#nod-status}

HTTP 요청/응답 전체 중 에러(4xx, 5xx)의 비율

* *초록색*: 에러율 5% 미만

* <span class='slow'>주황색</span>: 에러율 5% 이상

* <span class='vslow'>빨간색</span>: 에러율 10% 이상

* 회색: 요청 수 없음

##### Pod 노드 상태 {#pod-status}

쿠버네티스 Pod의 Phase

* *초록색*: Running 상태의 Pod

* <span class='slow'>주황색</span>: Pending 상태의 Pod

* <span class='vslow'>빨간색</span>: Failed 상태의 Pod

##### 엣지 색상 {#edge-colors}

두 노드 간의 HTTP 요청에서 발생한 에러(4xx, 5xx)의 비율

* *초록색*: 에러율 10% 미만

* <span class='slow'>주황색</span>: 에러율 10% 이상

* <span class='vslow'>빨간색</span>: 에러율 20% 이상

* <span class='normal'>파랑색</span>: TCP 통신

* 회색: 상태 알 수 없음
