홈 화면 > 프로젝트 선택 > ***리소스*** > ***퍼포먼스 요약***

최근 시간별 차트를 통하여 컨테이너/Pod 수량, 노드 수량 등 쿠버네티스 주요 지표들에 어떤 변화가 있었는지 확인할 수 있습니다.

![퍼포먼스 요약](https://img.whatap.io/media/images/whatap_kubernetes_resource_performance_summary.png)

노드 수량 위젯 오른쪽 위에 ![더보기 아이콘](/img/ico-more.svg) > ![바로가기 아이콘](/img/ico-shortcut.svg) 버튼을 차례로 선택하면 ***리소스*** > ***노드 목록*** 메뉴로 이동합니다. 

위젯으로 제공하는 노드 정보는 다음과 같습니다.

* 최근 30분 차트
* ***Node CPU (%)***, ***Node Memory (%)*** 등

노드 CPU 위젯과 메모리 위젯 오른쪽 위에 ![더보기 아이콘](/img/ico-more.svg) > ![바로가기 아이콘](/img/ico-shortcut.svg) 버튼을 차례로 선택하면 ***분석*** > ***메트릭스 차트*** 메뉴로 이동합니다. 해당 메뉴를 사용하면 더욱 자세한 조건을 통해 차트를 확인할 수 있습니다.

위젯으로 제공하는 컨테이너 정보는 다음과 같습니다.

* 최근 10분 차트
* ***Container Status***, ***Container CPU (%)***, ***Container Memory (%)*** 등

위젯으로 제공하는 애플리케이션 정보는 다음과 같습니다.

* 최근 10분 차트
* ***Active TX***, ***TPS***, ***Response Time*** 등

:::note

애플리케이션 정보는 애플리케이션 에이전트 설치 대상이 존재할 때 표시합니다.

:::
