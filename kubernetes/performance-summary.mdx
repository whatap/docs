홈 화면 > 프로젝트 선택 > ***대시보드*** > ***퍼포먼스 요약***

최근 시간별 차트를 통하여 컨테이너/Pod 수량, 노드 수량 등 쿠버네티스 주요 지표들에 어떤 변화가 있었는지 확인할 수 있습니다. 다음 이미지는 클러스터 프로젝트(<code class='my'>Kubernetes</code>) 예시입니다.

<ImgLang img='k8s-performance-summary.png' desc='퍼포먼스 요약' />

* **바로 가기 옵션**

  위젯별 오른쪽 상단에 위치한 ![더보기 아이콘](/img/ico-more.svg) ***더보기*** 아이콘을 선택하면 다음과 같이 ![바로가기 아이콘](/img/ico-shortcut.svg) ***바로 가기*** 옵션을 확인할 수 있습니다.

  ![바로 가기](/img/k8s-performance-summary-shortcut.png)


* **라인 차트 설정 옵션**

  라인 차트 위젯별 오른쪽 상단에 위치한 ![더보기 아이콘](/img/ico-more.svg) ***더보기*** 아이콘을 선택하면 ![설정 아이콘](/img/ic-setting.svg) 설정 옵션을 확인할 수 있습니다. 라인 차트의 경우 설정 옵션을 통해 id가 되는 값을 변경할 수 있습니다. 해당 설정은 저장되지 않습니다.  

  > 예, ***Node CPU (%)*** 위젯의 경우 ***Node***별 데이터 조회가 가능합니다. 메트릭스를 변경해 ***ContainerName***별로 조회할 수 있습니다.

## 쿠버네티스 클러스터 통합 퍼포먼스

클러스터 프로젝트(<code class='my'>Kubernetes</code>)의 경우 ![지시선 1](/img/number-01.png) <UI>쿠버네티스 클러스터 통합 퍼포먼스</UI> 영역에서 와탭 노드 에이전트 설치 여부와 관계없이 클러스터 전체 정보를 표시합니다. 네임스페이스 프로젝트로 분리된 정보도 통합해 표시합니다.

## 쿠버네티스 네임스페이스 퍼포먼스

네임스페이스 프로젝트(<code class='my'>Namespace</code>)의 경우 ![지시선 1](/img/number-01.png) <UI>쿠버네티스 네임스페이스 퍼포먼스</UI> 영역에서 와탭 노드 에이전트 설치 여부와 관계없이 네임스페이스 정보를 표시합니다. 네임스페이스와 연관성이 있는 컨테이너, Pod, 노드의 정보를 표시합니다. 

## 프로젝트 퍼포먼스

![지시선 2](/img/number-02.png) ***프로젝트 퍼포먼스*** 영역은 프로젝트 내의 와탭 노드 에이전트 및 애플리케이션 에이전트가 설치된 대상의 정보를 표시합니다. 네임스페이스 프로젝트로 분리된 정보는 네임스페이스 프로젝트에서 확인할 수 있습니다. 

:::note 

에이전트 설치가 정상적으로 완료되지 않은 경우 데이터가 표시되지 않을 수 있습니다. 

:::

### 노드 정보

위젯으로 제공하는 노드 정보는 다음과 같습니다.

* 최근 30분 차트

  * ***Node CPU (%)***, ***Node Memory (%)*** 등

:::note 

* 노드 수량 위젯 오른쪽 위에 ![더보기 아이콘](/img/ico-more.svg) ***더보기*** 아이콘 > ![바로가기 아이콘](/img/ico-shortcut.svg) ***바로 가기*** 아이콘을 차례로 선택하면 ***클러스터*** > ***노드 목록*** 메뉴로 이동합니다. 

* ***Node CPU (%)*** 위젯과 ***Node Memory (%)*** 위젯 오른쪽 위에 ![더보기 아이콘](/img/ico-more.svg) ***더보기*** 아이콘 > ![바로가기 아이콘](/img/ico-shortcut.svg) ***바로 가기*** 아이콘을 차례로 선택하면 ***분석*** > ***메트릭스 차트*** 메뉴로 이동합니다. 해당 메뉴를 사용하면 더욱 자세한 조건을 통해 차트를 확인할 수 있습니다.

:::

### 컨테이너 정보

위젯으로 제공하는 컨테이너 정보는 다음과 같습니다.

* 최근 10분 차트

  * ***Container Status***, ***Container CPU (%)***, ***Container Memory (%)*** 등

### 애플리케이션 정보

위젯으로 제공하는 애플리케이션 정보는 다음과 같습니다.

* 최근 10분 차트

  * ***Active TX***, ***TPS***, ***Response Time*** 등

:::note

애플리케이션 정보는 애플리케이션 에이전트 설치 대상이 존재할 때 표시합니다.

:::
