### 리소스 

대상 선택 후 상세 보기의 <Cmdname sid="resource" className="uitext" /> 탭에서 선택한 리소스와 관련된 리소스 목록 및 관계도를 조회할 수 있습니다. 

<ImgLang img='k8s-app-list-detail.png' desc='애플리케이션 목록' />

#### 요약 정보

선택한 리소스와 관련된 리소스 목록의 요약 정보를 제공합니다. 리소스 명 옆에 상세보기 아이콘을 선택 시 해당 리소스 목록 화면으로 이동합니다. 

* <Cmdname sid="container" className="uitext" />: 해당 컨테이너만 필터링된 상태의 <Cmdname sid="container_list" className="uitext" />으로 이동합니다. 

* <Cmdname sid="podName" className="uitext" />: 해당 파드만 필터링된 <Cmdname sid="podList" className="uitext" />으로 이동합니다. 

* <Cmdname sid="node" className="uitext" />: 해당 노드 이름이 선택된 상태의 <Cmdname sid="node_detail" className="uitext" />로 이동합니다. 

#### 리소스 관계도 

리소스간 연결 관계와 상태를 시각적으로 확인할 수 있습니다. 리소스간 연결 관계와 상태를 시각적으로 확인할 수 있습니다. 목록과 동일하게 Pod, 컨테이너, 애플리케이션에 마우스 오버 시 상세 지표를 확인할 수 있습니다. 리소스 선택 시 해당 리소스 목록 페이지로 이동합니다.
