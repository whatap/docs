**액세스 로그**는 서버에 접속하는 클라이언트의 접속 정보를 남기는 로그 파일입니다. 액세스 로그에 포함된 내용은 다음과 같습니다. 

* 원격 호스트 IP 주소(요청자)
* 요청 시간
* HTTP 상태 코드
* HTTP 헤더를 제외한 전송 바이트 수
* 요청을 처리하는 데 걸린 시간(ms)
* 리퍼러(Referrer)
