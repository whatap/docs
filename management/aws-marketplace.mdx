AWS 사용자는 AWS Marketplace에서 와탭 모니터링 서비스를 구독할 수 있으며, 서비스 이용료를 AWS 비용과 함께 한 번에 결제할 수 있습니다. 이를 위해서는 AWS 계정과 와탭 서비스 계정을 연결해야 하며, 이때 와탭 서비스 계정은 연결 과정에서 새로 생성해야 합니다.
기존에 사용 중인 와탭 서비스 계정은 연결이 불가능합니다.


:::note 

이미 와탭 서비스를 유료로 이용 중인 경우, 와탭 서비스 비용을 AWS 클라우드 비용과 함께 결제하려면 반드시 AWS Marketplace 구독 절차에 따라 **신규 계정**으로 가입해야 합니다. 이후, 기존 서비스의 결제 권한을 새로 생성한 계정으로 이관해야 합니다.

:::

## AWS Marketplace에서 와탭서비스 찾기

1. AWS에 로그인한 후, 검색창에 “whatap”을 입력하세요.

2. 아래와 같이 3개의 검색 결과가 표시됩니다.원하는 국가에 맞는 서비스를 선택하면 됩니다.

    * WhaTap Observability Platform

    * WhaTap Observability Platform(Japan)

    * WhaTap Observability Platform(Korea)

## 와탭 서비스 구독하기

선택한 WhaTap Observability Platform의 기본 정보를 확인한 후, 구독을 위해 우측 상단의 “View purchase options” 버튼을 클릭하여 구독 페이지로 이동합니다.

구매 페이지에서는 Offer details, Pricing and unit details, Terms and conditions 등 다양한 정보를 확인할 수 있습니다. Offer details에서 와탭 서비스의 과금 방식을 선택할 수 있으며, AWS Marketplace에서는 사용량 기반의 종량제 방식을 제공합니다. 또한, 15일 무료 체험(Free Trial)도 지원합니다.

<ImgLang img='awsmp-view-purchase-offer.png' desc='AWSMP 와탭 구독 상세 내역' />

가격 정보와 이용 조건을 확인한 후, 하단의 “Subscription” 버튼을 클릭합니다.

<ImgLang img='awsmp-view-purchase-pricing.png' desc='AWSMP 와탭 서비스 가격정보' />

<ImgLang img='awsmp-view-purchase-terms-conditions.png' desc='AWSMP 와탭 서비스 계약정보' />

## AWS 계정과 연결할 와탭 서비스 계정 생성하기

Subscribe 버튼이 비활성화된 경우, 페이지 상단으로 이동하여 와탭 서비스 계정 연결 작업을 진행합니다. 상단에 표시되는 초록색 알림창의 우측 “Set up your account” 버튼을 클릭합니다. AWS 계정과 연결할 와탭 서비스 계정을 생성하기 위해, 와탭 계정 생성 페이지가 새 탭으로 열립니다.

<ImgLang img='awsmp-view-purchase-linking-whatap-account.png' desc='AWSMP 와탭 서비스 연동' />

와탭 계정 생성 페이지에서 계정을 생성한 후 와탭 서비스에 로그인합니다. 모니터링하려는 제품의 [프로젝트를 생성](https://docs.whatap.io/quick-guide#cproject)하고 에이전트를 설치합니다. 

모니터링 대상에 와탭 에이전트를 설치한 후, 대시보드에서 성능 데이터가 정상적으로 수집되는지 확인합니다.

AWS 마켓플레이스를 통해 구독했는지는 우측 상단 프로필을 클릭한 후, **이용 내역 > 결제 정보**에서 확인할 수 있습니다.

## 구독 취소하기

1. AWS Console에 와탭 서비스를 구독한 계정으로 로그인합니다.

1. Console Home 화면에서 AWS Marketplace를 선택하여 이동합니다.

1. AWS Marketplace > Manage subscriptions를 클릭합니다. 현재 구독 중인 서비스 목록에서 WhaTap Observability Platform을 선택합니다.

1. 선택한 제품의 상세 페이지에서 Agreement 영역의 Actions 드롭다운 메뉴에서 Cancel subscription을 클릭합니다.

    <ImgLang img='awsmp-cancel-subscription.png' desc='AWSMP 와탭 서비스 구독 취소' />

1. 구독 취소 여부를 확인하는 창이 나타나면, 구독 취소 버튼을 클릭합니다.

1. 구독 내역에서 와탭 서비스가 삭제된 것을 확인합니다.
