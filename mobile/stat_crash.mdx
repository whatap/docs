**크래시 통계 메뉴**는 모바일 앱에서 발생한 크래시와 예외를 다양한 기준으로 분석할 수 있는 대시보드입니다.  
운영체제, 앱 버전, 디바이스, 예외 타입 등으로 그룹화하여 문제 패턴을 파악할 수 있습니다.  

- 상단 영역: 조회 시간 설정, 공통 필터 제공  
- 하단 영역:  
  - 왼쪽: 그룹화 선택  
  - 오른쪽: 선택한 그룹화 기준별 상세 화면 (로드 수 추이, 크래시 추이 그래프, 크래시 목록 테이블 제공)  


## 주요 구성

### 그룹화 선택

조회 결과를 다양한 기준으로 그룹화할 수 있습니다.  
각 그룹화 항목별 크래시 비율은 **파이 차트**, 시간별 크래시 발생 건수는 **바 차트**로 제공합니다.  
그룹화를 적용하지 않은 경우, 프로젝트별 로드 수 추이 그래프를 표시합니다.

| 항목       | 설명 |
| ---------- | ---- |
| 앱 버전    | 앱 버전별 크래시 발생 비율 및 추이 |
| 운영체제   | OS별 크래시 발생 비율 및 추이 |
| 디바이스   | 기기별 크래시 발생 비율 및 추이 |
| 예외 타입  | 예외 클래스/타입별 발생 비율 및 추이 |
| 쓰레드 이름 | 크래시가 발생한 쓰레드 비율 분석 |
| 예외 메시지 | 예외 메시지별 크래시 발생 비율 및 추이 |

**활용**: 특정 기준에 따른 크래시 패턴 분석  

### 크래시 목록 테이블

조회된 크래시를 상세 테이블로 제공합니다.  

| 항목       | 설명 |
| ---------- | ---- |
| 그룹 | 그룹화 기준 값 (예: iOS,Android) |
| 예외 메시지 | 발생한 예외 클래스 및 메시지 |
| 건수       | 해당 크래시의 총 발생 횟수 |
| 앱 버전    | 크래시가 발생한 앱 버전 |
| 운영체제   | 크래시가 발생한 OS 종류 |
| 디바이스   | 크래시가 발생한 기기 |
| 예외 타입  | 예외 유형 분류 |
| 쓰레드 이름 | 예외가 발생한 쓰레드 이름 |
