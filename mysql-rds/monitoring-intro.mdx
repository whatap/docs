와탭 MySQL RDS 모니터링 서비스는 기존의 MySQL 모니터링 기능을 제공할 뿐 아니라 에이전트 설치와 관리도 와탭에서 제공합니다. 데이터베이스 연결만 성공하면 DB 에이전트(DBX)를 다운받아 설치하고 설정하는 등의 번거로운 과정 없이 편리하게 DB 모니터링을 시작할 수 있습니다.
