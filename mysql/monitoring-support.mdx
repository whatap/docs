MySQL 모니터링을 시작하기 전에 다음의 지원 환경을 확인하세요.

## 데이터베이스 지원 버전

MySQL 5.5 버전 이상 / MariaDB 5.5 버전 이상을 지원합니다.

## 에이전트 구성

와탭 데이터베이스 모니터링 서비스는 에이전트와 수집 서버로 구성되며, 에이전트와 수집 서버 간에는 보안을 위해 데이터를 암호화합니다. 외부 라이브러리 미사용으로 제작되어 설치 및 업데이트 시 시스템 재기동이 필요 없습니다.

![에이전트 구성도](/img/agent_xos_architecture.png)

* 와탭 데이터베이스 모니터링 서비스는 모니터링 대상 데이터베이스 서버에 직접 에이전트를 설치하지 않고도 모니터링이 가능합니다.

* XOS 에이전트는 데이터베이스 서버의 프로세스 사용량을 모니터링할 수 있는 부가 옵션 에이전트 입니다. 데이터베이스 서버의 프로세스 사용량을 모니터링 하고 싶은 경우, 데이터베이스 서버에 별도의 에이전트를 띄워 데이터를 수집 할 수 있습니다.

* 에이전트와 서버는 바이너리 기반의 통신 프로토콜을 사용합니다. 또한 데이터를 선별적으로 암호화하여 데이터의 보안을 보장하면서도 암호화의 부담을 낮추었습니다.

:::note

* 모니터링 대상 데이터베이스 서버와의 네트워크 통신이 가능한 서버에 와탭 에이전트를 설치해야 합니다.
* 와탭 DBX 에이전트 서버에는 **Java 6** 이상 **JDK**를 설치해야 합니다.

:::
