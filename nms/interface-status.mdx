<Cmdname sid="side_interfaceStatus" className="uitext" /> 메뉴에서 디바이스의 네트워크 인터페이스 상태를 실시간으로 모니터링할 수 있습니다. 인터페이스의 연결 상태를 시각적으로 표시하고, 각 인터페이스에 대한 상세한 정보를 제공하여 네트워크 성능을 쉽게 분석할 수 있습니다.

## 기본 화면 안내 

<ImgLang img="nms-interface-status.png" desc="Interface Status" />

<Cmdname sid="side_interfaceStatus" className="uitext" /> 메뉴는 왼쪽 인터페이스 목록 영역과 오른쪽 상단 인터페이스 상태, 오른쪽 하단 인터페이스 상세 영역으로 구성되어있습니다. 왼쪽 인터페이스 영역에서 조회를 원하는 인터페이스 선택 시 오른쪽 영역에서 인터페이스 상태 및 상세를 확인할 수 있습니다. 

### 인터페이스 상태

인터페이스의 연결 상태를 확인할 수 있습니다. 

* <span class='snormal'>Connected</span> ![커넥션 아이콘](/img/ic-connected.svg): 인터페이스가 정상적으로 연결된 상태입니다. Link Status와 Admin Status 모두 "up"일 때 표시됩니다.

* <span class='k8s-normal'>Link Down</span> ![링크 다운 아이콘](/img/ico-connection-link-down.svg): 네트워크 링크가 다운된 상태입니다. Link Status는 "down", Admin Status는 "up" 상태일 때 나타납니다.

* <span class='k8s-normal'>Admin Down</span> ![링크 다운 아이콘](/img/ico-connection-down.svg): 관리자가 해당 인터페이스를 비활성화한 상태입니다. Link Status와 Admin Status 모두 "down"일 때 표시됩니다.

* <span class='k8s-normal'>Unavailable</span> ![물음표 아이콘](/img/ic-question-circle.svg): 물리적으로 활성화되지 않은 가상 인터페이스, 루프백 인터페이스, 또는 SFP 모듈이 미삽입된 Fiber 인터페이스 등 사용할 수 없는 인터페이스 수를 나타냅니다.


### 인터페이스 상세 

각 인터페이스의 상세 정보를 목록 형식으로 표시하며, 사용자가 인터페이스를 클릭하면 해당 디바이스의 추이 페이지로 이동하여 더 구체적인 상태를 확인할 수 있습니다.
