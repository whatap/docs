Node.js 웹 애플리케이션 서버에 연결한 사용자의 수를 집계합니다.

* **trace_user_enabled** <span class='type'>Boolean</span>

  기본값 `true`

  실시간 사용자 집계 여부를 설정합니다. 사용자 추적 옵션이 중복 설정된 경우 동작 우선 순위는 다음과 같습니다.

  > 1. **trace_user_using_ip**
  > 1. **user_header_ticket**

{@include: ../common-items/_common-collect-user.mdx} 

* **user_header_ticket_enabled** <span class='type'>Boolean</span>

  기본값 `false`

  사용자 아이디를 http 헤더의 특정 값으로 구분하고 싶을 때 사용합니다. 모바일에서 접속할 때 전달되는 경우가 많습니다.
