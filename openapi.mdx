수집 중인 모니터링 정보를 추출하여 별도로 활용하고자 하는 경우 Open API를 통해 해당 정보를 추출할 수 있는 기능을 제공합니다. 계정 API 토큰 및 프로젝트 API 토큰 및 프로젝트 코드를 파라미터로 전달하여 수집된 정보를 획득할 수 있습니다.

## Open API Token 확인

토큰은 계정 API 토큰과 프로젝트 API 토큰으로 구분됩니다. 계정 API 토큰은 프로젝트의 목록 정보를 취득하기 위한 용도로, 프로젝트 API 토큰은 프로젝트의 모니터링 정보를 취득하기 위한 용도로 활용합니다.

프로젝트 코드는 프로젝트 진입 시 브라우저 화면 상단의 윈도우 또는 탭 제목에 '`[{프로젝트 코드}]` Cloud Application Monitoring'와 같이 표시되는 정보를 통해 확인할 수 있습니다. API 토큰을 변경하려면 각 화면의 <Cmdname sid="reissue" className="uitext" /> 또는 <Cmdname sid="reissue_token" className="uitext" /> 버튼을 선택하세요.

### 계정 토큰 확인하기

1. 화면의 오른쪽 위에 프로필 아이콘을 선택하세요.

1. 팝업 메뉴가 나타나면 <Cmdname sid="MNU07360" className="uitext" /> 버튼을 선택하세요.

1. 화면을 아래로 스크롤해 <Cmdname sid="api_token" className="uitext" /> 섹션으로 이동하세요.

### 프로젝트 토큰 확인하기

1. 홈 화면에서 프로젝트를 선택하세요. 

1. 프로젝트 진입 후 왼쪽에 <Cmdname sid="management" className="uitext" /> > <Cmdname sid="project_management" className="uitext" /> 메뉴를 선택하세요.

1. <Cmdname sid="api_token" className="uitext" /> 항목을 확인하세요.
