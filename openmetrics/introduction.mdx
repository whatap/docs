> **통합 모니터링을 위한 새로운 접근**

**WhaTap** <Cmdname sid="openmetrics" className="uitext" />는 Prometheus 기반의 OpenMetrics 지표를 자동 수집하여, 복잡한 설정 없이 손쉽게 통합 모니터링 환경을 구축할 수 있는 솔루션입니다.
새로운 클라우드나 오래된 서버 환경과 상관없이 다양한 지표를 수집해 대시보드와 차트로 한눈에 제공합니다.

### 자동 수집과 간편한 설정

* [OpenMetrics 엔드포인트 등록만으로 자동 수집 시작](../openmetrics/install)

* 설치형 및 SaaS 환경 모두 지원

### 직관적인 시각화와 분석

수집된 지표는 실시간 차트로 제공되어, 운영자가 주요 지표를 즉시 확인할 수 있습니다.
예를 들어, 아래 그래프는 API 서버의 에러 코드 분포를 보여주며, 특정 시간대의 오류 유형과 빈도를 직관적으로 파악할 수 있습니다.

<ImgLang img='openmetrics-intro.png' desc='OpenMetrics 소개' />

이처럼 요청 유형별 오류 발생 추이를 시각화하면, 특정 오류(예. 404, 500)가 집중된 시점을 쉽게 확인할 수 있습니다. 또한 빠르게 원인을 분석하고 대응할 수 있습니다.

### 알림과 안정적 운영 지원

* 조건 기반 알림 정책

* 임계치 초과 시 실시간 알림 전송

* 장애 조기 감지 및 서비스 안정성 향상

### 다양한 활용 시나리오

* **인프라 운영**: CPU, 메모리, 네트워크 사용량 추적 및 용량 계획

* **애플리케이션 모니터링**: API 처리량, 오류율, 응답 지연 분석

* **DB 성능 분석**: 슬로우 쿼리, 커넥션 풀, 레플리케이션 지연, 데드락 탐지

### 운영 효율성과 확장성

WhaTap OpenMetrics는 단순한 지표 수집기를 넘어, 운영 효율성을 높이는 통합 모니터링 플랫폼입니다.
글로벌 SaaS 환경에서 검증된 확장성을 기반으로, 수십만 개 이상의 메트릭도 안정적으로 수집할 수 있습니다.

이를 통해 장애 대응 시간을 단축하고, 서비스 품질을 향상시킬 수 있습니다.
