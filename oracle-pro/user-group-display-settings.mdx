<Cmdname sid="users_group_display_settings" className="uitext" />은 특정 사용자 그룹에게만 메뉴나 프리셋을 표시할 수 있도록 제한하는 기능입니다. 관리자는 표시 그룹을 생성한 뒤, 원하는 메뉴 또는 프리셋과 연결하고, 보기 허용 여부를 설정함으로써 사용자의 역할이나 권한에 따라 불필요한 항목은 숨기고 필요한 정보만 노출할 수 있습니다.

:::note

**Oracle Pro 프로젝트에서 멤버 관리 권한이 있는 사용자**만 설정할 수 있습니다.

:::

## 기본 화면

<Cmdname sid="users_group_display_settings" className="uitext" /> 메뉴는 특정 프리셋, 메뉴, 기능에 대해 표시 여부를 설정할 수 있습니다.

### 보기 허용 설정

표시 그룹이 설정되지 않은 경우, 보기 허용의 기본값은 **비활성화**입니다.

- **보기 허용 ON:** 설정된 표시 그룹에 포함된 사용자에게만 페이지가 표시
- ![아이콘](/img/ic-toggle-off.svg) **보기 허용 OFF:** 모든 사용자에게 페이지가 표시(표시 그룹 설정과 무관)
