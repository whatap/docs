PHP 애플리케이션 서버의 통신을 설정하는 방법을 제공합니다.

* **whatap.server.host** <span class='type'>ip\_address</span>

  기본값 `127.0.0.1,127.0.0.1`

  에이전트가 수집한 데이터를 전송할 서버를 지정합니다. 수집 서버 이중화로 2개 이상의 IP를 가진 경우 콤마(,)로 분리하여 지정할 수 있습니다. 지정된 IP에는 수집 서버 proxy 데몬이 리스닝 상태로 서비스되어야 합니다.

* **whatap.server.port** <span class='type'>tcp\_port</span>

  기본값 `6600`

  수집 서버 PORT를 지정합니다. 포트는 하나만 지정할 수 있으므로 whatap_server_host에 지정된 수집 서버들은 동일 PORT를 사용해야 합니다.

* **whatap.tcp_so_timeout** <span class='type'>Millisecond</span>

  기본값 `60000`

  수집 서버와 통신하는 TCP 세션의 Socket Timeout 값을 지정합니다.

* **whatap.tcp_connection_timeout** <span class='type'>Millisecond</span>

  기본값 `5000`

  수집 서버와 통신하는 TCP 세션의 Connection Timeout 값을 지정합니다.

* **whatap.net_send_max_bytes** <span class='type'>Byte</span>

  기본값 `5242880`

  수집 서버로 데이터를 전송할 때 한 번에 전송되는 최대 크기를 지정합니다.

* **whatap.net_udp_port** <span class='type'>tcp\_port</span>

  기본값 `6600`

  와탭 에이전트는 트레이서에서 UDP를 통해 수집한 데이터를 수집 서버로 전송합니다. 처음 UDP 서버의 포트를 지정할 수 있습니다. 기본값으로 제공되는 6600포트가 사용 중일 때 이 옵션을 사용합니다. PHP 컴파일 설치로 *php.ini*에 whatap 항목이 있으면 *whatap.ini*가 아닌 *php.ini*에 추가하세요.
