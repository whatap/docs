설치 스크립트를 이용해 트레이서 즉 PHP 확장 모듈과 whatap-php 서비스 데몬을 삭제하세요. 이후 패키지(yum, apt-get) 삭제를 진행하고 필요에 따라서 */usr/whatap/php* 아래에 로그 파일 및 기타 파일을 삭제하세요.

## Linux / Unix

1. PHP 확장 모듈과 whatap-php 서비스를 삭제하세요.

    ```bash
    $ /usr/whatap/php/install.sh remove
    ```

1. 패키지를 삭제하세요.

    <Tabs>
    <TabItem value="rdct" label="Red Hat/CentOS" default>

        ```bash
        $ sudo yum remove whatap-php
        ```
    
    </TabItem>
    <TabItem value="dbub" label="Debian/Ubuntu">

        ```bash
        $ sudo apt-get purge whatap-php
        ```

    </TabItem>
    <TabItem value="frbsd" label="FreeBSD">

        ```bash
        $ pkg delete whatap-php
        ```

    </TabItem>
    </Tabs>

## Windows

:::caution

Windows에서 제거 시 웹 서버를 중지하고 제거 후 다시 시작하는 걸 권장합니다. 

:::

**GUI 제거**

```cmd
# 1. 웹 서버 중지 (권장)
net stop Apache2.4
REM 또는
iisreset /stop

# 2. 제거 프로그램 실행
"C:\Program Files\WhaTap\PHP\unins000.exe"

# 3. 웹 서버 시작
net start Apache2.4
REM 또는
iisreset /start
```

**CLI 자동 제거**

```cmd
# 1. 웹 서버 중지 (권장)
net stop Apache2.4
REM 또는
iisreset /stop

# 2. 무인 제거
"C:\Program Files\WhaTap\PHP\unins000.exe" /VERYSILENT

# 3. 웹 서버 시작
net start Apache2.4
REM 또는
iisreset /start
```

**예시. 배치 스크립트**

```batch
@echo off
echo Stopping web server...
net stop Apache2.4

echo Uninstalling WhaTap PHP Agent...
"C:\Program Files\WhaTap\PHP\unins000.exe" /VERYSILENT

timeout /t 3

echo Starting web server...
net start Apache2.4
echo Uninstallation completed!
```
