PHP 모니터링 서비스를 일시 중지하려면 PHP 확장 모듈을 중지하고, whatap-php 서비스를 중지해야 합니다. 

## PHP 확장 모듈 비활성화

### Linux/Unix

1. PHP 확장 모듈(PHP Extension module)을 중지하세요. 

    *whatap.ini* 파일의 `extension=` 구문을 주석 처리하세요. 주석은 세미콜론(*;*)으로 설정하세요.
    
    ```bash
    $ sudo vi whatap.ini

    extension=whatap.so

    ;주석
    ;extension=whatap.so
    ```

1. 수동 설정으로 *php.ini* 파일에 직접 설정한 경우도 동일하게 `extension=` 구문을 주석 처리하세요. 
    
    :::note
    
    Apache 또는 PHP-FPM 서비스(Service)를 시작합니다.

    :::

### Windows

whatap.ini 파일에서 extension 지시문을 주석 처리합니다.

**Scan INI Directory 사용 시**
```cmd
REM whatap.ini 파일 편집
notepad "C:\PHP\conf.d\whatap.ini"

REM extension 줄을 주석 처리
;extension=php_whatap.dll
```

**php.ini에 직접 추가한 경우**
```cmd
notepad "C:\PHP\php.ini"

REM [whatap] 섹션의 extension 줄을 주석 처리
[whatap]
;extension=php_whatap.dll
```

**웹 서버 재시작**
```cmd
REM Apache
net stop Apache2.4
net start Apache2.4

REM IIS
iisreset /restart
```

## WhaTap PHP 서비스 중지

whatap-php 서비스(Service)를 중지하세요.

<Tabs>
<TabItem value="rdct" label="Red Hat/CentOS" default>

    ```bash
    $ sudo service whatap-php stop
    ```
    
</TabItem>
<TabItem value="dbub" label="Debian/Ubuntu">

    ```bash
    $ sudo service whatap-php stop
    ```
    
</TabItem>
<TabItem value="frbsd" label="FreeBSD">

    ```bash
    $ service whatap_php stop
    ```
    
</TabItem>
<TabItem value="windows" label="Windows">

    ```cmd
    REM 서비스 중지
    net stop "WhaTap PHP"

    REM 또는
    sc stop "WhaTap PHP"
    ```
</TabItem>
</Tabs>  

## 서비스 재개

<Tabs>
<TabItem value="windows" label="Windows">

    ```cmd
    REM extension 주석 제거
    extension=php_whatap.dll

    REM 웹 서버 재시작
    net stop Apache2.4 && net start Apache2.4

    REM WhaTap 서비스 시작
    net start "WhaTap PHP"
    ```
