PHP 에이전트는 필요한 설정을 *whatap.ini* 파일에 작성하고 변경할 수 있습니다. 사용자의 편의성을 위해 [와탭 모니터링 서비스](https://service.whatap.io)에서도 에이전트 설정 기능을 제공하고 있습니다.

:::note

일반적으로 설정은 런타임에 바로 reload 됩니다. 다만 해당 설정의 메커니즘상 Apache 또는 PHP-FPM이 기동될 때 적용되는 옵션들은 다시 시작해야 할 수 있습니다.

:::
