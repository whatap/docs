## PHP 에이전트 지원 환경

와탭 PHP 모니터링 에이전트를 설치하기 전에 지원 환경을 확인하세요.

* **지원 버전**

    * PHP extension module로 동작
    * PHP 5.2 버전 이상, 7.0 버전 이상, 8.0 ~ 8.2 버전
    * NTS / ZTS (Zend Thread Safe)


* **운영체제**

    * CentOS/Red Hat 6.x (64bit) 이상
    * Debian/Ubuntu 12.04 (64bit) 이상
    * Alpine Linux (64bit)
    * FreeBSD 10.x (64bit) 이상

* **데이터베이스**

    * Oracle (OCI8)
    * MySQL
    * MSSQL (mssql, sqlsrv)
    * PostgreSQL (psql)
    * Redis (phpredis)
    * CUBRID
    * Tibero (odbc)

* **지원 라이브러리**

    * PDO
    * MySQLi
    * cURL
    * oci8
    * mssql
    * sqlsrv
    * psql
    * phpredis
    * CUBRID
    * odbc

* **애플리케이션 서버**

    * Apache
    * PHP-FPM (Nginx)
