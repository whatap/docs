## PHP 에이전트 지원 환경

와탭 PHP 모니터링 에이전트를 설치하기 전에 지원 환경을 확인하세요.

* **지원 버전**

  * PHP extension module로 동작
  * PHP 5 >= 5.2.0, PHP 7, PHP 8
  * NTS / ZTS (Zend Thread Safe)

* **Windows 환경 지원 버전**

  * PHP 7.0 ~ 7.4
  * PHP 8.0 ~ 8.4

  :::warning
  Windows는 PHP 5.x를 지원하지 않습니다.
  :::


* **운영체제**

  * CentOS/Red Hat 6.x (64bit) 이상
  * Debian/Ubuntu 12.04 (64bit) 이상
  * Alpine Linux (64bit)
  * FreeBSD 10.x (64bit) 이상
  * Linux-aarch64 (ARM64)
  * Windows Server 2016 이상
  * Windows 10 이상

  :::note

  모든 운영체제는 64비트만 지원합니다.  
  Windows 환경 지원은 v2.13.0부터 제공되며, PHP 7.0 이상만 지원합니다.

  ::: 

* **애플리케이션 서버**

  * Apache
  * PHP-FPM (Nginx)

* **애플리케이션 서버(Windows)**

  * Apache 2.4 이상
  * Microsoft IIS 8.5 이상 (Windows Server 2016+)

* **라이브러리**

    * PDO
    * MySQLi
    * cURL
    * oci8
    * mssql
    * sqlsrv
    * psql
    * phpredis
    * CUBRID
    * odbc

* **데이터베이스**

    * Oracle (OCI8)
    * MySQL
    * SQL Server (mssql, sqlsrv)
    * PostgreSQL (psql)
    * Redis (phpredis)
    * CUBRID
    * Tibero (odbc)
