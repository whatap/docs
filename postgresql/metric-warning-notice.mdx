## 템플릿{#template}

사용자가 손쉽게 경고 알림을 설정할 수 있도록 메트릭스 및 복합 메트릭스 이벤트에 대한 템플릿을 제공합니다.

### 메트릭스 이벤트 템플릿{#template-metrics}

<Cmdname sid="event_statistic" className="uitext" /> &gt; <Cmdname sid="event_setting" className="uitext" /> &gt; <Cmdname sid="metrics" className="uitext" /> 탭에서 <Cmdname sid="metrics" className="uitext" /> 섹션의 <span class="uitext">+</span> <Cmdname sid="add_notification" className="uitext" /> 버튼을 선택하세요. <Cmdname sid="template" className="uitext" /> 옵션에서 원하는 항목을 선택하세요. 다음은 선택할 수 있는 템플릿 목록입니다.

:::note

선택한 템플릿의 <Cmdname sid="metrics_event_rules" className="uitext" />은 사용자가 직접 수정할 수 있습니다.

:::

* <p><Cmdname sid="cpu_usage" className="uitext" /><span class="uitext">_Warning</span></p>

  조건: `cpu(xos2)` > 70

  개별 인스턴스에서 CPU 사용률이 70%를 초과하면 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.

* <p><Cmdname sid="memory_usage" className="uitext" /><span class="uitext">_Warning</span></p>

  조건: `mem(xos)` > 70

  개별 인스턴스에서 메모리 사용량이 70%를 초과하면 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.

* ***Active Session_Warning***

  조건: `active_sessions` > 30

  개별 인스턴스에서 활성화 중인 세션 수가 30개를 초과하면 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.

* ***Lock Wait_Warning***

  조건: `lock_wait_sessions` > 3

  개별 인스턴스에서 대기 중인 세션 수가 3개를 초과하면 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.

* ***Long Query_Warning***

  조건: `long_running_sessions` > 3

  개별 인스턴스에서 장시간 수행 중인 세션 수가 3개를 초과하면 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.

* <p><Cmdname sid="TTL07136" className="uitext" /><span class="uitext">_Warning</span></p>

  조건: (`total_sessions` / `max_connections` * 100) > 90

  개별 인스턴스에서 장시간 실행 중인 쿼리 수가 80%를 초과하면 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.

* <p><Cmdname sid="TTL07137" className="uitext" /><span class="uitext">_Warning</span></p>

  조건: `blks_hit_ratio` \<\= 80

  개별 인스턴스에서 Memory Block Hit율이 80% 이하인 경우 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.

* <p><Cmdname sid="TTL07138" className="uitext" /><span class="uitext">_Warning</span></p>

  조건: `replication_delay` \>\= 10

  개별 인스턴스에서 replication 데이터베이스에 반영되는 지연 시간이 10초 이상이면 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.

### 복합 메트릭스 이벤트 템플릿{#composite-metrics-template}

* ***Replication Broken***

  개별 에이전트의 Replcation 상태가 Broken 또는 Fallover 상태이면 <span class='vslow'>Critical</span>(위험) 수준의 알림을 보냅니다.

* <p><Cmdname sid="TTL07141" className="uitext" /></p>

  60초 이상 커밋하지 않은 세션이 2개를 초과하면 <span class='slow'>Warning</span>(경고) 수준의 알림을 보냅니다.
