<Cmdname sid="users_group_display_settings" className="uitext" />은 특정 사용자 그룹에게만 메뉴나 프리셋을 표시할 수 있도록 제한하는 기능입니다. 관리자는 표시 그룹을 생성한 뒤, 원하는 메뉴 또는 프리셋과 연결하고, 보기 허용 여부를 설정함으로써 사용자의 역할이나 권한에 따라 불필요한 항목은 숨기고 필요한 정보만 노출할 수 있습니다.

:::note

**데이터베이스 제품의 프로젝트에서 멤버 관리 권한이 있는 사용자**만 설정할 수 있습니다.

:::

## 기본 화면

<Cmdname sid="users_group_display_settings" className="uitext" /> 메뉴는 특정 프리셋, 메뉴, 기능에 대해 표시 여부를 설정할 수 있습니다.

### 보기 허용 설정

표시 그룹이 설정되지 않은 경우, 보기 허용의 기본값은 **전체**입니다. 토글을 활성화해 표시 그룹의 보기를 허용하세요. 

- **보기 그룹**: 설정된 표시 그룹에 포함된 사용자에게만 페이지가 표시

- **전체**: 모든 사용자에게 페이지가 표시(표시 그룹 설정과 무관)

### 보기 역할 추가

표시 그룹을 추가하면서 이 표시 그룹은 어떤 페이지에 보기 역할을 부여할 것인지 설정할 수 있습니다.

1. **사용자 그룹 별 표시 설정** 화면의 오른쪽 위의 **+ 보기 역할 추가** 버튼을 클릭합니다.

2. **표시 그룹 생성**에서 표시 그룹 이름, 표시 그룹 설명을 작성합니다.

    a. **+ 추가** 버튼을 클릭해 이메일 또는 사용자 이름으로 사용자를 추가합니다.

    b. 생성할 표시 그룹에 추가할 메뉴 / 프리셋 또는 기능의 토클을 활성화해 **표시 그룹**에 추가합니다.

3. **확인** 버튼을 클릭해 **표시 그룹**을 생성합니다.

## 표시 그룹 추가

1. **사용자 그룹 별 표시 설정**에서 메뉴/프리셋에 표시 그룹을 연결하려면 **메뉴 / 프리셋**의 **표시 그룹**을 클릭합니다.

2. 드롭 다운에서 + **표시 그룹 생성** 클릭합니다.

3. **표시 그룹 생성**에서 표시 그룹 이름, 표시 그룹 설명을 작성합니다.

    a. **+ 추가** 버튼을 클릭해 이메일 또는 사용자 이름으로 사용자를 추가합니다. 

    b. 생성할 표시 그룹에 추가할 **메뉴 / 프리셋** 또는 기능의 토클을 활성화해 **표시 그룹**에 추가합니다. 

4. **확인** 버튼을 클릭해 표시 그룹을 생성합니다. 

5. 생성된 표시 그룹을 체크한 후, **적용** 버튼을 클릭해 적용합니다. 

## 표시 그룹 수정

1. 표시 그룹을 추가하고 싶은 **메뉴 / 프리셋**의 **표시 그룹**을 클릭합니다.

2. 드롭 다운에서 수정할 표시 그룹의 ![수정 아이콘](/img/ic-edit.svg) 아이콘을 클릭합니다. 

## 표시 그룹 삭제

1. 삭제하고 싶은 표시 그룹이 있는 **메뉴 / 프리셋**의 **표시 그룹**을 클릭합니다.

2. 드롭 다운에서 삭제할 **표시 그룹**의 ![수정 아이콘](/img/ic-edit.svg) 아이콘을 클릭합니다. 

3. **표시 그룹 삭제** 버튼을 클릭합니다. 

4. **적용** 버튼을 클릭하면 표시 그룹이 삭제됩니다. 

    - 표시 그룹이 없는 경우, 모든 페이지와 기능이 표시됩니다.
