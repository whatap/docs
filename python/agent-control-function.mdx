Python 애플리케이션 서버의 기능을 제어합니다.

* **enabled** <span class='type'>Boolean</span>

  기본값 `true`

  전체 기능을 활성화합니다. 값이 `false`인 경우에도 서버와 최소한의 통신을 유지하기 위한 정보는 전송됩니다.

* **counter_enabled** <span class='type'>Boolean</span>

  기본값 `true`

  성능 카운터 추적을 활성화합니다. 성능 카운터는 액티브 트랜잭션, 사용자, CPU, Memory, SQL, 외부 HTTP 호출 등의 데이터를 포함합니다. `enabled` 값이 `false`면 무시됩니다.

* **stat_enabled** <span class='type'>Boolean</span>

  기본값 `true`

  통계 정보 추적을 활성화합니다. 5분마다 트랜잭션, SQL Query, 외부 HTTP 호출, User Agent, Client IP 등의 통계 데이터가 수집되는데 이들 정보의 수집이 중단됩니다. `enabled` 값이 `false`면 무시됩니다. 


* **active_stack_enabled** <span class='type'>Boolean</span>

  기본값 `true`

  액티브 스택 추적을 활성화합니다.

  :::note

  `counter_enabled` 값이 비활성화된 경우 액티브 트랜잭션 데이터가 수집되지 않기에 액티브 스택 또한 수집되지 않습니다.
  
  :::

* **eactive_stack_second** <span class='type'>Seconds</span>

  기본값 `10`

  액티브 스택을 추적하는 간격을 설정합니다.

  :::caution

  값을 바꾸는 것을 권장하지 않습니다.
  
  :::

* **realtime_user_thinktime_max** <span class='type'>MiliSeconds</span>

  기본값 `300000`

  실시간 사용자를 측정할 때 동일 사용자로 인정되는 최대 호출 간격을 지정합니다.

* **time_sync_interval_ms** <span class='type'>MiliSeconds</span>

  기본값 `300000`

  에이전트와 서버 간 시간 동기화 주기를 지정합니다. 동기화하지 않을 경우 0으로 지정합니다.  

* **realtime_user_thinktime_max** <span class='type'>Int</span>

  기본값 `0`

  와탭 에이전트는 한번 보낸 텍스트 유형 데이터는 hash 처리되므로 다음날까지 다시 전송하지 않습니다. 이전 설정값과 다른 값을 설정하는 경우 다시 전송합니다.

  :::note

  트랜잭션 URL, SQL String 등이 텍스트 유형 데이터에 해당합니다.

  :::
