Python 에이전트의 HTTPC, API Call 관련 옵션을 제공합니다.

* **profile_httpc_resource_enabled** <span class='type'>Boolean</span>

  기본값 `false`

  트레이스에서 HTTP Call 스텝이 수집될 때 해당 스텝에서 사용한 CPU와 Memory 사용량을 추적합니다.

* **trace_httpc_normalize_enabled** <span class='type'>Boolean</span>

  기본값 `true`

  외부 HTTP 호출(HTTP Call) URL을 파싱하여 정규화하는 기능을 활성화합니다. 

* **trace_httpc_normalize_urls** <span class='type'>String</span>

  외부 HTTP 호출하는 URL을 파싱하여 정규화합니다. 호출 URL 패턴을 파싱하여 패스 파라미터를 제거합니다. 
  
  예를 들면, `/a/{v}/b`라고 선언하면 `/a/123/b` → `/a/{v}/b`로 치환합니다. 여러 개를 등록할 때는 콤마(*,*)를 사용합니다. `/a/*/b`로 사용하여 특정 형식의 패스 파라미터를 제거하여 수집할 수 있습니다.
