애플리케이션 서버가 실행되면 모니터링 정보를 수집합니다. 수집한 정보는 로그 파일에 저장합니다.

## 로그 설정

* **log_rotation_enabled** <span class='type'>Boolean</span>

  기본값 `true`

  에이전트 로그 파일을 매일 변경합니다. 로그 파일의 기본 경로는 *$WHATAP_HOME/logs/whatap-xxx.log* 입니다.

* **log_keep_days** <span class='type'>Day</span>

  기본값 `7`

  로그 파일 보관 기간을 설정합니다.
