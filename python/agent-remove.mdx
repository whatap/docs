Python 모니터링 에이전트를 삭제하기 위해 다음 과정을 진행해 주세요.

1. 애플리케이션 서버 재시작 시 함께 추가한 `WHATAP_AGENT` 시작 명령어를 삭제합니다.

1. `whatap-stop-agent` 명령어로 와탭 Python 에이전트 프로세스를 종료합니다.

    ```javascript title=SH
    $ whatap-stop-agent
    ```

1. 프로세스가 죽지 않는다면 다음 명령어를 통해 프로세스를 종료시켜주세요.

    ```javascript title=SH
    $ killall whatap-python
    ```

1. 프로세스가 정상적으로 죽었는지는 다음 명령어로 확인할 수 있습니다.

    ```javascript title=SH
    $ ps -ef|grep whatap_python
    ```
