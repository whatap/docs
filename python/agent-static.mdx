에이전트가 수집한 데이터의 통계 수집 기능과 관련한 옵션입니다.

* **stat_domain_enabled** <span class='type'>Boolean</span>

  기본값 `false`

  도메인별 트랜잭션 통계를 수집합니다.

* **stat_domain_max_count** <span class='type'>Int</span>

  기본값 `7000`

  실5분 동안 애플리케이션에서 수집할 도메인별 트랜잭션 통계의 최대 레코드 수입니다.

* **stat_mtrace_enabled** <span class='type'>Boolean</span>

  기본값 `false`

  멀티 서버 트랜잭션에서 Callser&Callee 간에 버전별 의존성 통계를 수집합니다.

* **stat_mtrace_max_count** <span class='type'>Int</span>

  기본값 `7000`

  멀티 서버 트랜잭션에서 Callser&Callee 간에 버전별 의존성 통계의 최대 레코드 수입니다.
  
* **stat_login_enabled** <span class='type'>Boolean</span>

  기본값 `false`

  로그인별 트랜잭션 통계를 수집합니다.

* **stat_login_max_count** <span class='type'>Int</span>

  기본값 `7000`

  5분 동안 애플리케이션에서 수집할 로그인별 트랜잭션 통계의 최대 레코드 수입니다.

* **stat_referer_enabled** <span class='type'>Boolean</span>

  기본값 `false`

  Referer 별 트랜잭션 통계를 수집합니다.

* **stat_referer_max_count** <span class='type'>Int</span>

  기본값 `7000`

  5분 동안 애플리케이션에서 수집할 Referer 별 트랜잭션 통계의 최대 레코드 수입니다.

* **stat_tx_max_count** <span class='type'>Int</span>

  기본값 `5000`

  트랜잭션 통계 정보의 개수를 제한합니다. 5분 동안 수집되어 서버에 전송되는 통계정보에서 최대 레코드 수를 제한합니다.

* **stat_httpc_max_count** <span class='type'>Int</span>

  기본값 `5000`

  Http Call 통계 정보의 개수를 제한합니다. 5분 동안 수집되어 서버에 전송되는 통계정보에서 최대 레코드 수를 제한합니다.

* **stat_error_max_count** <span class='type'>Int</span>

  기본값 `1000`

  Error 통계 정보의 개수를 제한합니다. 5분 동안 수집되어 서버에 전송되는 통계정보에서 최대 레코드 수를 제한합니다.

* **stat_useragent_max_count** <span class='type'>Int</span>

  기본값 `500`

  User Agent 통계 정보의 개수를 제한합니다. 5분 동안 수집되어 서버에 전송되는 통계정보에서 최대 레코드 수를 제한합니다.
