와탭 Python 애플리케이션 모니터링은 Python 기반 웹 애플리케이션 서버 모니터링 서비스를 제공합니다. Web framework, Server, Batch를 한 번에 모니터링하는 국내 유일 상품입니다.

주요 웹 프레임워크의 애플리케이션에 적용 가능하며, DB Query, 외부 호출을 추적하여 어느 곳에서 호출이 지연되고 있는지 찾을 수 있습니다. http 혹은 https가 아닌 특정한 트랜잭션에 대해 데이터 추적을 하고 싶은 경우, 특정 메소드 모니터링이 가능합니다.
