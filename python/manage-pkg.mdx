## 패키지 설치 확인

다음 명령어를 통해 와탭 Python 에이전트 패키지의 설치 상태를 확인할 수 있습니다.

```javascript title=SH
$ pip list | grep whatap-python
```

## 패키지 업데이트{#update-pkg}

다음 명령어를 통해 와탭 Python 에이전트 패키지를 업데이트할 수 있습니다.

```javascript title=SH
$ pip install -U whatap-python
```

특정 버전을 명시한 경우 다음을 참고하세요.

```javascript title=SH
$ pip install -U whatap-python==[특정 버전]
```

## 패키지 제거

다음 명령어를 통해 와탭 Python 에이전트 패키지를 제거할 수 있습니다.

```javascript title=SH
$ pip uninstall whatap-python
```
