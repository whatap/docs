와탭 에이전트는 에이전트 별 필요한 설정을 *whatap.conf* 파일에 작성합니다. 에이전트는 환경변수를 통해 설정 파일의 위치를 파악하고 로딩합니다. 사용자의 편의성을 위해 [와탭 모니터링 서비스](https://service.whatap.io)에서도 에이전트 설정 기능을 제공하고 있습니다.
