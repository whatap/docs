애플리케이션 내부에서 사용되는 LLM API의 데이터를 수집합니다. 수집한 데이터는 로그 모니터링과 매트릭스 차트에서 확인할 수 있습니다. 

:::note

**요구사항**

와탭 Python 에이전트 1.8.12 버전 이상이 필요합니다.

:::

## 옵션 설정

- **trace_llm_log_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    LLM 로그 모니터링 수집 기능을 활성화합니다.

- **counter_thread_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    Python 에이전트의 메트릭스 전송 스레드를 활성화합니다.

- **llm_stat_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    LLM 메트릭스 수집 기능을 활성화합니다. **trace_llm_log_enabled**, **counter_thread_enabled** 옵션도 함께 `true` 설정되어 있어야 활성화됩니다.

- **llm_stat_interval** <span class='type'>Int</span>

    기본값 `5` (초 단위)

    LLM 메트릭스의 수집 주기를 설정합니다.
