## Python 에이전트 지원 환경

와탭 Python 모니터링 에이전트를 설치하기 전에 지원 환경을 확인하세요.

* **지원 버전**

  * WSGI Application (Python 2.7 & Python 3.3 이상)

  * ASGI Application (Python 3.7 이상)

* **운영체제**

  * Red Hat/CentOS 6.x (64bit) 이상

  * Debian/Ubuntu 12.04 (64bit) 이상

  * OSX 10.15 Catalina

* **애플리케이션 서버**

  * Django
  
  * Bottle
  
  * CherryPy
  
  * Flask
  
  * Tornado
  
  * Nameko
  
  * Celery
  
  * FastAPI(0.84.0 이상)

* **라이브러리**

  * 외부 호출 : HttpLib, Requests, Urllib

  * Kombu, Pika

* **데이터베이스**

  * MySQL
  
  * SQL Server
  
  * PostgreSQL
  
  * Redis
