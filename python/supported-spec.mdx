## Python 에이전트 지원 환경

와탭 Python 모니터링 에이전트를 설치하기 전에 지원 환경을 확인하세요.

* **운영체제**

    * CentOs 6 이상(64bit)
    * Ubuntu 14 이상(64bit)

* **지원 버전**

    * Python 2.7 이상 & Python 3.3 이상
    * WSGI Application

* **지원 라이브러리**

    * 외부 호출 : HttpLib, Requests, Urllib
    * 데이터베이스 : MySQL, PostgreSQL

* **애플리케이션 서버**

    * Django, Bottle, CherryPy, Flask, Tornado
