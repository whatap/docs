3. 압축을 해제한 폴더로 진입해서 *whatap.conf* 파일을 확인하세요. *whatap.conf* 파일에 프로젝트 액세스 키(`license`) 정보 및 와탭 서버 정보(`whatap.server.host`), DB 접속 정보(`dbms`, `db_ip`, `db_port`)를 입력하세요.

    ```ini title='whatap.conf'
    license={Access_Key}
    whatap.server.host=13.124.11.223/13.209.172.35 # WhaTap server information

    dbms=redis
    db_ip={DB_server_IP_address}
    db_port={DB_server_port}
    ```
