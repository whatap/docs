와탭 모니터링 서비스에서 모니터링 데이터에 대한 수집 및 보존 기간을 안내합니다.

## 데이터 수집

| 조회 범위 | 기본 조회 | 선택 조회 |
| --------- | --------- | --------- |
| 3시간 이하 | RAW 데이터(실시간 데이터) | 5분 데이터 / 1시간 데이터 |
| 3시간 초과, 15일 이하 | 5분 데이터 | 1시간 데이터 |
| 15일 초과 | 1시간 데이터 | - |

## 데이터 보존 기간

와탭의 모든 상품별 데이터 보존 기간은 다음과 같이 제공합니다.

- RAW 데이터(실시간 데이터) 보존 기간: **1개월**

- 통계 데이터(5분/1시간) 보존 기간: **1년**

- 프로파일 데이터 보존 기간: **1개월**

:::note

* **Network** 상품의 경우, 통계 데이터(5분/1시간) 보존 기간은 **100일**입니다.

* 로그 모니터링 기능을 활성화한 경우, 로그 데이터의 보존 기간은 기본 **1일**입니다. 사용자가 원하는 기간으로 변경할 수 있습니다. 자세한 내용은 [다음 문서](../log/log-setting#log-data-setting)를 참조하세요.

:::
