## v1.2.1

2022년 12월 11일

### 코드 오류로 수집서버와 연결 단절시 Task 다운되는 현상 수정  

와탭 에이전트 코드 오류로 수집 서버와 연결 단절시 Task 다운되는 현상을 아래와 같이 수정했습니다. 
* 에이전트가 다운되는 경우 Task 실행 스크립트에 에이전트를 안전하게 재실행하는 코드 추가
* 수집 서버와 연결 단절 시 재접속 시도 코드 오류(비동기 루틴) 수정
