2023년 12월 09일

## 공통

- <Status>Changed</Status> Windows 환경에 설치 시 필요한 파일을 에이전트 설치 파일에 포함: *instsrv.exe*, *srvany.exe*

- <Status>Fixed</Status> *install_windowsService.bat* 파일 버그 수정(jdbc 디렉토리의 JAR 파일을 읽지 못하는 문제)

## PostgreSQL

- <Status>Fixed</Status> 테이블 레이아웃 컬럼 수정
- <Status>Fixed</Status> <code>pg_stat_statements</code> 수집 시 부하 감소 및 버그 수정

## MySQL

- <Status>Feature</Status> <code>trx_rseg_history_len</code>(MySQL 5.7 버전 이상 지원), <code>active_transactions</code>(MySQL 5.5 버전 이상 지원) 지표 추가

  다음과 같이 *whatap.conf* 파일 설정 옵션을 추가해야 합니다.

  ```ini title="whatap.conf"
  trx_rseg_history_len=true
  active_transactions=true
  ```

- <Status>Feature</Status> DB 파라미터를 5초마다 수집하는 기능 추가

  ```ini title="whatap.conf"
  collect_variables_metric=true
  variables_metric=
  # e.g variables_metric=uptime,version
  ```

- <Status>Feature</Status> plan 전송 시 json 포맷 전송 추가

## Oracle

<Status>Fixed</Status> 테이블 레이아웃 컬럼 수정
