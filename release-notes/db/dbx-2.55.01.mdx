2025년 07월 16일

### 공통

- <Status priority='1'>New</Status> 쉘 스크립트 실행을 지원하는 `shellmon` 기능 추가
- <Status priority='1'>Feature</Status> Agentinfo에 DB 종류를 식별하는 `sub_type` 필드 추가
- <Status priority='1'>Feature</Status> Autoscale 에이전트도 `AgentinfoPack` 전송 가능하도록 기능 확장

### MySQL

- <Status priority='0'>Change</Status> `max_connection` 지표 수집 주기를 5분 간격으로 변경
- <Status priority='0'>Fixed</Status> MySQL 5.6 버전에서 액티브 세션 수집 시 발생하던 오류 수정
- <Status priority='0'>Fixed</Status> slave DB 손상 시 `node_state` 값을 2에서 -2로 변경되도록 처리 로직 개선
- <Status priority='0'>Fixed</Status> 타임스탬프 형식에 따라 슬로우 쿼리 텍스트가 누락되는 문제 수정

### PostgreSQL

- <Status priority='0'>Fixed</Status> 타임스탬프 형식에 따라 슬로우 쿼리 텍스트가 누락되는 문제 수정
