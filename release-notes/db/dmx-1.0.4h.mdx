2025년 04월 03일

- <Status priority='1'>New</Status> 한글 깨짐 시 코드 변환 처리 옵션 추가

  * `codefrom=` 및 `codeto=` 문자 인코딩 지정

  * `iconv -l` 명령어의 결괏값 중 선택하여 지정

  * 예시: `codefrom=EUC-KR`, `codeto=UTF-8`

- <Status priority='1'>Feature</Status> `plan_elapse` `bind_elapse` 기본값 5초 → 3초로 변경

- <Status priority='1'>Feature</Status> `sqlstat` 전송 주기의 기본값 5분 → 1분으로 변경

- <Status priority='1'>Feature</Status> `sqlstat`에 `!rectype`, `@id`, `@stime` 추가

- <Status priority='0'>Fixed</Status> 19 이상의 버전에서 Parallel query 정보가 수집되지 않는 문제 수정

- <Status priority='0'>Fixed</Status> pdb가 하나일 때 multi DB 별로 지표가 조회되지 않는 문제 수정

- <Status priority='0'>Fixed</Status> 에이전트 실시간 호출하는 `session list request/reply`에 `result2` 추가

- <Status priority='0'>Fixed</Status> `temp tablespace` 수치가 비정상적으로 나오는 문제 수정

  * Oracle 버그(18c, 19c) 영향으로, `distinct`로 수집 쿼리 로직 수정

- <Status priority='0'>Fixed</Status> 5분 데이터 생성이 불필요한 `_no_5m_hour_` 카테고리 정리

- <Status priority='0'>Fixed</Status> `plan_hash` 값으로 플랜 실시간 조회 시 `child_number`가 없을 경우 `timestamp` 최신값으로 조회하도록 수정
