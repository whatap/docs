2025년 06월 10일

* <Status priority='1'>New</Status> 쉘 스크립트를 주기적으로 수행하여 결과값을 수집할 수 있는 shmon 기능 추가

  * xos.conf 에 shmon=1 로 사용

  * agent 디렉토리 하위에 shmon 디렉토리를 만들어 .sh 파일을 두면 주기적 실행 및 결과 수집 가능

* <Status priority='1'>New</Status> AIX 머신에서 Oracle Pro 11.2.0.4, 19.3 지원 가능

* <Status priority='1'>Feature</Status> script manager thread 로 처리

* <Status priority='1'>Feature</Status> sql trace 시 tkprof 수행을 thread 로 처리

* <Status priority='1'>Feature</Status> trace_tkprof_path= 에이전트 옵션 추가

* <Status priority='0'>Fixed</Status> event 수집시  ‘event in waitclass Other’ 수집 제외 (간혹 큰값이 들어오는 오라클 버그가 있음)

* <Status priority='0'>Fixed</Status> planchange, plansave 시 11g 이하에서 sql 오류로 인해 수집되지 않는 문제
