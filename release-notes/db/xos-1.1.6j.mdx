2023년 10월 13일

- <Status>New</Status> 프로세스(process) 감시 기능 추가, 프로세스의 실행 상태를 모니터링 가능

  ```ini title='xos.conf'
  process={process_name}
  ```

- <Status>New</Status> 데이터베이스 서버의 IP 목록 수집, 초 단위(최소 3초)로 시간 간격(interval)을 설정해야 수집 가능

  ```ini title='xos.conf'
  iplist_interval={second}
  ```
