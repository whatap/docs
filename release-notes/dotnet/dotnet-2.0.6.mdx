#### 2023년 5월 2일

- <Status>New</Status> StackExchcange Redis 최신 라이브러리 추가 적용 (2.x)

- <Status>New</Status> Oracle.ManagedDataAccess 최신 라이브러리 추가 적용 (19.19.0 버전까지)

- <Status>New</Status> 특정 응용 애플리케이션 풀만 선별하여 추적하기

  * **application_pool_filter_enabled** <span class='type'>Boolean</span>

    기본값 `false`
  
    `application_pool_filter` 옵션에 등록한 애플리케이션만 추적하려면 `true`로 설정하세요.

  * **application_pool_filter** <span class='type'>String</span>

    추적하고 싶은 애플리케이션의 이름을 입력하세요. 쉼표(,)를 구분자로 이용해 복수 설정할 수 있습니다.

    ```ini title='whatap.conf'
    application_pool_filter="DefaultAppPool, userSiteAppName"
    ```

- <Status>Fixed</Status> <b>ASP.NET Web API</b>를 추적하지 못하는 버그 수정

- <Status>Fixed</Status> 액티브 트랜잭션을 추적하지 못하는 버그 수정

- <Status>Fixed</Status> 트랜잭션 ID가 가끔 중복되는 버그 수정

- <Status>Fixed</Status> 복수의 스레드가 같은 로그 파일에 동시에 접근하여 로그가 무시되는 현상 수정

- <Status>Fixed</Status> 사용자 수가 측정되지 않는 버그 수정

  [.NET 설정하기 - 사용자 수](../../dotnet/agent-number-of-user) 참조

- <Status>Fixed</Status> <code>trace_httpc_normalize_urls</code> 옵션이 동작하지 않는 버그 수정

  [.NET 설정하기 - HTTPC, API Call](../../dotnet/agent-httpcapicall) 참조
