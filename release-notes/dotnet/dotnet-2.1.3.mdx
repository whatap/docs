#### 2023년 8월 16일

* <Status>Change</Status> Application pool 선택 방식 변경

  - 기존: 모니터링 모듈이 메모리에 올라온 상태에서 기능이 동작하지 않도록 필터링

  - 변경

    - 대상 애플리케이션 풀이 아니면 모니터링 모듈을 로딩하지 않음
  
    - 모니터링에 관련된 아무런 작업을 하지 않기 때문에 고객의 .NET 애플리케이션에 어떠한 영향도 주지 않음
  
    - 설정을 변경하면 IIS를 재시작해야하는 것으로 조건 변경

* <Status>Change</Status> 서버 지원 환경 분석 툴의 .NET 버전을 4.61 버전으로 하향 조정
