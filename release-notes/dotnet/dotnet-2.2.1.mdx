2024년 04월 09일

- <Status>Feature</Status> 에이전트 옵션 추가

  * **prepend_app_pool_name** <span class='type'>Boolean</span>

    기본값 `false`

    옵션값을 `true`로 설정하면 트랜잭션의 URL 앞에 애플리케이션의 전체 이름을 표시합니다.

- <Status>Changed</Status> 오래된 버전의 호환성을 높이기 위해서 설치 파일 분리
  
  - .NET 4.6.1 버전 이상의 런타임을 사용하는 경우: [whatap_dotnet.exe](https://repo.whatap.io/windows/whatap_dotnet.exe)

  - .NET 4.6.1 버전 미만의 런타임을 사용하거나 Windows Server 2012 미만의 경우: [whatap_dotnet_450.exe](https://repo.whatap.io/windows/whatap_dotnet_450.exe)

    :::note

    Windows Server 2008부터 지원합니다.
    
    :::

- <Status>Fixed</Status> <code>webservice_method_enabled</code> 옵션: 메소드가 중첩으로 실행되는 경우 모니터링되지 않는 버그 수정

- <Status>Fixed</Status> 32비트 닷넷 런타임에서 <em>whatap.conf</em> 파일을 찾지 못하는 문제 수정
