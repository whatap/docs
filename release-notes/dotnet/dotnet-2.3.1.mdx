2024년 08월 26일

- <Status priority='1'>New</Status> 트랜잭션 외부에서 일어나는 트레이스 정보 수집 기능 추가

  * **expand_transaction_level** <span class='type'>Boolean</span>

    기본값 `false`

    옵션값을 `true`로 설정하면 트랜잭션 외부에서 발생하는 SQL, HTTP 호출을 트랜잭션으로 취급해서 히트맵에 표시합니다.

  * **expand_transaction_basetime** <span class='type'>Int</span>

    기본값 `1000`(1초) / 최소값 `100`(0.1초)

    `expand_transaction_level` 옵션이 `true`인 경우, 설정한 시간보다 응답 시간이 오래 걸릴 때에만 데이터를 수집합니다. 초당 최대 100건까지 수집합니다.

- <Status priority='1'>New</Status> DB 연결 정보 표시: 데이터베이스를 사용하는 트랜잭션은 데이터베이스 연결 정보를 표시하는 기능 추가

- <Status priority='1'>Changed</Status> 오류가 아닌 경우에도 트랜잭션의 상태 코드를 표시하도록 수정
