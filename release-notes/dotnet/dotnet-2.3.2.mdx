2024년 09월 05일

- <Status priority='1'>New</Status> 트랜잭션 수집에서 제외할 URL의 접미사(suffix)를 설정할 수 있는 에이전트 옵션 추가 

  **trace_ignore_url_suffix** <span class='type'>String</span>

  설정한 접미사와 일치하는 URL은 트랜잭션 성능을 수집하지 않습니다. 여러 값을 설정하려면 쉼표(,)로 구분하세요.

- <Status priority='0'>Fixed</Status> Parameterized Query 추적 기능 추가

  :::note

  이 기능과 관련한 에이전트 옵션(`profile_sql_param_enabled`)은 [다음 문서](../../dotnet/agent-dbsql)를 참조하세요.

  :::

- <Status priority='0'>Fixed</Status> REST API 트랜잭션의 상태 코드(status code) 표시에서 발생하는 오류 수정

- <Status priority='0'>Fixed</Status> `System.Data.Common.DbCommand` 관련 메서드 중 모니터링에 포함되지 않았던 메서드 추가

  다음의 비동기 메서드 중에서 오버로딩된 메서드의 일부가 누락되어 추가하였습니다.
  
  - `ExecuteReaderAsync`
  - `ExecuteNonQueryAsync`
  - `ExecuteScalarAsync`
