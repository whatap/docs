2024년 11월 25일

- <Status priority='1'>New</Status> 에이전트 옵션 추가

  **webservice_method_prefix_realtime** <span class='type'>Boolean</span>

  기본값 `false`

  값을 `true`로 설정하면 `webservice_method_prefix` 대상을 이전과 같이 실시간으로 처리합니다.

- <Status priority='1'>Changed</Status> `webservice_method_prefix` 옵션을 통해 수집하는 트랜잭션 처리 방식 변경

  - 재귀적인 호출에 의한 오류 발생을 방지하기 위해 트랜잭션 안에서 수집되는 다른 `webservice_method_prefix` 메소드는 무시하도록 수정

  - 완료되지 않은 트랜잭션은 무시하도록 수정

- <Status priority='1'>Changed</Status> 개별로 다운로드를 제공했던 .NET Framework와 .NET Core 환경의 설치 프로그램 병합
