#### 최신 버전: 0.5.0

## 자동 계측 도구 (whatap-go-inst)

소스코드를 직접 수정하지 않고 빌드 시점에 모니터링 코드를 자동 삽입하는 CLI 도구입니다.

### Go install
```bash
go install github.com/whatap/go-api-inst/cmd/whatap-go-inst@latest
```

### 바이너리 다운로드

| OS | 아키텍처 | 다운로드 |
| - | :-: | :-: |
| Linux | x86_x64 | [다운로드](https://github.com/whatap/go-api-inst/releases/latest/download/whatap-go-inst_linux_amd64.tar.gz) |
| \^ | arm64 | [다운로드](https://github.com/whatap/go-api-inst/releases/latest/download/whatap-go-inst_linux_arm64.tar.gz) |

## go-api (import library)

Go 애플리케이션에 import하여 사용하는 모니터링 라이브러리입니다.
```bash
go get github.com/whatap/go-api@latest
```
