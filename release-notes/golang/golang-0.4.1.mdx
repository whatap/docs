2024년 08월 28일

- <Status priority='1'>New</Status> Standard output 로그 수집

  로그 데이터를 적당량으로 모아서 압축 전송합니다.

  ```ini title="whatap.conf"
  logsink_enabled=true
  logsink_stdout_enabled=true
  logsink_zip_enabled=true
  ```

- <Status priority='1'>New</Status> 에이전트 설정 파일(*whatap.conf*)보다 환경 변수를 우선 처리하는 옵션 추가

  - 기존 에이전트 설정은 *whatap.conf*, 환경 변수 순서로 적용합니다.

  - `use_env_first` (기본값 `false`) 옵션을 설정하면 *whatap.conf* 파일의 설정보다 환경 변수를 우선 적용합니다.

    ```ini title="whatap.conf"
    use_env_first=true
    ```

  - `WHATAP_USE_ENV_FIRST=true` 형식으로 환경 변수를 이용해 설정할 수 있습니다.

- <Status priority='1'>Changed</Status> 다음 에이전트 옵션의 기본값 수정

  - TCP read timeout 시간을 30초에서 120초로 변경합니다.

    ```ini title="whatap.conf"
    net_so_time=120000
    ```

  - TCP write buffer 값을 8Mb에서 2Mb로 변경합니다.

    ```ini title="whatap.conf"
    net_write_buffer_size=2*1024*1024
    ```

- <Status priority='0'>Fixed</Status> `OKIND` 환경 변수 인식 오류 수정

  `OKIND` 또는 `WHATAP_OKIND` 환경 변수로 `okind`를 설정할 수 있습니다.

:::info
