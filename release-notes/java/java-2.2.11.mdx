2023년 7월 10일

- <Status>New</Status> mongodb-reactive 추적

  mongodb-3.8.2, mongodb-4.0.3, mongodb-4.4, mongodb-4.8 추적

  ```ini title='whatap.conf'
  # 적용 예시 (중복 적용 불가)
  weaving=mongodb-3.8.2
  ```

- <Status>Change</Status> <code>trace_ignore_url_prefix</code> 설정에 쉼표(,)를 구분자로 여러 개의 prefix를 등록할 수 있도록 수정

  ```ini title='whatap.conf'
  # 적용 예시
  trace_ignore_url_prefix=/api,/sample
  ```
