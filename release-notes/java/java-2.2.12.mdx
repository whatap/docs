2023년 7월 21일

- <Status>New</Status> <b>ignore_http_method</b> 옵션 추가

  - `ignore_http_method` 옵션에 등록한 `http_method` 요청 트랜잭션 무시

  - 별도값을 설정하지 않을 경우 기본값으로 {PATCH, OPTIONS, HEAD, TRACE} 요청 트랜잭션 무시

  ```ini title='whatap.conf'
  # 예시. http_method가 OPTIONS, HEAD인 트랜잭션 "추적"
  ignore_http_method=PATCH,TRACE

  # 예시. http_method가 OPTIONS, HEAD인 트랜잭션 "무시"
  ignore_http_method=OPTIONS,HEAD
  ```

- <Status>Fixed</Status> <code>http_method</code> 옵션이 GET, PUT, POST, DELETE가 아닌 경우 GET으로 표시하는 문제 수정

- <Status>Fixed</Status> method 추적 시 발생하는 ASM api 버전 에러 수정

- <Status>Fixed</Status> AppServiceStart.x 플러그인 사용 시 중복 호출되는 문제 수정
