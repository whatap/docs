2023년 8월 31일

- <Status>Feature</Status> kafka-clients-2.4.0 추적

  ```ini title='whatap.conf'
  weaving=kafka-clients-2.4.0
  ```

- <Status>Feature</Status> 특정 에러 코드를 가진 <code>java.sql.SQLException</code> 무시 기능 추가

  ```ini title='whatap.conf'
  # example
  ignore_sql_error_code_set=22001,2009
  ```

- <Status>Feature</Status> 트랜잭션에서 에러 스택 표시 기능 추가: 트랜잭션에서 여러 개의 에러 발생 시 첫번째 에러의 스택 표시

  트랜잭션 에러 스택에 대한 자세한 내용은 다음 문서를 참조하세요.

  <Link to='../../java/agent-transaction-error-stack' target='_blank' className='ext-link'>트랜잭션 에러 스택</Link>

- <Status>Changed</Status> <code>java.sql.SQLException</code> 발생 시 에러 코드 표시 기본값을 <code>true</code>로 수정

  ```ini title='whatap.conf'
  # default
  trace_sql_errorcode_enabled=true
  ```

- <Status>Changed</Status> 사용자 정의 exception을 우선하도록 <code>trace_exception_overwrite_enabled</code> 기본값을 <code>true</code>로 수정

  ```ini title='whatap.conf'
  # default
  trace_exception_overwrite_enabled=true
  ```

- <Status>Changed</Status> <code>java.util.Date</code> 타입의 SQL 파라미터 데이터 수집

- <Status>Fixed</Status> Java 에이전트의 logsink 기능 사용 시 로그가 <code>null</code>인 경우에 대한 처리 추가
