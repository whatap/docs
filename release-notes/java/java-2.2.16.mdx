2023년 9월 8일

- <Status>Feature</Status> Java 20 지원 (자바 옵션 필요)

  ```bash
  --add-opens=java.base/java.lang=ALL-UNNAMED
  ```

- <Status>Feature</Status> Redis의 Lettuce 드라이버 추적

  ```ini title='whatap.conf'
  weaving=lettuce-6.2
  ```

- <Status>Feature</Status> spring-boot-3.0 설정에 Lettuce 설정 포함

  ```ini title='whatap.conf'
  weaving=spring-boot-3.0
  ```

- <Status>Changed</Status> <code>oracle.jdbc.driver.OraclePreparedStatement</code> 추적 제외하는 옵션 추가

  ```ini title='whatap.conf'
  # default
  hook_pstmt_oracle_enabled=true
  ```

- <Status>Fixed</Status> kafka-clients-2.4.0 추적 시 발생하는 kafka 헤더 로그 제거

- <Status>Fixed</Status> okhttp-2.7, okhttp3-3.14, okhttp3-4.4 추적 시 HTTP Call 정보를 수집하지 못하는 문제 수정

- <Status>Fixed</Status> SQL 호출 시 발생할 수 있는 <code>java.sql.Date</code> 참조 에러 수정
