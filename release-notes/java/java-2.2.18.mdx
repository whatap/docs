2023년 9월 26일

- <Status>Feature</Status> HttpServlet의 super를 설정하여 해당 클래스를 상속받은 클래스에서 http 서비스를 시작할 수 있는 기능 추가, 쉼표(,)를 구분자로 이용할 수 있습니다.

  ```ini title='whatap.conf'
  # example, separator ','
  hook_httpservlet_supers=javax.servlet.http.HttpServlet
  ```

- <Status>Feature</Status> db2-11.5 추적

  ```ini title='whatap.conf'
  weaving=db2-11.5
  ```

- <Status>Feature</Status> Oracle 데이터베이스 추적 설정 추가, 옵션 적용 후 재기동 필요하며, 기본값은 `true`입니다.

  ```ini title='whatap.conf'
  # Default: true
  # Will require a restart of the agent to take effect.
  hook_oracle_enabled=true
  ```

- <Status>Changed</Status> lettuce-5.1 추적 시 수집 SQL의 key 값 표시하도록 변경

- <Status>Fixed</Status> container-id 수집 시 발생하는 로그를 제거하도록 수정

- <Status>Fixed</Status> Redis의 lettuce 드라이버(<b>lettuce-5.1</b>, <b>lettuce-6.2</b>) 추적 시 이후의 SQL이 표시되지 않는 현상 수정

- <Status>Deprecated</Status> HTTP 서비스 추적 시 <code>org.apache.catalina.filters.FilterBase</code> 클래스를 기본 추적에서 제외
