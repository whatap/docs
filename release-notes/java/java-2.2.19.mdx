2023년 10월 10일

- <Status>Feature</Status> quarkus-reactive-1.13 추적

  *quarkus-resteasy-reactive-1.13.6.FINAL.jar* 이상

  ```ini title='whatap.conf'
  weaving=quarkus-reactive-1.13
  ```

- <Status>Feature</Status> quarkus-reactive-2.10 추적

  *quarkus-resteasy-reactive-2.10.0.FINAL.jar* 이상

  ```ini title='whatap.conf'
  weaving=quarkus-reactive-2.10
  ```

- <Status>Feature</Status> Spring Boot 2.5 이상의 Webflux 서비스에서 엑티브 스택 추적

- <Status>Feature</Status> 트랜잭션의 thread id 추적

  ```ini title='whatap.conf'
  # default: false
  trace_thread_id_enabled=false
  ```

- <Status>Changed</Status> 자바 애플리케이션의 메트릭 지표(<code>java_memory</code>, <code>java_gc</code>, <code>java_os</code>) 수집 활성화

  ```ini title='whatap.conf'
  # default: true
  perfx_java_enabled=true
  ```

- <Status>Changed</Status> Java 20 버전의 system property 값 대응

- <Status>Changed</Status> Java 6, Java 7, Java 8의 ASM api 버전을 ASM5에서 ASM6으로 수정

- <Status>Fixed</Status> spring-boot-3.0 추적 시 잘못된 버전의 로그가 표시되는 버그 수정
