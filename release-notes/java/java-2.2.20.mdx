2023년 10월 19일

- <Status>Feature</Status> apache-camel-seda-2.22 추적

  ```ini title='whatap.conf'
  weaving=camel-seda-2.22

  # default: false
  trace_camel_header_enabled=false
  ```

- <Status>Feature</Status> apache-camel-seda-3.2 추적

  ```ini title='whatap.conf'
  weaving=camel-seda-3.2

  # default: false
  trace_camel_header_enabled=false
  ```

- <Status>Feature</Status> `biz_exceptions_status` 옵션 추가

  ```ini title='whatap.conf'
  biz_exceptions_status=java.util.concurrent.TimeoutException:400&404&408,java.lang.Exception:200
  ```

  이 옵션을 적용하면 히트맵 info(파란색)로 표시하고 에러 통계를 추가합니다.

- <Status>Feature</Status> MongoDB 동기 방식 추적(mongodb-3.8.2, mongodb-4.0.3, mongodb-4.4, mongodb-4.8)

  ```ini title='whatap.conf'
  # example
  weaving=mongodb-4.0.3

  # spring-boot-3.x user
  # spring-boot-3.x: mongodb-4.8
  weaving=spring-boot-3.0

  # spring-boot-2.7.x user
  # spring-boot-2.7.x: mongodb-4.4
  weaving=spring-boot-2.7

  # spring-boot-2.5.x user
  # spring-boot-2.5.x: mongodb-4.0.3
  weaving=spring-boot-2.5
  ```

- <Status>Changed</Status> <code>biz_exceptions</code> 처리한 에러를 에러 통계로 수집

- <Status>Fixed</Status> SQL 파싱 오류 수정

- <Status>Deprecated</Status> Jedis, Tomcat, JMX metric 지표(<code>java_jedis</code>, <code>perfx_jmx_interval</code>, <code>perfx_tomcat_enabled</code>) 수집 중단

  ```ini title='whatap.conf'
  # deprecated
  perfx_jedis_enabled=false
  perfx_jmx_enabled=false
  perfx_tomcat_enabled=false
  ```
