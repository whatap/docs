2023년 12월 20일

- <Status>Feature</Status> 로그 모니터링 자바애플리케이션 로그에 멀티 트랜잭션 아이디(<code>@mtid</code> 태그) 추가

- <Status>Changed</Status> <code>logsink_enabled</code> 옵션 설정 시 트랜잭션 아이디와 멀티 트랜잭션 아이디를 수집할 수 있도록 수정 (기존: <code>logsink_trace_enabled</code> 옵션 추가)

  ```ini title="whatap.conf"
  # default: false
  logsink_enabled=false
  ```

- <Status>Changed</Status> log4j 사용 시 appender를 기본 수집하도록 수정

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=log4j-2.17

  # example: collect Console, AccessLog, ErrorLog 
  trace_log4j_appender_set=Console,AccessLog,ErrorLog
  ```

- <Status>Changed</Status> logback 사용 시 appender를 기본 수집하도록 수정

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=logback-1.2.8

  # example: collect ErrorAppender, FileAppender log
  trace_logback_appender_set=ErrorAppender,FileAppender
  ```
