2024년 01월 11일

- <Status>Feature</Status> spring-boot-2.5 이상에서 reactor-kafka-1.3 추적

  ```ini title='whatap.conf'
  # using spring-boot-2.5 ~
  weaving=spring-boot-2.5
  
  # using spring-boot-2.7 ~
  weaving=spring-boot-2.7
  
  # using spring-boot-3.0 ~
  weaving=spring-boot-3.0
  ```

- <Status>Feature</Status> spring-boot-2.5 이상에서 RxJava 사용 시 RxJava의 schedule 추적

  ```ini title='whatap.conf'
  # using spring-boot-2.5 ~
  weaving=spring-boot-2.5
  
  # using spring-boot-2.7 ~
  weaving=spring-boot-2.7
  ```

- <Status>Feature</Status> spring-boot-3.0 이상에서 RxJava 3 사용 시 RxJava 3의 schedule 추적

  ```ini title='whatap.conf'
  # using spring-boot-3.0 ~
  weaving=spring-boot-3.0
  ```

- <Status>Feature</Status> 에이전트 로그에 <code>threadName</code>, <code>className</code>, <code>methodName</code>, <code>lineNumber</code> 수집하여 표시

  ```log title='whatap.log'
  [{threadName}][{className}.{methodName}({lineNumber})] 
  ```

- <Status>Feature</Status> 에이전트 로그 형태 수정

  ```log title='whatap.log'
  {date} {time} [{threadName}][{className}.{methodName}({lineNumber})] ({work}) {log}
  ```

  ![whatap log](/img/release/release-java-log.png)

- <Status>Feature</Status> Tibero의 <code>CallableStatement</code> 추적

- <Status>Changed</Status> 에이전트 로그의 텍스트 색상 변경

- <Status>Changed</Status> 에이전트 기동 시 에이전트 로그의 파일명을 <em>whatap-<code>date</code>.log</em>에서 <em>whatap.log</em>로 변경

- <Status>Changed</Status> UTC 기준 날짜가 변경되면 <em>whatap.log</em> 파일명을 <em>whatap-<code>date</code>.log</em>로 저장
