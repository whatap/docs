2024년 02월 06일

- <Status>Feature</Status> mule-3.9.5 추적 범위 확대

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=mule-3.9.5
  ```

- <Status>Feature</Status> camel-cxf-3.15 추적

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=camel-cxf-3.15
  ```

- <Status>Feature</Status> <code>OracleCallableStatement</code> 추적 추가

- <Status>Changed</Status> Lettuce 드라이버 사용 시 Redis의 key 수집 범위 추가

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=lettuce-6.2
  ```

- <Status>Changed</Status> Lettuce 드라이버 사용 시 Redis의 key 수집 여부 수정

  ```ini title="whatap.conf"
  # default: false
  trace_sql_param_enabled=true
  
  # default: false
  trace_redis_lettuce_key_enabled=true
  ```

- <Status>Changed</Status> Lettuce 드라이버 사용 시 Redis의 key 수집 숫자를 제한

- <Status>Fixed</Status> spring-boot에서 netty 사용 시 request의 URL에서 querystring 분리

- <Status>Fixed</Status> spring-boot에서 netty 사용 시 request의 HTTP 파라미터를 수집하지 못하는 문제 수정

- <Status>Fixed</Status> 히트맵의 트랜잭션 건수, 에러 건수 오류 수정

- <Status>Deprecate</Status> Lettuce 드라이버 사용 시 Redis의 value 수집 중단
