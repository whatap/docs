2024년 02월 27일

- <Status>Feature</Status> logback-1.2.8 추적 시 appender 제외 옵션 추가

  ```ini title="whatap.conf"
  # example: ignore 'CONSOLE', 'ACCESS_APPENDER' appender
  ignore_logback_appender_set=CONSOLE,ACCESS_APPENDER
  ```

- <Status>Deprecated</Status> logback-1.2.8 추적 시 지정한 appender만 수집하는 옵션 사용 중단

  ```ini title="whatap.conf"
  # deprecated
  trace_logback_appender_set=
  ```

- <Status>Feature</Status> log4j-2.17 추적 시 appender 제외 옵션 추가

  ```ini title="whatap.conf"
  # example: ignore 'CONSOLE', 'ACCESS_APPENDER' appender
  ignore_log4j_appender_set=CONSOLE,ACCESS_APPENDER
  ```
    
- <Status>Deprecated</Status> log4j-2.17 추적 시 지정한 appender만 수집하는 옵션 사용 중단

  ```ini title="whatap.conf"
  # deprecated
  trace_log4j_appender_set=
  ```

- <Status>Changed</Status> 컨테이너 환경에서 Pod 이름을 설정하지 않은 경우 환경 변수(<code>env</code>)의 <code>HOSTNAME</code>에서 추출하도록 변경

- <Status>Changed</Status> JBoss Connection Pool 연결 시 로그 숫자 제한하도록 수정

- <Status>Changed</Status> 통계 데이터 전송 시 로그 수정

- <Status>Fixed</Status> <code>trace_sampling_enabled</code> 옵션을 사용할 경우 TPS를 과하게 수집하는 문제 수정

- <Status>Fixed</Status> spring-boot-3.x의 spring-security exception 수집 시 발생하는 에러 수정
