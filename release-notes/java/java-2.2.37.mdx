- <Status priority='1'>Feature</Status> spring-boot ExceptionHandler를 통과하는 예외(exception)가 발생하면 이전에 발생한 예외를 대체

  <Cmdname sid="TTL06209" className="uitext" /> 창의 <Cmdname sid="TTL06125" className="uitext" /> 탭에서 예외(exception)를 spring-boot의 ExceptionHandler에 등록한 예외(exception)로 표시할 수 있습니다.

  ```ini title="whatap.conf"ㅓ
  # default: true
  trace_exception_handler_overwrite_enabled=true
  ```
