2025년 01월 22일

- <Status priority='1'>New</Status> logstash-logback-encoder-7.3 추적

  logback 인코더로 `net.logstash.logback.encoder.LogstashEncoder` 설정한 경우 추적

  ```ini title="whatap.conf"
  weaving=logback-1.2.8,logstash-logback-encoder-7.3
  ```
    

- <Status priority='1'>New</Status> logback-ecs-encoder-1.5.0 추적
  
  logback 인코더로 `co.elastic.logging.logback.EcsEncoder` 설정한 경우 추적

  ```ini title="whatap.conf"
  weaving=logback-1.2.8,logback-ecs-encoder-1.5.0
  ```

- <Status priority='1'>Feature</Status> logback async appender 추적

  logback appender로 async appender 설정한 경우 추적

  ```ini title="whatap.conf"
  weaving=logback-1.2.8
  ```

- <Status priority='1'>Feature</Status> `service_name_mode` 옵션에 return 방식 추가

  ```ini title="whatap.conf"
  service_name_mode=return
  ```

  예, `io.home.Test` 클래스의 test 메소드의 return 값을 트랜잭션 이름으로 지정

  ```ini title="whatap.conf"
  hook_service_patterns=io.home.Test.test
  service_name_mode=return
  ```
