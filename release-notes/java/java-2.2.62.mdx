2025년 07월 03일

- <Status priority='0'>Fixed</Status> 자바 에이전트의 web_static_content_extensions 설정에 등록된 .html, .js 등의 정적 리소스 호출 시, 메트릭스 지표인 arrival_rate가 과도하게 수집되던 현상을 수정

    ```bash 
    # default: js,htm,html,gif,png,jpg,css,swf,ico
    web_static_content_extensions=js,htm,html,gif,png,jpg,css,swf,ico
    ```

    - 애플리케이션 대시보드의 액티브 트랜잭션 스피드 차트에서 RPS 항목 수정

    - 메트릭스 조회의 app_counter 카테고리 내 arrival_rate 항목 수정

- <Status priority='0'>Fixed</Status> 일부 http call 수행시 exception을 수집하지 못하는 현상 수정

- <Status priority='0'>Fixed</Status> weaving=lettuce-6.2 설정 시, lettuce 6.5 이상 버전에서 오류가 발생해 호출이 누락되던 현상 수정

- <Status priority='0'>Deprecated</Status> log4j-1 추적 기능 제거 (별도 플러그인으로 제공)
