2025년 09월 03일

- <Status priority='1'>Feature</Status> apache httpclient로 Opensearch를 호출하는 경우 추적

    ```
    # requires agent restart
    weaving=opensearch
    ```

- <Status priority='0'>Fixed</Status> 동일한 Linux host에서 여러 개의 docker를 수행하는 경우 vCPU 중복 집계 방지하도록 수정

- <Status priority='0'>Fixed</Status> spring-boot-3.2 이상 버전에서 WebClient로 opentelemetry 에이전트를 사용하는 애플리케이션 연계 추적 시 멀티 트랜잭션탭에서 연결되지 않는 현상 수정

- <Status priority='0'>Fixed</Status> Java 1.7 미만인 경우 jmx를 통해 system cpu 지표를 수집하는 현상 방지

    - 에이전트 로그에 sigar 라이브러리 사용하도록 표시

    ```
    # default: false
    sigar_enabled=true
    ```
