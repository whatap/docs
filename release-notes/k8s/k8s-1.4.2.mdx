- <Status>Feature</Status> <code>kube_pod</code> 카테고리에 수집 데이터 추가

  - <code>qosClass</code>: Pod에 대한 서비스 품질
  - <code>startTime</code>: Pod 시작 시간
  - <code>podHash</code>: Pod 이름의 해시값
  - <code>con_cpu_limit_total</code>: Pod 내부 컨테이너의 cpu limit 합계
  - <code>con_mem_alloc_fail_total_count</code>: Pod 내부 컨테이너의 memory 할당 실패 수 합계
