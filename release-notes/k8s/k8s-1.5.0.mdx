2024년 01월 16일

&nbsp; **Java v2.2.26**

- <Status>Feature</Status> 오브젝트 매니페스트 수집 기능 추가

  하루에 한번 사용자가 설정한 타임존의 자정 시각에 클러스터의 오브젝트 정보를 수집하여 저장하는 기능 추가했습니다. 이 기능을 이용하려면 다음과 같이 에이전트 옵션을 설정하세요. 자세한 내용은 [다음 문서](../../kubernetes/object-manifest)를 참조하세요.
  
  - `store_k8s_object_manifest_enabled` <span class='type'>Boolean</span>

    기본값 `true`

    오브젝트 매니페스트 수집 기능을 활성화할 수 있습니다.

  - `store_k8s_object_manifest_timezone` <span class='type'>String</span>

    기본값 `UTC`

    설정한 타임존의 자정 시간에 오브젝트 매니페스트 수집 스케줄을 실행합니다.

  - `store_k8s_object_manifest_max_fetch_limit` <span class='type'>Int</span>

    기본값 `50`

    클러스터에서 한번에 조회할 오브젝트의 개수를 설정할 수 있습니다.

- <Status>Fixed</Status> 컨트롤 플레인 모니터링 에이전트 성능 개선, 클러스터와 통신하는 클라이언트의 성능 개선 포함
