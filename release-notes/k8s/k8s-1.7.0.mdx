2024년 05월 13일

<Status>Feature</Status> <b>TagCount</b>에 데이터 추가

- `kube_pod` 카테고리에 `reason`, `message` 필드 추가

  - `reason`: 파드 status의 reason
  - `message`: 파드 status의 message

- `container` 카테고리에 `kube_state`(컨테이너 state) 필드 추가

- pending 상태 파드, 컨테이너 데이터(`kube_pending_pod`, `kube_pending_container`)를 수집

  관련 데이터는 <Cmdname sid="side_lab" className="uitext" /> &gt; <Cmdname sid="side_pendingPod" className="uitext" /> 메뉴에서 확인할 수 있습니다.

- 서비스(`kube_service`)에 대한 데이터 수집

- 에이전트 부팅 시 <code>#whatapEvent</code> 카테고리로 사용자 설치 Runtime, 클러스터의 Runtime을 전송
