2024년 07월 01일

- <Status>Feature</Status> 태그 카운트 추가

  - **kube_service_pod_mapping** 항목 추가
  - **kube_pod** 카네고리에 `podUid` 태그 추가
  - **kube_service** 카테고리에 `externalIp` 태그, PORT(`portName`, `nodePort`, `servicePort`, `targetPort`) 필드 추가
  - **kube_event** 카테고리에 `watch_event_type`, `resource_version` 데이터 추가
  - **kube_event** 중복 알림 현상 제거

- <Status>Feature</Status> **whatap-node-helper** 컨테이너 환경변수(`env`)에 `WHATAP_JAVA_AGENT_PATH` 추가

  :::note

  - 해당 환경변수에 와탭 자바 에이전트 jar 파일 경로를 설정하면 컨테이너 id 연동할 수 있습니다. 
  
  - 이 기능은 [Java agent 2.2.33](../java/java-2_2_33) 버전 이상에서 지원합니다.

  :::

:::info

Agent download: [kube_mon_1.7.6.tar](https://repo.whatap.io/docker/kube_mon_1.7.6.tar)

:::
