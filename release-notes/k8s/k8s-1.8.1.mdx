<Status priority='1'>New</Status> 새로운 태그 카운트 메트릭 추가

* **kube_scheduler** 

  샘플링 주기 `1분`

  Kubernetes 컨트롤 플레인 구성 요소인 스케줄러 관련 메트릭을 수집합니다.

* **kube_pv** 

  샘플링 주기 `30초`

  Kubernetes 스토리지 Persistent Volume(PV) 관련 메트릭을 수집합니다.

* **kube_pvc** 

  샘플링 주기 `30초`

  Kubernetes 스토리지 Persistent Volume Claim(PVC) 관련 메트릭을 수집합니다.

* **kube_pod_volume_list\{h1\}** 

  샘플링 주기 `1시간`

  Kubernetes 파드의 볼륨 마운트 정보를 수집합니다.

* **kube_container_volume_list\{h1\}** 

  샘플링 주기 `1시간`

  Kubernetes 컨테이너의 볼륨 정보를 수집합니다.
