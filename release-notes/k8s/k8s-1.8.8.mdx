> **Java Agent v2.2.61 포함**

2025년 09월 04일 

- <Status priority='1'>New</Status> 쿠버네티스 환경 APM 설치 시 컨테이너 맵 애플리케이션 정보 표시 개선

    컨테이너 맵에 애플리케이션 정보가 표시되지 않는 경우, APM이 설치된 컨테이너 매니페스트(*YAML*)의 환경변수에 해당 컨테이너 내 에이전트 절대 경로를 나타내는 옵션을 추가합니다.

    언어별 옵션은 다음과 같습니다.

    - **Java**

        `WHATAP_JAVA_AGENT_PATH`

        :::note
        JAVA APM을 사용하지만, 컨테이너 내부에서 Java 명령어를 수행할 수 없는 최소 이미지를 사용하는 경우가 있습니다. 이러한 경우 `WHATAP_EXECUTABLE_JAVA_PATH` 키로 컨테이너 환경변수에 java full path를 등록해줍니다.
        :::

        ```yaml title="Java example"
        apiVersion: apps/v1
        kind: Deployment
        metadata:
          name: my-java-app
        spec:
          replicas: 1
          selector:
            matchLabels:
              app: my-java-app
          template:
            metadata:
              labels:
                app: my-java-app
            spec:
              containers:
                - name: my-java-container
                  image: your-java-app-image:latest
                  env:
                    # Java APM Jar 경로
                    - name: WHATAP_JAVA_AGENT_PATH
                      value: /whatap/whatap.agent-2.2.35.jar
                    # 최소 이미지에서 java 실행 파일의 전체 경로 지정
                    - name: WHATAP_EXECUTABLE_JAVA_PATH
                      value: /layers/paketo-buildpacks_adoptium/jre/bin/java
        ```

    - **Python**

        `WHATAP_PYTHON_AGENT_PATH`

    - **PHP**

        `WHATAP_PHP_AGENT_PATH`

    - **.NET**

        `WHATAP_DOTNET_AGENT_PATH`

    - **Go**

        `WHATAP_GO_AGENT_PATH`
