> **Java Agent v2.2.61 포함**

2025년 11월 07일 

- <Status priority='0'>Fixed</Status> **CPU 통계 개선**

    - **CPU 통계 음수 값 방지**: 카운터 리셋이나 경쟁 조건으로 인해 CPU 메트릭이 음수 값을 표시할 수 있는 문제 수정

        - 여러 CPU 메트릭에 적용: `user`, `sys`, `nice`, `irq`, `softirq`, `steal`, `guest`, `guest_nice`

        - 계산된 값이 음수면 0을 반환하여 모든 CPU 통계가 음수가 되지 않도록 보장

- <Status priority='0'>Fixed</Status> **Ingress 백엔드 서비스 포트 처리**

    - **백엔드 서비스 포트 처리 개선**: Ingress 구성 처리 시 엣지 케이스 수정

        - 백엔드 서비스에 대해 숫자 포트와 이름 포트를 모두 올바르게 처리
