2024년 01월 11일

- <Status>Fixed</Status> <code>trace_http_client_ip_header_key=x-forwarded-for</code> 옵션을 설정한 상태에서 헤더로 <code>x-forwarded-for</code> 값에 여러 개의 IP가 전달될 경우(쉼표 기준) 첫 번째 값으로 IP를 설정하도록 수정

  :::note

  이 옵션에 대한 자세한 내용은 [다음 문서](../../nodejs/agent-number-of-user#trace_http_client_ip_header_key)를 참조하세요.

  :::

- <Status>Fixed</Status> TCP, UDP 소켓 모니터링 수정

- <Status>Fixed</Status> <code>profile_http_header_enabled</code> 옵션의 값이 <code>true</code>이면 profile의 headers 값을 나타내는 방식 수정 (JSON → key=value\n)

:::info

* 패키지 정보: https://www.npmjs.com/package/whatap
