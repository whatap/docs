<Status priority='1'>New</Status> Prisma ORM 사용을 위한 옵션 추가 

- **prisma_read_func_name** <span class='type'>String</span>

  기본값 `read`

  Replication 사용 시 Read-only 클라이언트를 얻기 위해 호출하는 함수명을 설정합니다.

- **prisma_database_url_name** <span class='type'>String</span>

  기본값 `DATABASE_URL`

  Prisma ORM에서 데이터베이스 연결 URL이 저장된 환경 변수명을 설정합니다.
