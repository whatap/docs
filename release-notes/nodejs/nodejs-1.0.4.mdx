2025년 10월 16일

- <Status priority='1'>New</Status> **hook_method_patterns**

    - 사용자 애플리케이션의 특정 메소드를 모니터링할 수 있는 옵션

    - 설정 파일에 메소드 경로를 지정하면 해당 메소드의 실행 시간과 호출 인자를 수집

    - `false`: 기본값

        :::note

        **설정 방법** 
        - 클래스 메소드: `./경로/파일명/클래스명.메소드명`
        - 일반 함수: `./경로/파일명/함수명`
        - 여러 개 지정 시 쉼표로 구분

        :::

        :::caution

        **주의** 
        - TypeScript 프로젝트는 빌드된 파일 경로를 설정 (예. `./dist/services/user.service`)
        - 옵션 설정 후 제거하기 위해서는 애플리케이션 재기동이 필요합니다.

        :::

- <Status priority='0'>Fixed</Status> PM2 클러스터 모드로 여러 인스턴스를 실행할 때 whatap_nodejs 모듈 프로세스를 효율적으로 관리하도록 개선

    - 첫 번째 인스턴스만 whatap_nodejs 에이전트 시작

    - 나머지 인스턴스는 이미 실행 중인 에이전트에 자동 연결
