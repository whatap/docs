2025년 02월 25일

* <Status priority='1'>Feature</Status> Span의 `name` 값을 트랜잭션 이름으로 지정하는 옵션 추가

  * **otel_tx_path_setby_name_enabled** <span class='type'>bool</span> (기본값 `false`)

* <Status priority='1'>Feature</Status> Trace Span의 `SpanID`를 외부 연결의 `ParentSpanID`로 인식하는 멀티 트랜잭션 연결 관련 옵션 추가

  * **otel_mtrace_stepid_setby_trace_spanid_enabled** <span class='type'>bool</span> (기본값 `false`)

* <Status priority='1'>Feature</Status> 파일 로그와 동일한 내용을 `Stdout`으로 출력하는 로그 옵션 추가 

  * **log_stdout_enabled** <span class='type'>bool</span> (기본값 `false`)

  :::note 

  Docker 및 쿠버네티스 환경에서 환경변수를 등록하여 사용할 수 있습니다. 
  
  :::

* <Status priority='0'>Fixed</Status> *whatap.conf* 파일이 없을 경우 파일 없음 오류 로그가 계속 출력되는 현상 수정

* <Status priority='0'>Fixed</Status> 트랜잭션 ID가 `0`으로 표기되는 현상 수정

:::info
