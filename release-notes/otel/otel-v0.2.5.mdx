2025년 07월 09일

* <Status priority='2'>Feature</Status> Parent 없는 Span을 트랜잭션으로 인식  
  * Parent가 없는 Span은 kind에 관계없이 트랜잭션으로 처리  
  * 기존에는 `SPAN_KIND_SERVER`만 트랜잭션으로 처리  
  * 관련 옵션: `otel_tx_root_span_enabled` (기본값: true)

* <Status priority='2'>Feature</Status> 메트릭스 수집 방식 개선  
  * 기존 로그 기반 수집에서 프로메테우스 호환 방식으로 변경  
  * Flex 보드(MXQL 데이터 조회)로 확인 가능

* <Status priority='1'>Change</Status> `query_string_enabled` 옵션 동작 방식 변경  
  * `query_string_enabled`가 true이고 `query_string_urls`, `query_string_keys`가 설정되지 않은 경우  
    모든 URL의 query string 수집

* <Status priority='0'>Fixed</Status> HTTP Status 카운터 수집 오류 수정  
  * 500번대 에러의 합산 값이 수집되지 않던 문제 수정
