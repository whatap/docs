2025년 09월 01일

* <Status priority='1'>Change</Status> Span 데이터 처리 방식 개선 
  * 부모·자식 트리 구조로 Span 데이터를 수집  
  * 중간에 부모 Span 데이터가 없어도 이후 자식 Span 데이터를 처리하도록 변경  

* <Status priority='1'>Change</Status> 속성 키 이름 추가    
  * HTTP 상태 코드를 수집하는 속성 키 이름 http.response.status_code를 추가  
    * otel_attr_tx_status_keys  
      * 기본값: http.status_code, http.response.status_code  

  * 에러 정보를 수집하는 속성 키 이름 error 를 추가  
    * otel_attr_error_keys  
      * 기본값: error, error.type  

:::info
