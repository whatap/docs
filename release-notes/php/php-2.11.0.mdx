2025년 07월 09일

- <Status priority='0'>Fixed</Status> whatap-php.service 등록 및 실행 실패 문제 수정

  - 신규 설치 시, /etc/systemd/system/whatap-php.service에 environment 값 누락으로 접속 정보(Access Key 및 서버 주소)를 인식하지 못해 에이전트가 비활성화됨

  - /usr/whatap/php/install.sh 실행 후 /usr/lib/systemd/system/whatap-php.service → /etc/systemd/system/whatap-php.service 복사로 해결 가능

  - 기존 설치 환경에서는 업데이트 시 정상 설치됨

2025년 06월 18일

- <Status priority='0'>Feature</Status> CPU 부하 감소

  - 모니터링 대상 판정 기준 변경으로 CPU 부하 감소

  - 사용자 함수 모니터링 해제 시 추가 부하 감소

    `whatap.profile_method_enabled=false`

- <Status priority='1'>New</Status> CLI 호출 파라미터 URL 표기

  - CLI 호출 시 파라미터를 포함해 URL 수집

    - 예) php /home/www/test.php aaa bbb
    → /home/www/test.php aaa bbb로 표기

- <Status priority='1'>New</Status> file 파라미터 정보 수집

  - 최대 20개, 각 256byte까지 파라미터 일부만 수집

- <Status priority='0'>Fixed</Status> 메모리 릭 수정

  - DB 함수 실패 시 error 함수 호출로 인한 메모리 릭 수정
