배포일: 2021-11-10

## Change (업데이트)

### file 관련 함수 추가

기존 수집 중인 `fopen()` 함수에 추가로, `file()`, `file_get_contest()` 함수의 실행시간과 파일명을 수집합니다.

파일관련 수집 대상 함수

```php
fopen()
file()
file_get_contents()
```

## Fixed (버그 수정)

### GET 파라미터 수집 오류 수정  

코드이그나이터 프레임워크에서 rewrite module로 URL이 변경되는 경우 GET 파라미터를 수집 못하는 오류 수정, URI의 `Query_string`에서 파라미터를 분리하여 중복없이 추가하였습니다. 

### PHP 8.0 에서 JIT disabled 경고

PHP 8.0 에서 `opcache.jit=on` 설정 시에 아래와 같은 경고가 발생하는 오류를 수정하였습니다.

```bash
PHP Warning: JIT is incompatible with third party extensions that override zend_execute_ex(). JIT disabled. in Unknown on line 0
```

:::note

PHP 8.0.3 이상을 권장합니다.

:::
