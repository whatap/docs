## Change (업데이트)

### 로그모니터링 압축 전송, 기본 설정으로 변경

`whatap.logsink_zip_enabled`의 기본값을 `true` 로 설정합니다. 압축 전송을 기본으로 합니다. 

### 프로젝트 라이센스 키의 명칭을 엑세스 키로 변경

에이전트 옵션 중 `whatap.license` 를 `whatap.accesskey` 로 변경합니다. 기존 `license`도 사용 가능합니다.

### TCP queue 크기 변경

256, 512 에서 512, 1024 로 확장합니다. 로그모니터링 파일이 많이 설정되어 있는 경우 tcp 패킷 증가를 처리하기 위해 확장합니다. 

## Fixed (버그 수정)

### segmentation fault 수정

멀티트랜잭션 추적 옵션(`whatap.mtrace_enabled`)이 설정된 상태에서 `curl_setopt_array` 함수 사용 중 segmentation fault가 발생하는 오류를 수정했습니다.

### biz_exceptions 옵션

`biz_exceptions` 옵션으로 제외한 오류는 히트맵에는 오류로 표기 되지 않고, 트랜잭션 프로파일과 통계정보에만 적용됩니다. 통계정보에 적용 안되는 오류를 수정했습니다.
