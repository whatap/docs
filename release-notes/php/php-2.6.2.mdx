* <Status>Feature</Status> 대용량 프로파일 처리 옵션 추가

  - 기존 프로파일은 최대 1024개의 프로파일 정보를 수집했습니다. 설정으로 최댓값을 늘릴 수 있지만, 메모리 증가는 필수적으로 발생했습니다.
  
  - 일정량의 프로파일을 압축 및 분할 전송하여 메모리 증가 없이 더 많은 프로파일 정보를 수집할 수 있도록 개선했습니다.
  
  - **whatap.large_profile_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    약 만 건 이하의 프로파일(스탭) 정보를 수집할 수 있습니다. 더 많은 양을 수집할 경우 `split_profile_enabled` 옵션으로 사용하는 것을 권장합니다.

  - **whatap.split_profile_enabled** <span class='type'>Boolean</span>

    기본값 `false`

    일정량의 프로파일을 가상 트랜잭션으로 분리하여 더 많은 프로파일 정보를 수집합니다. 여러 개의 트랜잭션 목록으로 분할 표기합니다.

  :::note

  * 두 옵션을 모두 설정한 경우 `whatap.split_profile_enabled` 옵션이 우선 적용됩니다.

  * 위 옵션과 관련한 에이전트 설정에 대한 자세한 내용은 [다음 문서](../../php/agent-transaction#php-trs-trc)를 참조하세요.

  :::

  * **whatap.profile_zip_enabled** <span class='type'>Boolean</span>  기본값 `true`

  :::note 

  * 위 옵션과 관련한 에이전트 설정에 대한 자세한 내용은 [다음 문서](../../php/agent-transaction#php-trs-trc)를 참조하세요.

- <Status>Changed</Status> 특정 HTTP Method 수집을 제외하는 기능을 모든 트랜잭션을 대상으로 변경

  - `whatap.ignore_http_method` 옵션은 모든 트랜잭션 대상으로 변경합니다.

  - 기존 제외 기능은 `whatap.http_method_urls` 옵션으로 설정된 트랜잭션만 제외했습니다. `whatap.http_method_urls` 옵션을 삭제하고 모든 트랜잭션 대상으로 변경합니다.

  :::note

  * 위 옵션과 관련한 에이전트 설정에 대한 자세한 내용은 [다음 문서](../../php/agent-transaction#php-track-http-trs)를 참조하세요.

  :::
