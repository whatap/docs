2024년 01월 10일

- <Status>New</Status> PHP 8.3 지원

- <Status>Feature</Status> 멀티 트랜잭션 추적에서 Trace context(W3C) 지원

  Trace context를 지원하는 **OpenTelmetry**와 멀티 트랜잭션 추적 연계를 지원합니다.

- <Status>Changed</Status> SQL, Method, HTTP call의 리소스 지표 수집 수정

  트랜잭션 시작 시점의 리소스(cpu, memory)와 각 스탭 시작 시점의 리소스 차이를 수집합니다.

- <Status>Changed</Status> 설정 파일(<em>whatap.conf</em>)이 없을 경우 새로 생성하도록 수정

- <Status>Fixed</Status> 액티브 트랜잭션 목록에서 외부 호출 누적 건수가 누적 시간으로 잘못 수집되는 오류 수정

- <Status>Fixed</Status> 리눅스 환경에서 whatap-php 서비스(<code>whatap-php.service</code>) 실행 오류 수정

- <Status>Fixed</Status> PHP 8.1.0 버전에서 8.1.2 버전까지 발생하는 segmentation 오류 수정(사용자 함수에 대한 수집 중지)
