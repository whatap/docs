2024년 12월 11일

- <Status priority='1'>New</Status> PHP 8.4 지원

- <Status priority='0'>Fixed</Status> `session_start` 프로파일 정보의 위치 수정

  `session_start` 함수 내부에서 Redis 쿼리가 실행될 때, `session_start` 함수 정보를 Redis 쿼리 보다 먼저 표시하도록 수정합니다.

- <Status priority='0'>Fixed</Status> 멀티 트랜잭션 샘플링 수정

  멀티 트랜잭션 샘플링이 비활성화되면, 외부 HTTP 호출이 여러 개여도 분산 트랜잭션 추적을 중단합니다.

:::info
