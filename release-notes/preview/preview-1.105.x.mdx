## Preview 1.105.01

2023년 7월 3일

<div class='indentTab'>

## `공통`

### 대시보드

- <Status>Fixed</Status> <UI>트랜잭션 분석</UI> 창에서 트랜잭션 목록 조회 시 에이전트 필터가 적용되지 않는 버그 수정

- <Status>Fixed</Status> <UI>트랜잭션 모달</UI> 창에서 로그 미 설정 프로젝트의 경우 트랜잭션 로그 탭이 노출되는 버그 수정

### Flex 보드

- `Change` 통합 Flex 보드 메뉴에서 대시보드의 모든 프로젝트에 대한 수정 권한이 없어도 공유 버튼을 노출하도록 수정

---

## `Application`

- <Status>Fixed</Status> Python 프로젝트 > <UI>관리</UI> > <UI>에이전트 설치</UI> 메뉴에서 <UI>에이전트 다운로드</UI> 섹션의 다운로드 명령어 수정

- <Status>Fixed</Status> Java 프로젝트 > <UI>관리</UI> > <UI>에이전트 설치</UI> 메뉴에서 <UI>JVM 옵션 추가</UI> 섹션의 JVM 옵션 코드 수정

---

## `Server`

<p><Status>Fixed</Status> <UI>서버 목록</UI> 메뉴에서 비교할 서버 선택하면 <UI>기간 조회</UI> 버튼이 활성화 되지 않는 버그 수정</p>

## Preview 1.105.00

2023년 6월 30일

<div class='indentTab'>

### `공통`

- <Status>Fixed</Status> 무료 프로젝트 10개 초과 생성 시 표시되는 에러 메시지 수정

- <Status>Fixed</Status> 프로젝트 내에서 조직이 변경되지 않는 버그 수정

- <Status>Fixed</Status> <UI>조직 관리</UI> 메뉴에서 <UI>가이드 문서 보기</UI> 버튼 클릭 시 사용자가 선택한 언어의 가이드 문서로 이동하도록 수정

- <Status>Fixed</Status> <UI>Flex 보드</UI> 메뉴의 대시보드에서 테이블 위젯의 경우 boolean 값을 확인할 수 있도록 수정

### `Server`

#### 경고 알림

- <Status>Fixed</Status> <UI>이벤트 설정</UI> 메뉴에서 비활성화된 서버의 알림 설정 여부가 표시되지 않는 버그 수정
