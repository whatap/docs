    | :--: |

  - <Status>Fixed</Status> <UI>멀티 트랜잭션</UI> 탭의 <UI>차트</UI> 뷰 모드에서 트랜잭션 노드를 클릭하면 <UI>테이블</UI> 뷰로 전환되는 오류 수정

  - <Status>Feature</Status> <UI>트랜잭션 로그</UI> 탭에서 데이터가 없는 경우 다음 상황에 맞는 메시지를 출력하도록 기능 추가

    - 에이전트 미설정으로 로그를 수집하지 못하는 경우

  <div class='.release-compare'>

  | 변경 전 | 변경 후 |
  | :-: | :-: |
  | ![](/img/release/interval-option-before.png) | ![](/img/release/interval-option-after.png) |

* <Status>Changed</Status> <UI>시간</UI> 조회 구간이 변경될 때 <UI>인터벌</UI> 선택 옵션에서 <UI>기본</UI> 항목을 선택하도록 로직 수정

  조회 구간을 변경할 때 ***인터벌*** 옵션이 ***기본*** 항목이 선택되는 경우는 다음과 같습니다.

  - ***인터벌*** 옵션에서 이전 선택 항목이 ***기본***인 경우

  - ***인터벌*** 옵션에서 이전 선택 항목이 ***기본***이 아니면서 변경된 ***인터벌*** 옵션 목록에 없는 경우

  - ***인터벌*** 옵션에서 이전 선택 항목이 ***기본***이 아니면서 선택한 ***인터벌*** 옵션 값이 조회 기간의 절반 이하인 경우

* <Status>Fixed</Status> 일부 상황에서 <UI>컬럼 설정</UI> 팝업의 <UI>확인</UI> 버튼을 눌렀을 때 불명의 오류 메시지를 표시하며 작동하지 않는 오류 수정

* <Status>Fixed</Status> <UI>계정 정보 관리</UI> 메뉴에서 WhatsApp으로 전화번호 입력 시 국가 코드 번호가 빈 값으로 기본 설정되는 오류 수정

* <Status>Fixed</Status> 초기 화면 <UI>프로젝트 목록</UI>의 카드 뷰 화면에서 <UI>애플리케이션</UI> 항목의 카운트가 0으로 출력되는 버그 수정
