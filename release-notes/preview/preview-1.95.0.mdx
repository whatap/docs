## Official Release: 2023-01-16

### `공통`

#### 용어 변경

* <Status>Change</Status> <b>프로파일</b> 용어를 <b>트레이스</b>로 변경합니다.

#### 로그 설정

* <Status>Change</Status> <UI>카테고리별 데이터 유지기간</UI> 테이블에서 저장량 컬럼 삭제

* <Status>Feature</Status> <UI>카테고리별 데이터 유지기간</UI> 테이블에서 <UI>금일 로그수</UI> 컬럼 추가 및 컬럼 별 도움말 추가

* <Status>Feature</Status> <UI>로그 Parser 설정</UI> 섹션에서 상태 성공률 파서, 4xx, 5xx 파서 등록 시 동일한 카테고리에 파서를 중복 등록할 수 없도록 유효성 검사 추가

#### 계정 정보 관리

* <Status>Feature</Status> <UI>계정 정보</UI> 메뉴에서 멀티팩터 인증 활성화 시 최종 확인을 위한 메시지 창 추가

#### Flex 보드

<Status>Change</Status> <UI>Log RPS Board</UI> 대시보드 템플릿 수정

- 데이터 조회 기간의 기본값 변경: 실시간 30분 → 실시간 12시간
- ***Log RPS trend***, ***서버 상태*** 위젯의 레이아웃 높이 수정
- ***4XX, 5XX Count*** 위젯의 보조 테이블 데이터 단위 설정
- 일부 위젯 데이터 호출 간격을 5초에서 1분으로 변경


### `Application`

#### 트랜잭션 상세

다음 메뉴 경로에서 추가 및 수정한 사항입니다.

:::note

* ***대시보드*** > ***애플리케이션 대시보드*** > ***액티브 트랜잭션*** 위젯 오른쪽에 '![자세히 보기 아이콘](/img/right-arrow.svg)' 버튼 클릭 > 종료된 트랜잭션 항목 선택 > ***트랜잭션 보기*** 버튼 클릭
* ***대시보드*** > ***애플리케이션 대시보드*** > ***히트맵*** 위젯 오른쪽에 '![자세히 보기 아이콘](/img/right-arrow.svg)' 버튼 클릭 > ***히트맵 트랜잭션*** 차트 드래그 > ***TX 트레이스*** 목록에서 항목 클릭
* ***대시보드*** > ***트랜잭션 맵*** > ***트랜잭션*** 차트 드래그 > ***TX 트레이스*** 목록에서 가장 왼쪽의 ![자세히 보기 아이콘](/img/ico-detail-view.svg) 버튼 클릭
* ***대시보드*** > ***액티브 트랜잭션*** > ***트랜잭션*** 탭에서 종료된 트랜잭션 컬럼 선택 > ***트랜잭션 보기*** 버튼 클릭
* ***분석*** > ***히트맵*** > ***히트맵 트랙잭션*** 차트 드래그 > ***TX 트레이스*** 목록에서 항목 클릭

:::
