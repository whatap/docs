## Preview-1.97.0 

2023년 3월 8일

### `공통`

#### User Interface (UI)

사용자 만족도를 높이기 위해 다음과 같이 사용자 인터페이스 디자인을 개선합니다.

- <Status>Change</Status> <UI>이용내역</UI> > <UI>청구서 미리보기</UI> 메뉴의 UI 개선

    - 프로젝트, 상품별로 그룹화 된 청구서 미리보기 정보 제공
    - 그룹화된 데이터의 관련 상품 정보 및 과금 집계량, 청구금액 합 확인 가능

- <Status>Change</Status> 화면 상단에 위치한 '메뉴 경로'를 '메뉴명'으로 표시하도록 변경

    - 변경 전: 상위 경로 > 메뉴명
    - 변경 후: 메뉴명

- <Status>Change</Status> 화면 오른쪽 위에 와탭 기술 문서 링크 버튼 추가 및 채널톡 내 기술 문서 링크 제거

- <Status>Fixed</Status> 서비스 로고 옆에 위치한 브라우저 시간 불일치 알림 아이콘 위치 수정

- <Status>Fixed</Status> 프로젝트가 없는 그룹을 선택하면 프로젝트 생성 페이지로 이동하지 않도록 수정

#### Flex 보드

- <Status>Change</Status> 대시보드 및 목록을 화면에 불러올 때 로딩 인디케이터 적용

- <Status>Feature</Status> Series 위젯 타입의 차트 설정에서 범례 표시 추가

- <Status>Feature</Status> Series 위젯 타입에 누적 차트 타입 추가

- <Status>Feature</Status> Series 위젯 타입의 차트 설정에서 차트 색상 기준 옵션 추가

    - 기존: 태그를 기준으로 차트 색상 표현
    - 추가: 필드 기준으로 차트 색상을 나타낼 수 있는 옵션

- <Status>Feature</Status> DB 상품군에서 다음 위젯 템플릿 추가

    - **DB 공통**
        - CPU % (xos) 위젯
        - Memory % (xos) 위젯
    - **MySQL**
        - 슬로우 쿼리
    - **PostgreSQL**
        - 슬로우 쿼리
        - PostgreSQL Wait Event 건수 위젯
        - PostgreSQL Client Ip 건수 위젯
        - PostgreSQL 통계 테이블
        - PostgreSQL 액티브 세션 건수
        - PostgreSQL 액티브 세션 테이블

#### 로그

- <Status>Change</Status> <UI>로그</UI> 메뉴의 기능에 대한 빠른 이해를 위해 메뉴명 변경

    * <UI>라이브 Tail</UI> → <UI>라이브 테일</UI>
    * <UI>탐색기</UI> → <UI>로그 트렌드</UI>
    * <UI>검색기</UI> → <UI>로그 검색</UI>

- <Status>Change</Status> <UI>로그 검색</UI> 메뉴에서 조회 내용을 <UI>Timestamp</UI> 기준으로 정렬하는 기능(<UI>최근 순</UI>/<UI>과거 순</UI>) 변경

- <Status>Feature</Status> <UI>로그 트렌드</UI>에서 검색키를 최대 2개까지 선택할 수 있도록 수정(검색값은 제한 없음)

- <Status>Change</Status> <code>Server</code> 프로젝트의 <UI>로그 설정</UI> 메뉴에서 에이전트 설정 옵션 변경

#### 경고 알림

- <Status>Feature</Status> <UI>이벤트 설정</UI> 메뉴에서 <UI>에이전트 중복 알림</UI> 옵션 추가

  > 모니터링 중인 에이전트 가운데 동일한 ID를 설정한 에이전트가 2개 이상일 때 지표를 중복으로 수집하면서 발생하는 경고 알림입니다. 에이전트 이름을 중복으로 설정하지 않았는지 에이전트 설정에서 관련 옵션을 확인하세요. 블루/그린과 같은 배포 상황에서 에이전트가 중복되어 경고 알림이 발생할 경우 이 기능을 이용하길 권장합니다. 같은 이름의 에이전트가 동시에 2개 이상 기동된 경우 선택한 시간 동안 알림을 받지 않습니다.

- <Status>Feature</Status> <UI>이벤트 수신 설정</UI> 메뉴에서 <UI>수신 태그 미설정 알림 받기</UI> 옵션 추가

  > ***이벤트 수신 태그***가 설정되지 않은 경고 알림을 받지 않으려면 해당 옵션을 선택하세요.
