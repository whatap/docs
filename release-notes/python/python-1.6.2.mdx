<Status>Feature</Status> <code>exit_with_parent_process_enabled</code> 옵션 추가

* <b>exit_with_parent_process_enabled</b> <em>bool</em> <br/>
  
  기본값 <code>false</code>

  Python 에이전트 <code>WHATAP_HOME</code> 경로에 있는 <em>whatap.conf</em> 파일을 통해 해당 옵션의 값을 설정하세요. 모니터링 대상 Python 프로세스 종료 시 <code>whatap_python</code> 프로세스의 종료 여부를 설정할 수 있습니다. 옵션의 값을 <code>true</code>로 설졍할 경우 모니터링 대상 애플리케이션 종료 시 <code>whatap_python</code> 프로세스가 함께 종료됩니다. 최대 1분이 소요됩니다.
