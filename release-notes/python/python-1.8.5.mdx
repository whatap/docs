2025년 06월 18일

<Status priority='1'>New</Status> *whatap.conf*에서 매트릭스 카테고리 `app_context_stat` 옵션 추가
    
    * **app_context_enabled** <span class='type'>Boolean</span>
        * `false`: 기본값
        * `true`: `app_context_stat` 수집 기능 전체 활성화
    
    * **app_context_parser**

        * `default, prefix, match`중 사용
        * `false`: 기본값
        * `default`: 기본 파서(첫 번째 경로 세그먼트 추출)
        * `prefix`: 접두사 매칭 파서(가장 유연함)
        * `match`: 정확한 매칭 파서(완전 일치만)
    
    * **app_context_parser_reset** <span class='type'>Number</span>

        * 파서의 내부 캐시와 설정을 강제로 재설정하는 카운터
        * `0`: 기본값
    
    * **app_context_path_depth** <span class='type'>Number</span>

        * URL에서 추출할 경로 세그먼트의 깊이 지정 (default 파서에서만 사용)
        * `1`: 기본값
    
    * **app_context_path_set** <span class='type'>String</span>

        * 기본값 없음
        * 모니터링할 경로와 해당 이름을 정의하는 매핑 설정 (`,`로 구분하여 사용)
            > 예제: `name1@url1,name2@url2,url3,name4@url4`
