2025년 07월 15일

- <Status priority='1'>New</Status> URL을 정규화한 경우, 원본 URL이 step에서 노출되는 옵션 추가
    
    * **step_origin_url_enabled** <span class='type'>Boolean</span>
        * `false`: 기본값

- <Status priority='1'>New</Status> standalone Python 프로세스 추적 옵션 추가
    
    * **standalone_enabled** <span class='type'>Boolean</span>
        * `false`: 기본값

    * **standalone_type** (`single-transaction`, `multiple-transaction` 중 사용)
        * `single-transaction`: 기본값, 파이썬 실행을 하나의 트랜잭션으로 모니터링함
        * `multiple-transaction`: 사용자가 지정한 모듈·클래스·메소드 단위로 트랜잭션 포인트를 설정해 추적함

    * **standalone_transaction_patterns** <span class='type'>String</span>
        * 기본값 없음
        * multiple-transaction 옵션에서 모니터링할 모듈·클래스·메소드를 정의하는 옵션
        * `[MODULE_NAME]:[CLASS_NAME].[FUNCTION_NAME]` 혹은 `[MODULE_NAME]:[FUNCTION_NAME]` 형식으로 값을 사용합니다. 콤마(,) 구분자를 통하여 여러 메소드 패턴을 지정할 수 있음
        * 예제. `myPackage:MyClass.__init__, myPackage:my_function`

- <Status priority='1'>Feature</Status> `Neo4j`드라이브 모듈 추적 기능 추가 (v5.0 이상만 지원)

- <Status priority='0'>Fixed</Status> URL을 정규화한 경우, 레코드 요약에서 원본 URL이 노출되지 않던 현상 수정
