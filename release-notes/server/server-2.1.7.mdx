## 프로세스 이름 수집 최적화#2

프로세스 중 최종 사용자 세션에 연동되는 휘발성 프로세스 이름은 대표이름으로 수집하도록 수정

- 대상: 유져가 oracle이며 실행 파일이 aaa_bbb_ccc 패턴의 프로세스는 aaa로 합산하여 수집
