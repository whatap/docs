## `공통`

***로그 이벤트 설정***

- 복합 로그 이벤트 기능 추가
- 실시간 로그 이벤트의 로그 필터링 조건 추가

***이벤트 수신 태그 설정***

1. 사용자별 이벤트 수신 설정 또는 3rd 플러그인에 태그를 추가
2. 메트릭스 이벤트 설정 시 태그를 선택하여 수신자를 지정

***조직 관리*** - 사용성 개선 및 기능 추가

- 조직, 그룹 관리 기능 사용성 개선
- 멤버 권한 수정 UI 개선
- 멤버 초대 기능 사용성 개선
- DB 프로젝트 지원팀 초대 기능 추가 (dbx@whatap.io)

***메트릭스 이벤트*** 

- 이벤트 대상 필터링 함수 추가 (like, and, or, isNull, isNotNull, isEmpty, isNotEmpty)
- 가이드 바로가기 링크 추가 

***이용 내역*** - 구매이력 정보 표시, 프로젝트 유료 전환 목록 CSV 다운로드 기능 추가 

***보고서*** - 보고서 종류 간소화

***Flex 보드***

- ActiveStatus 위젯 상세 조회 기능 수정
- 위젯 한글 검색 기능 수정 

***회원가입*** - 이메일 검증 로직 수정

***MFA*** - 이메일을 이용한 2차 인증 기능 수정, 2차 인증 비활성화 기능 추가

## 애플리케이션

***트랜잭션 프로파일*** 

- 에러가 있는 SQL 구간 다이어그램에 표시 추가
- SQL 통계 조회 링크 수정

***트랜잭션맵*** - 필터링 조건 직접 입력 시 단어 부분 검색 제한, 안내 문구 추가

***애플리케이션 대시보드*** - 히트맵 도움말에 가이드 문서 링크 추가
