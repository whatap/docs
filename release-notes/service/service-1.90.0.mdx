### 컨테이너 맵

- <Status>Change</Status> 요약보기 창이 열린 상태에서 상태별 개수 텍스트 짤림 현상 수정
- <Status>Change</Status> 로그 검색 탭에서 (로그 카테고리가 존재할 경우) 로그 대상 카테고리 수정
- <Status>Fixed</Status> 카테고리가 Pod일 때 컨테이너가 복수이면 일부 컨테이너 정보만 표시하는 버그 수정

### 애플리케이션 현황/성능 추이

- <Status>Fixed</Status> 에이전트 정보가 잘못 노출되는 오류 수정

### 로그

* <Status>Feature</Status> <UI>라이브 Tail</UI>, <UI>검색기</UI>, <UI>탐색기</UI>에 테이블 컬럼 순서 변경 기능 추가
* <Status>Change</Status> 로그 설정의 데이터 유지 기간 옵션 수정

  * 트라이얼 프로젝트: 최대 3일까지 데이터 유지
  * 유료전환 프로젝트: 최대 40일까지 데이터 유지
  * *front.conf* 파일에 `ui_log_retention_limit` 옵션을 추가하면 해당 값까지 데이터 유지하도록 수정

### 쿠버네티스

- <Status>Change</Status> 클러스터 프로젝트에서 헤더 메뉴를 통해 네임스페이스 이동 시 기존 메뉴를 유지한 상태로 이동할 수 있도록 수정
- <Status>Change</Status> 퍼포먼스 요약 페이지 위젯 데이터 수정

### `공통`

* <Status>Change</Status> 조회 및 분석 권한을 OFF 할 수 없도록 비활성화 처리 수정
