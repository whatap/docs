### `Server`

- <Status priority='1'>Feature</Status> **GPU 인벤토리** 키워드 검색 필터 추가

- <Status priority='1'>Feature</Status> **GPU 성능요약** 컬럼 설정의 필드별 설명 툴팁 추가

- <Status priority='1'>Feature</Status> <Cmdname sid="side_serverDetail" className="uitext" /> 서버 목록에 검색 입력 필드 추가

- <Status priority='1'>Feature</Status> <Cmdname sid="side_dashboard" className="uitext" /> 브라우저 크기에 따라 위젯 크기가 자동 조정되는 모드를 기본 적용하고, 스크롤이 생기더라도 위젯을 확대해 정상적으로 볼 수 있도록 위젯 높이 자동 조정 스위치를 추가

- <Status priority='1'>Changed</Status> **GPU 성능요약** `Performance_State` 필드값이 P0~P15 사이 값으로 표시되도록 변경

- <Status priority='1'>Changed</Status> **GPU 성능요약** &gt; <Cmdname sid="compare" className="uitext" /> 패널 

    - 단위가 bytes/s인 차트 값이 자동으로 단위 변환되도록 변경

    - `Performance_State` 차트의 Y축 범위를 P0~P15 고정하도록 변경

- <Status priority='1'>Changed</Status> **서버 목록** 하위의 **GPU 인벤토리**와 **GPU 성능요약** 메뉴를 **GPU** 그룹 하위로 이동

- <Status priority='0'>Fixed</Status> <Cmdname sid="side_serverDetail" className="uitext" /> 서버 선택 후 페이지 로드 시, 서버 목록에 마우스 호버해도 즉시 표시되지 않는 문제 수정

- <Status priority='0'>Fixed</Status> <Cmdname sid="side_serverMetricsExplorer" className="uitext" /> by GPU Process 하위의 지표 선택 시, 차트의 범례 label의 형식을 `process_name, pid, (process_type) - index(uuid)`로 표시되도록 수정

- <Status priority='0'>Fixed</Status> <Cmdname sid="side_serverInventory" className="uitext" /> 신규 에이전트 수집 필드 추가 시, 테이블 헤더에 필드명 표시되지 않는 현상 수정

- <Status priority='0'>Fixed</Status> <Cmdname sid="side_serverDetail" className="uitext" /> 차트 상세 보기 드로워가 닫힐 때, 선택 서버가 아닌 default 서버 선택으로 초기화되는 문제 수정

- <Status priority='0'>Fixed</Status> **GPU 성능요약** &gt; <Cmdname sid="compare" className="uitext" /> 패널 

    - 단위가 % 인 차트의 y축 최댓값이 100%를 넘지 않도록 수정

    - 단위가 없는 차트에서 제목에 빈 괄호가 표시되던 현상 수정
