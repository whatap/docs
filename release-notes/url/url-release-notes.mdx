## URL Monitoring Agent v2.0.2

배포일: 2022-01-05 

### Minor Change 

URL 모니터링 User-Agent 이름을 Java/1.8 에서 WhaTap URL Monitoring Agent 로 수정했습니다. 

## URL Monitoring Agent v2.0.1

배포일: 2021-10-07

### 버그 픽스

* FullChain 인증서를 사용하지 않는 일부 URL에서 `sun.security.validator.ValidatorException ` 발생하는 현상을 개선했습니다.

* 구 버전에서 지원하지 않던 SHA-256 ECDSA 알고리즘 인증서에 대한 URL 체크가 가능 하도록 수정했습니다.

## URL Monitoring Agent v0.0.3

배포일: 2019-04-25

### 버그 픽스

https 프로토콜의 URL 모니터링이 불가하여 하기 에러가 발생하던 건을 개선하였습니다.

```java title='Java'
sun.security.provider.certpath.SunCertPathBuilderException: unable to find valid certification path to requested target
```

SSL 기반으로 통신할 경우 인증서를 받아들여 정상 통신이 가능하도록 하는 매커니즘이 필요함에 따라, 클라이언트 모듈이 등록된 URL로 접근시에 받아들이게 되는 인증서를 허용하도록 개선하였습니다.
