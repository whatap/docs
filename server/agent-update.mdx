## Linux

설치된 에이전트를 최신 버전으로 업데이트하려면 다음 명령어를 실행하세요.

* **Debian / Ubuntu**

  ```bash title=SH
  sudo apt-get update
  sudo apt-get install whatap-infra
  sudo service whatap-infra restart
  ```

  특정 버전을 설치하려면 설치 시 버전 정보를 다음과 같이 명시하세요.
    
  ```bash
  sudo apt-get install whatap-infra={version}
  ```

* **CentOS / Amazon Linux / SUSE**
    
  ```bash title=SH
  sudo yum update whatap-infra
  sudo service whatap-infra restart
  ```
  
  특정 버전을 설치하려면 설치 시 버전 정보를 다음과 같이 명시하세요.
  
  ```bash
  sudo yum install whatap-infra-{version}
  ```

### GPU 인벤토리

GPU 인벤토리 기능을 사용하려면 다음 에이전트 버전 이상의 업데이트 및 설정이 필요합니다.

:::tip

지원 버전:  Linux 2.8.9 이상

:::

* **Amazon Linux, RHEL, Fedora, Oracle Linux, CentOS, Rocky Linux, SUSE**

  ```bash
  sudo yum update whatap-infra
  ```

* **Debian, Ubuntu**

  ```bash
  sudo apt-get update
  sudo apt-get install whatap-infra
  ```

#### GPU 옵션 

GPU 인벤토리 기능을 활성화하기 위해 아래 옵션이 활성화되어야 합니다.

* **nvidiav2.enabled** <span class='type'>Boolean</span>

  기본값 `false`
  
  nvidia gpu 지표 수집 옵션으로 물리GPU 구성정보 및 리소스 사용량 정보입니다.

* **nvidiav2.mig.enabled** <span class='type'>Boolean</span>

  기본값 `false`
  
  nvidia gpu mig 관련 지표 수집 옵션으로 MIG 구성정보 및 리소스 사용량 정보입니다. **nvidiav2.enabled** 옵션 `true` 설정이 필요합니다.

* **nvidiav2.proc.enabled** <span class='type'>Boolean</span>

  기본값 `false`

  nvidia gpu 할당 process 정보 수집 옵션입니다. **nvidiav2.enabled** 옵션 `true` 설정이 필요합니다.

* **nvidiav2.dcp.enabled** <span class='type'>Boolean</span>

  기본값 `true`

  nvidia dcp(Data Center Profiling) 지표 수집 옵션입니다. **nvidiav2.enabled** 옵션 `true` 설정이 필요합니다.

* **nvidiav2.interval.sec**

  기본값 `30초`

  nvidia 지표 수집 주기입니다. 

## Unix

* **FreeBSD**

  설치된 에이전트를 최신 버전으로 업데이트하려면 다음 명령어를 실행하세요.

    ```bash title=SH
    wget http://repo.whatap.io/freebsd/10/whatap-infra.txz
    pkg install whatap-infra.txz
    ```

* **AIX** 

  설치된 에이전트를 최신 버전으로 업데이트하려면 다음 명령어를 실행하세요. 

  ```bash 
  stopsrc -s whatap-infra
  cp /usr/whatap/infra/conf/whatap.conf /tmp/whatap.conf
  rpm -e whatap-infra
  rpm -Uvh http://repo.whatap.io/aix/noarch/whatap-infra-latest.noarch.rpm
  cp /tmp/whatap.conf /usr/whatap/infra/conf/whatap.conf
  startsrc -s whatap-infra
  ```

* **Oracle Solaris** 

  설치된 에이전트를 최신 버전으로 업데이트하려면 다음 명령어를 실행하세요. 

  ```bash 
  cp /usr/whatap/infra/conf/whatap.conf /tmp
  /etc/init.d/whatap-infra stop
  pkgrm whatap-infra
  wget https://repo.whatap.io/sunos/11/whatap-infra.latest.SPARC.pkg.tar.gz
  tar xzvf whatap-infra.latest.SPARC.pkg.tar.gz
  pkgadd -d .
  cp /tmp/whatap.conf /usr/whatap/infra/conf/whatap.conf
  /etc/init.d/whatap-infra start
  ```

* **HP-UX** 

  설치된 에이전트를 최신 버전으로 업데이트하려면 다음 명령어를 실행하세요.

## Windows

설치된 에이전트를 최신 버전으로 업데이트하려면 <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_agentInstall" className="uitext" /> 메뉴로 이동하세요. <Cmdname sid="agent-setup-guide" className="uitext" /> 섹션의 <Cmdname sid="server_installAgent_win_download" className="uitext" /> 안내에 따라 윈도우 서버 운영 체제용 설치 파일을 다운로드 후 *whatap_infra.exe* 파일을 실행하세요. 에이전트 업데이트 후 서비스가 재시작됩니다.

:::note

보안 설정 때문에 *.exe* 형식의 파일이 받아지지 않는 사용자를 위해 *.zip* 형식의 파일도 함께 제공합니다. 자세한 내용은 [다음 문서](../server/server-os/server-windows)를 참조하세요.

:::
