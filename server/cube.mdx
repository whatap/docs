## 큐브란?

와탭은 5분 단위로 만든 성능 통계를 **큐브**라고 부릅니다. 큐브 분석은 큐브에 저장된 5분 단위 성능 데이터를 활용한 분석 기능입니다.

<ImgLang img='cube.png' desc='큐브'/>

## ![](/img/number-01.png)큐브 셀렉트 패널

왼쪽의 큐브 셀렉트 패널에서 특정 시간 기준으로 큐브를 선택하세요. 큐브는 5분 단위로 저장되어 있기 때문에 특정 시간을 선택해야 합니다. 

* **패널 관심 지표**

  서버 큐브 셀렉트 패널의 관심 지표는 <Cmdname sid="cpu" className="uitext" />와 <Cmdname sid="memory" className="uitext" />입니다.

* **큐브 표시색**

  큐브는 선택된 지표의 수치에 따라 다른 색으로 표현됩니다.

  * <span class='highfg'>남색</span>: <span class='highfg'>높은 수치</span>

  * <span class='lowfg'>하늘색</span>: <span class='lowfg'>낮은 수치</span>

## ![](/img/number-02.png) 애플리케이션 선택자

상단 ![number 2](/img/number-02.png) 영역의 애플리케이션 선택자 옵션을 통해 특정 애플리케이션의 데이터를 선택할 수 있습니다. 오른쪽의 ![down-arrow](/img/down-arrow.svg) 버튼을 선택하면 다음과 같이 선택 가능한 전체 애플리케이션을 확인할 수 있습니다. 

<ImgLang img='cube-select-app.png' desc='큐브 애플리케이션 셀렉터'/>

* <Cmdname sid="select_all" className="uitext" />: 모든 애플리케이션을 조회합니다. 기본은 <Cmdname sid="select_all" className="uitext" />과 <Cmdname sid="oid_active" className="uitext" />이 선택되어있습니다. 

* <Cmdname sid="checked_only" className="uitext" />: 애플리케이션 중 사용자가 지정한 해당 애플리케이션만 조회합니다. 

* <Cmdname sid="search" className="uitext" />: 애플리케이션이 많은 경우 입력창을 통해 해당 애플리케이션을 검색할 수 있습니다.

* <font color='5B7DEC'>숫자</font>/<font color='979797'>숫자</font>: <font color='5B7DEC'>지정한 애플리케이션 개수</font>/<font color='979797'>전체 애플리케이션 개수</font>를 표시합니다.

## ![](/img/number-03.png) 큐브 데이터

:::note

* 차트 상단 오른쪽의 ![바 차트 아이콘](/img/ic-dashboard-bar-chart.svg) 아이콘을 선택하면 바형 차트를 제공합니다. 
* 차트 상단 오른쪽의 ![개별 아이콘](/img/ic-multiline-chart.svg) 아이콘을 선택하면 선형 차트를 제공합니다.

:::

![number 3](/img/number-03.png) 영역에서 해당 큐브 데이터를 <Cmdname sid="cpu" className="uitext" />, <Cmdname sid="memory" className="uitext" />, <Cmdname sid="disk" className="uitext" />, <Cmdname sid="network" className="uitext" /> 등의 차트를 통해 확인할 수 있습니다.

* <Cmdname sid="cpu" className="uitext" />

  프로젝트 내 애플리케이션을 CPU 사용량이 많은 순서대로 정렬해 표시합니다. 

* <Cmdname sid="memory" className="uitext" />

  프로젝트 내 애플리케이션을 메모리 사용량이 많은 순서대로 정렬해 표시합니다.

* <Cmdname sid="disk" className="uitext" />

  프로젝트 내 애플리케이션을 디스크 사용량이 많은 순서대로 정렬해 표시합니다.

* <Cmdname sid="network" className="uitext" />

  프로젝트 내 애플리케이션을 네트워크 입출력 트래픽량이 많은 순서대로 정렬해 표시합니다.
