**GPU 인벤토리**는 서버에 장착된 GPU의 사양과 상태를 체계적으로 관리할 수 있습니다.

:::note

**지원 에이전트 버전**

Linux에서만 지원되며, 에이전트 2.6.5 이상 버전이 필요합니다.

:::

## 기본 화면 안내

GPU 인벤토리는 시스템에 할당된 GPU 및 MIG(Multi-Instance GPU) 리소스의 전체 목록을 모니터링합니다. 리소스 필터 및 검색 기능이 제공되며, 각 GPU 인스턴스의 상세 정보를 확인할 수 있습니다. 

* 검색창에서 키워드를 검색할 수 있습니다.
* 조회 결과의 항목명을 클릭하면 오름차순 또는 내림차순으로 정렬을 변경할 수 있습니다.

### **컬럼 설정**

화면 오른쪽 상단의 **컬럼 설정** 버튼을 클릭한 후, 표시할 컬럼을 선택할 수 있습니다. 컬럼 설정 창에서 컬럼 선택 및 커스텀 컬럼 추가 후, **확인** 버튼을 클릭해 설정을 저장하세요. 표시 항목에서 컬럼의 순서를 드래그해 변경할 수 있습니다. 

- **컬럼 선택**

    사용자는 테이블에 표시할 컬럼을 선택할 수 있습니다. **초기화** 클릭 시, 기본 항목 선택 상태로 돌아갑니다.

    :::note

    선택한 컬럼은 브라우저 쿠키값으로 저장되어 페이지를 새로 고침해도 상태가 유지됩니다. 쿠키가 삭제되거나 다른 이유로 설정 오류가 발생하면 기본 항목 선택 상태로 초기화합니다.

    :::

- **컬럼 추가**

    사용자는 **+ 컬럼 추가** 버튼 선택 후 기본 제공 항목 외, 자신이 원하는 항목을 추가하여 커스텀 컬럼(`custom`)을 구성할 수 있습니다. 필요에 맞게 컬럼을 추가하고 서버 상태 및 성능을 더욱 세밀하게 모니터링해 보세요.

### **수정**

**수정** 버튼을 선택하여 서버 인벤토리 목록을 수정할 수 있습니다.

- **서버 별명**과 **기본 그룹** 컬럼 내용을 설정할 수 있습니다.

- 사용자가 추가한 커스텀 컬럼(`custom`) 내용을 설정할 수 있습니다.
