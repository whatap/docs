**GPU 인벤토리**는 서버에 장착된 GPU의 사양과 상태를 체계적으로 관리할 수 있습니다.

:::note

**지원 에이전트 버전**

Linux에서만 지원되며, 에이전트 2.8.5 이상 버전이 필요합니다.

:::

## 기본 화면 안내

GPU 인벤토리는 시스템에 할당된 GPU 및 MIG(Multi-Instance GPU) 리소스의 전체 목록을 모니터링합니다. 각 GPU 인스턴스의 상세 정보를 확인할 수 있습니다. 

* 검색창에서 키워드를 검색할 수 있습니다.
* 조회 결과의 항목명을 클릭하면 오름차순 또는 내림차순으로 정렬을 변경할 수 있습니다.

### ![컬럼 아이콘](/img/ico-column.svg) 컬럼 설정

화면 오른쪽 상단의 **컬럼 설정** 버튼을 클릭한 후, 표시할 컬럼을 선택할 수 있습니다. 컬럼 설정 창에서 컬럼 선택 및 커스텀 컬럼 추가 후, **확인** 버튼을 클릭해 설정을 저장하세요. 표시 항목에서 컬럼의 순서를 드래그해 변경할 수 있습니다. 

- **컬럼 선택**

    사용자는 테이블에 표시할 컬럼을 선택할 수 있습니다. **초기화** 클릭 시, 기본 항목 선택 상태로 돌아갑니다.

    :::note

    선택한 컬럼은 브라우저 쿠키값으로 저장되어 페이지를 새로 고침해도 상태가 유지됩니다. 쿠키가 삭제되거나 다른 이유로 설정 오류가 발생하면 기본 항목 선택 상태로 초기화합니다.

    :::

- **컬럼 추가**

    사용자는 **+ 컬럼 추가** 버튼 선택 후 기본 제공 항목 외, 자신이 원하는 항목을 추가하여 커스텀 컬럼(`custom`)을 구성할 수 있습니다. 필요에 맞게 컬럼을 추가하고 서버 상태 및 성능을 더욱 세밀하게 모니터링해 보세요.

### CSV 다운로드 및 업로드

1. ![다운로드 아이콘](/img/ico-download.svg) **CSV 다운로드** 버튼을 클릭해 현재 시점 기준의 GPU 인벤토리 정보를 다운로드합니다. 

1. 해당 파일을 열어 컬럼의 정보를 수정합니다.

1. ![업로드 아이콘](/img/ico-upload.svg) **CSV 업로드** 버튼을 클릭해 수정한 파일을 업로드하면 값이 일괄 업데이트됩니다. 

    * 변경 건 수, Error, 미반영 건 수를 확인할 수 있습니다. 

1. **적용** 버튼을 클릭하면 GPU 인벤토리 목록에 반영됩니다. 

:::note

**반영되지 않는 항목**

기존 인벤토리 컬럼에 없는 컬럼 항목은 반영되지 않습니다. 
* 컬럼 추가는 인벤토리 목록 **+ 컬럼 추가** 버튼을 클릭해 추가할 수 있습니다. GPU 삭제 및 추가는 반영되지 않습니다. 

:::

### 수정

![수정 아이콘](/img/ic-edit.svg) **수정** 버튼을 클릭해 GPU 인벤토리 목록을 수정할 수 있습니다.

- **서버 별명**과 **기본 그룹** 컬럼 내용을 설정할 수 있습니다.

- 사용자가 추가한 커스텀 컬럼(`custom`) 내용을 설정할 수 있습니다.
