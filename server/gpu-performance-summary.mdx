홈 화면 &gt; 프로젝트 선택 &gt; <Cmdname sid="side_serverList" className="uitext" /> &gt; **GPU 성능요약**

시스템에 연결된 GPU의 상태 및 성능 지표를 실시간으로 모니터링하고 관리할 수 있습니다. 물리 GPU와 MIG 인스턴스의 정보를 함께 확인할 수 있으며, 각 항목을 정렬하거나 필터링하여 효율적으로 사용할 수 있습니다.

:::note

**지원 에이전트 버전**

Linux에서만 지원되며, 에이전트 2.6.5 이상 버전이 필요합니다.

:::

## **기본 화면 안내**

![Pause icon](/img/ic-pause-green.svg) <UI>Live</UI> 를 클릭하여 실시간 데이터 조회를 일시 정지하거나 실시간 데이터를 조회하거나 일시 정지할 수 있습니다. 검색창에서 GPU 정보를 검색할 수 있습니다. 

데이터 목록에서 조회 결과의 항목명을 클릭하면 오름차순 또는 내림차순으로 정렬을 변경할 수 있습니다. 

* **Total**: 전체 GPU 인스턴스 수 (Physical + MIG 포함)
* **Physical**: 실제 GPU 장치 수
* **MIG**: MIG(Multi-Instance GPU) 인스턴스 수

### GPU 정보

| 컬럼명 | 설명 |
| --- | --- |
| Status | 해당 GPU 인스턴스의 현재 상태 (예: `Inactive`, `N/A`) |
| HostName | GPU가 연결된 서버의 호스트명 |
| GPU Index | 물리 GPU 또는 MIG 인스턴스의 고유 인덱스 (MIG 인스턴스는 `0/6/0` 등 형태로 표기) |
| Model Name | GPU 모델명 (예: NVIDIA A100-SXM4 등) |
| GPU Type | `Physical` 또는 `MIG` 중 해당 인스턴스의 종류 |


### GPU 성능 지표

| 컬럼명 | 설명 |
|--------|------|
| GPU_Util (%) | GPU 사용률 (%) <br /> - Physical GPU에서만 표시되며, MIG 인스턴스는 제외 |
| Encoder_Util (%) | 하드웨어 인코더 사용률 (%) <br /> - Physical GPU에서만 표시 |
| Decoder_Util (%) | 하드웨어 디코더 사용률 (%) <br /> - Physical GPU에서만 표시 |
| GR_Engine_Active_Util (%) | Graphic 엔진 활성 시간 비율 |
| SM_Active_Util (%) | SM에서 최소 하나의 워프가 실행된 시간 비율 |
| SM_Occupancy (%) | SM에서 실제 실행 중인 워프 비율 |
| Tensor_Core_Util (%) | Tensor 코어가 활성 상태였던 시간 비율 |
| Memory_Util (%) | 메모리 복사 엔진 사용률 (%) <br /> - GPU와 호스트 간 메모리 전송 활동률 |
| DRAM_Active_Util (%) | DRAM이 활성화된 시간 비율 <br /> - Physical GPU에서만 표시 |
| FP64_Compute_Util (%) | 64비트 부동소수점 연산 파이프 활성 시간 비율 |
| FP32_Compute_Util (%) | 32비트 부동소수점 연산 파이프 활성 시간 비율 |
| FP16_Compute_Util (%) | 16비트 부동소수점 연산 파이프 활성 시간 비율|
| BAR1_Total_Memory (Bytes) | BAR1 메모리 총량 |
| BAR1_Used_Memory (Bytes) | 사용 중인 BAR1 메모리 크기 |
| BAR1_Free_Memory (Bytes) | 남은 BAR1 메모리 크기 |
| FB_Total_Memory (Bytes) | 프레임 버퍼 전체 메모리 크기 |
| FB_Free_Memory (Bytes) | 프레임 버퍼 사용 가능 메모리 크기 |
| FB_Used_Memory (Bytes) | 프레임 버퍼 사용 중인 메모리 크기 |
| FB_Reserved_Memory (Bytes) | 예약된 프레임 버퍼 메모리 크기 |
| FB_Memory_Usage (%) | 프레임 버퍼 메모리 사용률(%) |

### 컬럼 설정

화면 오른쪽 상단의 **컬럼 설정** 버튼을 클릭한 후, 표시할 컬럼을 선택할 수 있습니다. **컬럼 설정** 창에서 컬럼 선택 후, **확인** 버튼을 클릭해 설정을 저장하세요.

:::caution

컬럼 표시 항목 중 Status, HostName은 삭제할 수 없습니다. 

:::

- **컬럼 선택**

    사용자는 테이블에 표시할 컬럼을 선택할 수 있습니다. **초기화** 클릭 시 기본 항목 선택 상태로 돌아갑니다.

    :::note

    선택한 컬럼은 브라우저 쿠키값으로 저장되어 페이지를 새로 고침해도 상태가 유지됩니다. 쿠키가 삭제되거나 다른 이유로 설정 오류가 발생하면 기본 항목 선택 상태로 초기화합니다.

    :::
