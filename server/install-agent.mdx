## 설치 전 확인사항

에이전트를 설치하기 전에 다음을 확인하세요.

* [시작하기](getting-started)에서 프로젝트 생성 및 액세스 키 발급을 완료했는지 확인하세요.
* [지원 환경](supported-spec)에서 사용 중인 운영체제가 지원되는지 확인하세요.

## 기본 설치 절차

1. <Cmdname sid="side_agentInstall" className="uitext" /> 화면에서 모니터링 대상 서버의 운영체제를 선택합니다.

2. 운영체제별 상세 가이드에 따라 에이전트 패키지를 다운로드하고 설치합니다.

3. 설정 파일(*whatap.conf*)에 프로젝트 액세스 키와 수집 서버 정보를 입력합니다.

4. 에이전트를 시작합니다.

:::tip

<Cmdname sid="side_agentInstall" className="uitext" /> 화면으로 이동하지 않은 경우, 왼쪽 메뉴에서 <Cmdname sid="all_projects" className="uitext" /> > 해당 프로젝트를 선택하세요.

:::
