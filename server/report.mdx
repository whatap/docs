## 일간 서버 보고서 {#S001_ServerDaily_TagReport}

<Cmdname sid="S001_ServerDaily_TagReport" className="uitext" report />는 하루 동안의 서버 자원 사용량을 확인할 수 있는 보고서입니다. 시작일, 시작 시간, 종료 시간을 직접 설정하여 조회할 수 있습니다.

### 개요 {#overview}

<ImgLang img='server_daily_report_01.png' desc='일간 서버 보고서 개요'/>

프로젝트 이름과 호스트 수, 발생한 알림 건수를 요약해서 볼 수 있습니다. 호스트 건수와 알림 건수의 경우 전날과 비교하여 얼마나 변동이 있었는지를 함께 확인할 수 있습니다.

* <Cmdname sid="host_count" className="b600" report/>: 조회 기간 동안의 총 서버 개수입니다.
* <Cmdname sid="alert_count" className="b600" report/>: 조회 기간 동안 발생한 알림의 전체 건수입니다.

### 서버목록 {#server_list}

<ImgLang img='server_daily_report_02.png' desc='일간 서버 보고서 서버 목록'/>

서버별로 일간 주요 지표들을 확인할 수 있습니다. CPU 최댓값(CPU Max)과 메모리 최댓값(MEM Max)의 경우 해당 수치가 발생된 시간을 함께 표시합니다.

알림 건수는 7일 전, 1일 전, 조회 당일을 함께 조회하여 비교해 볼 수 있습니다.

:::note

CPU, 메모리의 평균, 최댓값이 90% 이상인 경우 <span class='loger'>빨간색</span>으로 표시됩니다.

:::

### 알림 기록 {#alert_history}

<ImgLang img='server_daily_report_03.png' desc='알림 목록'/>

조회한 시간 동안 발생한 알림 기록에 대해 발생 시간, 등급, 발생한 서버 이름, 해당 알림이 발생한 원인을 확인할 수 있습니다.
각 알림에 대한 상세 정보는 <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventHistory" className="uitext" /> 메뉴에서 조회할 수 있습니다.
해당 프로젝트에 대해 알림 기록이 없는 경우 <Cmdname sid="no_data" className="uitext" report />라는 안내 문구가 뜰 수 있습니다.

:::note

알림의 레벨에 따라 <span class='loger'>Critical</span>은 <span class='loger'>빨간색</span>, <span class='wer'>Warning</span>은 <span class='wer'>주황색</span>, <span class='mstop'>Info</span>는 <span class='mstop'>회색</span>으로 표시됩니다.

:::

## 주간 서버 보고서 {#S100_ServerWeeklyReport}

<Cmdname sid="S100_ServerWeeklyReport" className="uitext" report/>는 일주일간 서버의 CPU, Memory 사용량을 확인할 수 있는 보고서입니다.
시작일을 직접 설정할 수 있으며, 조회 당일 기준 7일 전부터 조회 가능합니다.
시작일로부터 7일간의 데이터를 조회합니다.

### 전체 자원 사용 목록 {#total_resource_used_list}

#### <Cmdname sid="_cpu" report/>

<ImgLang img='server_weekly_report_cpu.png' desc='주간 전체 자원 사용 목록 CPU'/>

조회 기간 동안 각 서버의 CPU 사용률을 일별로 조회합니다.
조회되는 항목으로는 CPU 평균, 최대 95%, 최댓값이 있으며, 각 수치는 백분율로 표시됩니다.

:::note 

**최대 95%**

상위 5% 데이터를 제외한 최댓값을 의미합니다. 서버 재기동 등의 이유로 인해 순간적으로 높게 나온 수치를 제외하고 보기 위함입니다.

:::

#### <Cmdname sid="_mem" report/>

<ImgLang img='server_weekly_report_memory.png' desc='주간 전체 자원 사용 목록 메모리'/>

조회 기간 동안 각 서버의 메모리 사용량을 일별로 조회합니다.
조회되는 항목으로는 메모리 사용 평균값, 최댓값이 있으며, 각 수치는 백분율로 표시됩니다.

#### <Cmdname sid="_disk" report/>

<ImgLang img='server_weekly_report_disk.png' desc='주간 전체 자원 사용 목록 디스크'/>

조회 기간 동안 각 서버에 마운트 된 파일 시스템별 지표입니다.
마운트 위치와 전체 디스크, 사용률을 조회하며 각 수치는 백분율로 표시됩니다.

:::note

* 수치가 90% 이상인 경우 <span class='loger'>빨간색</span>으로 표시됩니다. 
* 그룹명에는 `okindName`이 표시되며, `okindName`을 설정하지 않은 경우 빈칸으로 표시될 수 있습니다.

:::

## 월간 서버 보고서 {#S200_ServerMonthlyReport}

<Cmdname sid="S200_ServerMonthlyReport" className="uitext" report />는 한 달 동안의 서버의 자원 사용량 및 정보를 확인할 수 있는 보고서입니다.
시작일을 직접 설정할 수 있으며, 조회 당일 기준 30일 전부터 조회 가능합니다.

### 개요 {#overview}

<ImgLang img='server_monthly_report_01.png' desc='월간 서버 보고서 개요'/>

한 달 동안 서버의 전체 자원 사용 내역을 취합하여 볼 수 있습니다.

* <Cmdname sid="total_host" className="b600" report/>: 조회 기간 동안의 총 서버 개수입니다.
* <Cmdname sid="total_core" className="b600" report/>: 조회 기간 동안 사용된 총 코어 개수입니다.
* <Cmdname sid="total_disk" className="b600" report/>: 조회 기간 동안 마운트 된 총 파일 시스템 크기입니다.
* <Cmdname sid="total_alarms" className="b600" report/>: 조회 기간 동안 발생한 알림의 전체 건수입니다.

### 서버정보 {#server_info}

<ImgLang img='server_monthly_report_02.png' desc='서버 정보'/>

조회 기간 동안 가동된 서버의 IP 주소, OS 정보, CPU 정보, 메모리 사용량을 보여줍니다.

### 알림내역 {#alarms_list}

<ImgLang img='server_monthly_report_03.png' desc='알림 내역'/>

조회 기간의 전체 서버에 대한 총 알림 건수를 확인할 수 있으며, 서버별로 발생한 알림을 주 단위로 나누어 볼 수 있습니다.

### 서버 사용 목록 {#server_usage_history}

<ImgLang img='server_monthly_report_04.png' desc='서버 사용 목록'/>

서버별 CPU 평균, 최댓값과 메모리 평균, 최댓값을 일자별로 한눈에 확인할 수 있습니다.

:::note

수치가 90% 이상인 경우 <span class='loger'>빨간색</span>으로 표시됩니다. 

:::
