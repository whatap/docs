와탭 모니터링 서비스 초기 화면에서 프로젝트를 선택한 후, <Cmdname sid="side_serverList" className="uitext" /> &gt; <Cmdname sid="side_serverList" className="uitext" /> 메뉴를 선택하세요. <Cmdname sid="side_serverList" className="uitext" />은 수집 서버의 전체 현황을 나타내며 서버 관리기능 및 서버 간 비교 기능을 제공합니다.

<ImgLang img='server-list.png' desc='서버 목록'/>

## ![](/img/number-01.png) 데이터 갱신 · 서버 상태 · 검색

### 데이터 갱신 

서버 목록 페이지의 데이터는 60초마다 자동으로 갱신합니다. 사용자는 중지한 자동 갱신을 재시작할 수 있습니다.

* ![일시정지 아이콘](/img/ic-pause-green.svg) <Cmdname sid="paused_agent" className="uitext" />: 해당 에이전트의 이벤트 발생을 일지 정지합니다. 서버 목록에서 일시 중지 상태로 표시됩니다.

* ![재생 아이콘](/img/ic-play-grn.svg) <Cmdname sid="resume_agent" className="uitext" />: 일시 정지된 이벤트를 재시작합니다. 해당 에이전트의 이벤트 발생을 확인할 수 있습니다.

:::note 

시간 선택자 이용 방법에 대한 자세한 내용은 [다음 문서](usage-ui#timeselector)를 참조하세요.

:::

### 서버 상태

수집 서버의 상태를 표시합니다. 상태 정보는 1분 간격으로 자동 갱신됩니다.

* ![비활성 아이콘](/img/ico-disable-agent.svg) <Cmdname sid="status_inactive" className="b600" />: 에이전트가 해당 서버의 데이터 수집이 불가한 상태입니다.

* ![위험 아이콘](/img/ic-warning-critical.svg) <Cmdname sid="status_fatal" className="b600" />: 수집 서버에서 위험 이벤트가 발생한 상태입니다.

* ![경고 아이콘](/img/ic-warning-filled.svg) <Cmdname sid="status_warning" className="b600" />: 수집 서버에서 경고 이벤트가 발생한 상태입니다.

* ![정상 아이콘](/img/ic-check-circle-filled.svg) <Cmdname sid="status_ok" className="b600" />: 에이전트가 수집 서버에서 정상적으로 작동 중인 상태입니다.

* ![일시정지 아이콘](/img/ic-s-pause.svg) **알림 정지**: 선택된 서버의 이벤트 발생을 일시 정지한 상태입니다.


### 검색

상단 오른쪽 ![number-s](/img/number-01.png) 검색창을 활용해 에이전트 명과 IP를 검색할 수 있습니다.

## ![](/img/number-02.png) 필터

입력 창에 값을 직접 입력하거나 필터 추가하기로 필터를 지정할 수 있습니다. 필터 태그는 `필터 키`, `조건`, `검색 값`의 순서로 입력합니다. 

#### 필터 키, 조건, 검색 값 입력

* `필터 키` 입력 시 에이전트 명, CPU Used %, Memory Used %, Disk Used %, Disk I/O %, Network Traffic, private_ip 옵션을 제공합니다.

* `조건` 입력 시 `>`, `<`, `<=`, `>=`, `==`, `!=` 옵션을 제공합니다.

* `검색 값` 입력 시 일치 검색(`>`, `<`, `<=`, `>=`, `==`)일 때 <span class='normal'>파란색</span>으로, 제외 검색(`!=`)일 때 <span class='vslow'>붉은색</span>으로 하이라이팅합니다. 

* `검색 값` 입력 시 대소문자 구분 옵션을 활용해 검색할 수 있습니다.

:::note 

* 필터 태그가 2줄 이상 길어지는 경우 ![아이콘](/img/up-arrow.svg) **접기** 아이콘을 선택해 접어둘 수 있습니다.

* 필터 태그 입력 후 입력 창에서 Shift와 Tab 키를 동시에 사용하여 이전 필터 태그로 이동할 수 있습니다. 

* 필터 태그 입력 후 입력 창에서 Tab 키를 통해 다음 필터 태그로 이동할 수 있습니다. 

:::

#### 필터 태그 제거

* Backspace로 삭제할 수 있습니다.

* 태그의 **X** 아이콘 선택 시 태그를 삭제할 수 있습니다.

* 입력 창의 전체 삭제 **X** 아이콘 선택 시 전체 태그를 삭제할 수 있습니다.

#### 필터 즐겨찾기 {#log-filter-fav}

필터 **즐겨찾기** 기능을 제공합니다. 입력 창 오른쪽의 ![즐겨찾기 아이콘](/img/ic-star.svg) **즐겨찾기** 아이콘을 선택하여 원하는 필터 검색 조건을 즐겨찾기에 **추가**, **제거** 및 기존 즐겨찾기를 선택할 수 있습니다.

#### 필터 적용 예외 상황 

* 숫자만 입력할 수 있는 인덱스(`.n`으로 끝나는 `검색 키`)를 입력한 태그에서 `검색 값`은 숫자만 입력할 수 있습니다.

* 중복된 `조건 키`, `검색 값`은 입력할 수 없습니다.

* `조건 키`, `검색 값` 중 하나라도 없는 태그가 존재할 때 검색할 수 없습니다. 유효하지 않는 태그의 경우 <span class='mstop'>회색</span>으로 표시합니다.

## ![](/img/number-03.png) 서버 현황

에이전트가 할당된 전체 서버의 현황을 나타냅니다. 에이전트 명 목록에서 에이전트를 선택하면 <Cmdname sid="side_serverDetail" className="uitext" /> 페이지로 이동합니다. <Cmdname sid="side_serverDetail" className="uitext" /> 메뉴에 대한 자세한 내용은 [다음 문서](server-detail)를 참조하세요.

#### 파일 시스템 정보

**파일 시스템 정보**는 파일 시스템을 자세히 보고싶은 에이전트 명의 왼쪽 ![파일 시스템 상세보기 아이콘](/img/ico-detail-view-gr.svg) 아이콘을 클릭하면 확인할 수 있습니다.

### 컬럼 설정

* ![컬럼 아이콘](/img/ic-column.svg) <Cmdname sid="column_settings" className="uitext" />

  ![number-s](/img/number-03.png) 영역 상단 메뉴에서 ![컬럼 아이콘](/img/ic-column.svg) <Cmdname sid="column_settings" className="uitext" /> 버튼을 클릭하면 테이블에 표시할 컬럼을 선택할 수 있습니다.

  <ImgLang img='server-list-cl.png' desc='서버 목록 컬럼'/>

  :::note

  선택한 **컬럼**은 브라우저 쿠키값으로 저장되어 페이지 새로 고침 후에도 상태가 유지됩니다. 쿠키 삭제 또는 기타 이유로 설정 간 오류가 발생하면 전체 선택 상태로 초기화합니다.
  
  :::

* **컬럼 너비 설정**

  컬럼 제목의 가장자리를 드래그해 컬럼 너비를 설정할 수 있습니다.

* **컬럼 정렬**

  컬럼 제목을 클릭해 오름차순 또는 내림차순 정렬이 가능합니다.

  :::note

  **컬럼 너비** 및 **컬럼 정렬** 설정값은 브라우저 쿠키로 저장되어 페이지 새로 고침 후에도 상태가 유지됩니다. 쿠키 삭제 또는 기타 이유로 설정 간 오류가 발생하면 미 설정 상태로 초기화합니다.
  
  :::

#### 프리셋 관리

<Cmdname sid="side_serverList" className="uitext" /> &gt; <Cmdname sid="side_serverList" className="uitext" />에서 ![컬럼 아이콘](/img/ic-column.svg) <Cmdname sid="column_settings" className="uitext" /> 버튼을 클릭하면 프리셋을 관리하고 공유할 수 있습니다. 

선택한 컬럼 조합은 프리셋으로 저장할 수 있으며, 동일 프로젝트에 접근한 다른 사용자와 공유됩니다. 최근에 사용한 프리셋은 개인 브라우저에 저장되어, 재접속 시 자동으로 불러옵니다. 최근 프리셋이 삭제된 경우 기본 프리셋(<Cmdname sid="kube_default" className="uitext" />)으로 초기화됩니다. 프리셋 설정 방법은 [다음 문서](metrics-explorer#setpreset)를 참고하세요.

* 컬럼별 너비 및 정렬 상태는 사용자의 브라우저에 저장되며, 메뉴 재접속 시 자동으로 적용됩니다. 해당 설정은 사용자 개인에게만 적용되며, 다른 사용자와 공유할 수 없습니다.

:::caution

**프로젝트 수정 권한을 가진 사용자만 프리셋을 저장 및 삭제할 수 있습니다.** 멤버 권한에 대한 자세한 내용은 [다음 문서](https://docs.whatap.io/project/project-structure#member-auth)를 참고하세요.
  
:::

### 새로고침

![새로고침 아이콘](/img/ic-refresh2.svg) <Cmdname sid="refresh" className="uitext" /> 버튼을 클릭하면 사용자는 현재 시간을 기준으로 데이터를 갱신할 수 있습니다.

### 에이전트 이전

![이전 아이콘](/img/ic-swap.svg) <Cmdname sid="BTN07897" className="uitext" /> 버튼을 선택해 에이전트를 다른 프로젝트로 이전할 수 있습니다.

<ImgLang img='server-list-agent-transfer.png' desc='서버 목록 에이전트 이전'/>

1. 오른쪽 화면에서 이전할 대상 프로젝트를 선택합니다.

1. 왼쪽 화면에서 이전할 에이전트를 선택하세요.

1. ![right-arrow](/img/right-arrow.svg) **오른쪽 화살표** 버튼를 클릭해 대상 프로젝트로 이전합니다.

1. <Cmdname sid="save" className="uitext" /> 버튼을 누르면 이전이 완료됩니다.

:::note

* 실행 직후 10분 이내에 다시 동일 에이전트를 이동할 경우 충돌 현상을 유발할 수 있습니다. 
* 수집된 데이터는 이전되지 않습니다. 과거 데이터 조회를 위해 기존 프로젝트는 당분간 삭제하지 마세요.

:::


### 서버 간 비교

![차트 비교 아이콘](/img/ic-multiline-chart2.svg) <Cmdname sid="TTL06636" className="uitext" /> 버튼을 클릭하면 서버 간 비교 차트를 다음과 같이 제공합니다. 

<ImgLang img='server-list-compare.png' desc='서버 간 비교 sc'/>

1. ![차트 비교 아이콘](/img/ic-multiline-chart2.svg) <Cmdname sid="TTL06636" className="uitext" /> 버튼을 선택하세요.

1. ![number-s](/img/number-06.png)과 같이 서버 목록 왼쪽에 <Cmdname sid="TTL06636" className="uitext" /> 컬럼이 추가된 것을 확인하세요.

1. <Cmdname sid="TTL06636" className="uitext" /> 컬럼에서 비교를 원하는 서버를 선택하세요.

1. 비교 기능이 활성화되면 ![number-s](/img/ic-number-07.svg) 영역에서 비교 차트를 조회할 수 있습니다.

:::note

* 비교 서버는 5개까지 선택 가능합니다.

* 서버별 고유색이 할당됩니다. 고유색은 ![number-s](/img/ic-number-07.svg) <Cmdname sid="compare" className="uitext" />에서 서버 구별을 위해 사용됩니다.

* <Cmdname sid="check_period" className="uitext" /> 버튼을 선택하면 과거 데이터를 조회할 수 있습니다.

:::

### 모니터링 관리

에이전트 명령을 통해 서버를 관리하는 기능입니다. ![서버 관리 아이콘](/img/ico-setting.svg) <Cmdname sid="sms_management" className="uitext" /> 버튼을 클릭하면, 관리 기능을 적용할 서버를 선택할 수 있고, ![일시정지 아이콘](/img/ic-s-pause.svg) **알림 정지**, ![재시작 아이콘](/img/ic-caret-right.svg) **알림** <Cmdname sid="resume_agent" className="uitext" />, ![해지 아이콘](/img/ic-delete-critical.svg) **제외** 버튼이 활성화됩니다.

* ![일시정지 아이콘](/img/ic-s-pause.svg) **알림 정지**: 선택된 서버의 이벤트 발생을 일시 정지합니다.

* ![재시작 아이콘](/img/ic-caret-right.svg) **알림** <Cmdname sid="resume_agent" className="uitext" />: 선택된 서버 중 알림 정지 상태인 서버의 이벤트 알림을 다시 활성화합니다. 해당 에이전트의 이벤트 발생을 확인할 수 있습니다.

* ![해지 아이콘](/img/ic-delete-critical.svg) <Cmdname sid="exclude" className="uitext" />: 선택된 서버의 작동을 중지합니다. 서버 목록에서 제외되며 서버 현황을 확인할 수 없습니다.
