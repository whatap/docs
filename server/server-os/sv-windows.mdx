신규 생성한 서버 모니터링 프로젝트를 선택하면 ***에이전트 설치*** 화면이 나타납니다. 프로젝트 ***관리*** > ***에이전트 설치*** 메뉴를 통해서도 가능합니다.

![](/img/server-install-windows.png)

1. **에이전트 파일**을 다운로드하세요.

    ***관리*** > ***에이전트 설치***  화면 상단 영역의 ![number-s](/img/number-01.png) OS 탭에서 서버 OS를 선택하세요. ![number-s](/img/number-02.png) ***설치 파일 다운로드*** 탭에서 *whatap_infra.exe* 혹은 *whatap_infra.zip* 버튼을 선택해 에이전트 설치 파일을 다운로드 하세요.  

    :::note

    * 보안 설정으로 인해 *.exe* 형식의 파일이 받아지지 않는 사용자를 위해 *.zip* 형식의 파일도 함께 제공합니다.
    * 서버 보안을 위해 브라우저를 통한 직접 설치 보다 다운로드 받은 설치 파일을 실행하는 것을 권장합니다.
    
    :::

1. 모니터링 대상 서버에 접속해 **에이전트 파일**을 업로드 하세요.
  
1. 다운로드한 **설치 파일**을 두 번 클릭해 실행하세요.  

1. ![number-s](/img/number-03.png) ***설치 파일 실행*** 탭으로 이동하여 발급받은 프로젝트 액세스 키와 서버 IP를 확인하세요.

1. 화면의 안내에 따라 ***License Key*** 항목에는 프로젝트 액세스 키를, ***WhaTap IP*** 항목에는 와탭 서버의 IP 주소를 입력하세요.

    ![](https://img.whatap.io/media/user_guide_server/management/1230.png)  

1. ***Next*** 버튼을 클릭하세요.

1. 다음 화면에서 ***Install*** 버튼을 클릭하세요. 설치를 진행합니다.

1. ***Finish*** 버튼을 클릭해 설치를 완료하세요.
