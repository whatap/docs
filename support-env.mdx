## DB 모니터링

### 데이터베이스 종류별 지원 버전

|환경|지원 환경|
|----|----|
|Oracle|Oracle 10g 이상|
|Tibero|Tibero 5.x 이상|
|CUBRID|CUBRID 9 이상|
|MS-SQL|SQL Server 2005 이상|
|MySQL|MySQL 5.5 이상|
|MariaDB|MariaDB 5.5 이상|
|Altibase|Altibase 6 이상|
|Redis|Redis 6.0.5 이상|
|MongoDB|MongoDB 4.2 이상|
|PostgeSQL|EBD 9.2 이상|

:::note

* DB 모니터링 적용 시 DB Metadata(Dictionary, performance_schema) 조회 권한이 있는 모니터링용 계정이 필요합니다.

* XOS 에이전트는 x86 아키텍쳐 기반에서 동작하는 OS 환경에만 적용할 수 있습니다.

:::

|----|----|
|Java|Java Agent 2.1.0 버전 이상|
|PHP|PHP Agent 2.3.2 버전 이상|
|Python|Python Agent 1.2.2 버전 이상|
|Server|Server Agent 2.1.2 버전 이상|
