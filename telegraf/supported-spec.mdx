와탭 Telegraf 에이전트를 설치하기 전에 지원 환경을 확인하세요. 

* **운영체제**

  * Linux Red Hat 6.x x64 이상
  * Ubuntu 12.x x64 이상
  * FreeBSD 10.x x64 이상

* **Telegraf**

  * Telegraf 1.3 이상 

  :::note
  
  socket_writer ouput plugin을 지원하는 버전만 가능합니다.
  
  :::
