**카테고리 관리**에서 데이터를 체계적으로 분류하고 관리할 수 있습니다. 여러 카테고리로 정보를 나누고, 필요한 항목을 빠르게 찾고 정리할 수 있습니다. 카테고리를 **조회 · 추가 · 수정 · 삭제** 기능을 제공합니다. 

**카테고리 항목**에서 카테고리를 클릭하면, 해당 카테고리로 필터링된 **URL 목록 관리**로 이동합니다.

<ImgLang img='url-pro/category-management.png' desc='zkx' />

### 카테고리 추가

1. **카테고리 관리**의 오른쪽 위의 **+ 카테고리 추가** 버튼을 클릭하세요.
2. **카테고리 추가** 창에서 정보 입력하세요.
    - **카테고리명**(필수): 동일 레벨에서 중복 불가, 최대 50자
    - **적용할 URL**(선택): 카테고리 생성과 동시에 등록된 URL에 적용 가능
3. **추가** 버튼 클릭하세요.

### 카테고리 수정

1. 수정할 카테고리의 ![csv 아이콘](/img/ico-edit.svg) **수정** 아이콘을 클릭하세요.
2. **카테고리 수정** 창에서 수정한 후, **수정** 버튼을 클릭하세요.
    - 수정 가능 항목: 카테고리명

### 카테고리 삭제

**개별 삭제**

1. 삭제할 카테고리의 ![csv 아이콘](/img/ico-trash.svg) **삭제** 아이콘을 클릭하세요.
2. 삭제 확인 팝업에서 **삭제** 버튼을 클릭하세요.

**일괄 삭제**

1. 카테고리 목록에서 삭제할 항목들을 체크박스로 선택하세요.
2. 화면 오른쪽 위의 **삭제** 버튼 클릭하세요. 
3. 삭제 확인 팝업에서 **삭제** 버튼을 클릭하세요.

:::caution

삭제된 카테고리는 **복구할 수 없습니다**.

    * 해당 카테고리가 설정된 URL은 **자동으로 ‘미분류’로 변경**됩니다.

:::
