## 커스텀 필드 관리

커스텀 필드는 조직 내 특정 정보를 URL에 추가하여 관리할 수 있도록 지원하는 기능입니다.

### 커스텀 필드 추가

1. **커스텀 필드 추가** 버튼 클릭
2. **필드명** 입력
3. **추가** 버튼 클릭

### 커스텀 필드 수정

- 연필 아이콘 클릭 후 필드명 수정
- **수정** 버튼 클릭

### 커스텀 필드 삭제

- 휴지통 아이콘 클릭 또는 다중 선택 후 **삭제**
- 해당 필드를 사용하는 URL의 커스텀 정보는 제거됨
