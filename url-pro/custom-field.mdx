커스텀 필드는 사용자가 시스템에서 필요한 정보를 맞춤형으로 추가하고 관리할 수 있는 기능입니다. 이 기능을 통해 특정 데이터 항목을 생성하고, 업무 프로세스나 보고서에 맞게 활용할 수 있습니다.

<ImgLang img='url-pro/add-custom-field.png' desc='커스텀 필드 추가' />

### **커스텀 필드 추가**

1. **커스텀 필드 관리**의  **+ 커스텀 필드 추가** 버튼을 클릭하세요.
2. **커스텀 필드명**을 입력하세요.
3. **추가** 버튼을 클릭하세요.

### 커스텀 필드 수정

1. 수정할 필드의 ![csv 아이콘](/img/ico-edit.svg) **수정** 아이콘을 클릭하세요.
2. **커스텀 필드 수정** 창에서 **커스텀 필드명**을 수정하세요.
3. **수정** 버튼을 클릭하세요.

### 커스텀 필드 삭제

**개별 삭제**

1. 삭제할 커스텀 필드의 ![csv 아이콘](/img/ico-trash.svg) **삭제** 아이콘을 클릭하세요.
2. 삭제 확인 팝업창에서 **삭제** 버튼을 클릭하세요.

**일괄 삭제**

1. 커스텀 필드 목록에서 삭제할 항목들의 체크박스를 선택하세요.
2. 화면 오른쪽 위의 **삭제** 버튼을 클릭하세요.
3. 삭제 확인 팝업창에서 **삭제** 버튼을 클릭하세요.

:::caution

적용된 URL이 있어도 삭제 가능하며, 삭제 시 URL에서 해당 이슈 정보가 제거됩니다.

:::
