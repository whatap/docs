와탭은 세 가지 유형의 대시보드를 제공합니다. 각 대시보드는 실시간 URL 상태를 다양한 기준(카테고리, 이슈, 장애 상태)으로 시각화하여 제공합니다.

## 카테고리 현황 대시보드

- 카테고리별 전체/정상/비정상 URL 개수를 차트와 함께 제공합니다.
- 카테고리를 클릭하면 해당 카테고리에 속한 URL의 상세 목록을 확인할 수 있습니다.

이 대시보드는 URL 카테고리별 실시간 데이터를 조회할 수 있습니다. 사용자 입장에서 카테고리 단위의 웹 사이트의 정상 접근 여부를 브라우저 등을 통해 점검하는 운영 업무를 대신하여, 와탭의 URL 모니터링 에이전트가 해당 사이트에 대한 접속을 수행하고 접속의 정상 여부를 제공합니다.

### **카테고리 상태 현황**

카테고리에 등록된 URL의 상태 정보를 제공합니다. 
카테고리 이름, 전체 URL 수, 정상 URL 수, 비정상 URL 수를 건수와 차트로 표시합니다. 
이를 통해 카테고리별 전반적인 성능을 파악할 수 있습니다.

카테고리 위젯을 클릭하면 카테고리 상세가 표시됩니다.

### **카테고리 상세**

카테고리에 등록된 URL별 상태 현황 목록을 표시합니다. 상태, 페이지 명, 메시지, 추이에 대한 정보를 확인할 수 있습니다. 목록을 클릭하면 해당하는 URL에 대한 상세정보가 표시됩니다.

## 이슈 현황 대시보드

- 이슈별로 URL 상태를 확인할 수 있습니다.
- 각 이슈 항목을 클릭하면 상세 URL 목록을 확인할 수 있습니다.

## 장애 현황 대시보드

- **실시간 장애 발생 URL**: 장애 조건에 부합하지만 횟수 미달인 URL
- **장애 상태 지속 URL**: 장애 발생 조건을 충족해 지속적으로 장애 상태인 URL
- 사용자 확인 후 '정상 처리'할 수 있으며, 이후 동일 조건에서는 알림이 발생하지 않습니다.
