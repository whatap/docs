## 장애 판정 및 처리

장애 여부는 URL 응답 결과에 따라 자동으로 판정되며, 사용자에 의해 수동 처리가 가능합니다.

### 장애 단계 정의

1. **비정상(abnormal)**: 조건에 부합했지만 발생 횟수 미달 (경고, 주황색)
2. **장애(error)**: 조건 및 횟수 모두 충족 (장애 상태, 빨간색)

### 정상 처리 기능

- 일부 장애 유형(타이틀 변경, 최종 URL 변경)은 사용자 판단에 따라 정상으로 처리할 수 있습니다.
- '정상 처리' 시 해당 항목은 **허용 목록**에 등록되며, 이후 동일 조건에서는 장애로 감지되지 않습니다.

### 허용 목록 확인

- 허용된 URL 또는 타이틀 변경 내역은 URL 상세 페이지에서 확인할 수 있습니다.
