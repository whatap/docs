여러 항목에 동일한 **이슈**를 추가해 관련 항목을 그룹화할 수 있습니다. 대량의 데이터를 다룰 때, 중요한 특징이나 정보을 강조하고 빠르게 식별할 수 있도록 도와줍니다.

이슈 목록에서 항목을 클릭하면 해당 이슈가 적용된 **URL 목록**이 필터링된 상태로 표시됩니다.

<ImgLang img='url-pro/issue-management.png' desc='이슈 관리' />

### 이슈 추가

<ImgLang img='url-pro/add-issue.png' desc='이슈 추가' />

1. **+ 이슈 추가** 버튼을 클릭하세요.
2. 이슈 정보 입력하세요.
    - **이슈명**(필수): 중복 불가
    - **이슈 기간 설정**(필수): 이슈에 대한 기간 설정 여부 선택
    - **시작 시각**(필수), **종료 시각**(선택): 시작 시각만 설정하거나, 시작과 종료 시각을 모두 설정 가능
        - 기간 미설정: 상시 이슈로 관리
        - 시작일만 설정: 진행 중인 이슈
        - 시작일/종료일 모두 설정: 기간이 정해진 이슈
    - **적용할 카테고리**(선택)
    - **적용할 URL**(선택): 이슈 생성과 동시에 등록된 URL에 적용 가능
3. **추가** 버튼을 클릭하세요.

### 이슈 수정

1. 수정할 이슈의 ![csv 아이콘](/img/ico-edit.svg) **수정** 아이콘을 클릭하세요. 
2. **이슈 수정** 창에서 이슈명, 이슈 기간 설정, 시작 시각, 종료 시간(선택)을 수정하세요. 
3. **수정** 버튼을 클릭하세요.

### 이슈 삭제

**개별 삭제**

1. 삭제할 이슈의 휴지통 아이콘을 클릭하세요.
2. 삭제 확인 팝업창에서 ![csv 아이콘](/img/ico-trash.svg) **삭제** 버튼을 클릭하세요.

**일괄 삭제**

1. 이슈 목록에서 삭제할 항목들의 체크박스를 선택하세요.
2. 화면 오른쪽 위의 **삭제** 버튼을 클릭하세요. 
3. 삭제 확인 팝업창에서 **삭제** 버튼을 클릭하세요.


:::caution

적용된 URL이 있어도 삭제 가능하며, 삭제 시 URL에서 해당 이슈 정보가 제거됩니다.

:::
