이슈는 다수의 URL에 동일하게 적용되는 특이사항(장애, 캠페인 등)을 기록할 수 있는 기능입니다. 
이슈를 설정하면 관련 URL을 필터링하거나 대시보드에서 집중 관리할 수 있습니다.

### 이슈 추가

1. **이슈 추가** 버튼 클릭합니다.
1. 이슈 정보를 입력합니다. 

    - **이슈명** 입력 (중복 불가)
    - **이슈 기간 설정** 여부 선택
    - 시작일/종료일 입력 (필요 시)
    - 적용할 URL 선택 (선택 사항)

6. **추가** 버튼 클릭

### 이슈 수정

- 연필 아이콘 클릭 후 항목 수정

### 이슈 삭제

**개별 삭제**

1. 삭제할 이슈의 **삭제** 버튼(휴지통 아이콘) 클릭
2. 삭제 확인 팝업에서 **확인** 클릭

**일괄 삭제**

1. 이슈 목록에서 삭제할 항목들을 체크박스로 선택
2. 상단 옵션바의 **삭제** 버튼 클릭
3. 삭제 확인 팝업에서 **확인** 클릭

::: note

적용된 URL이 있어도 삭제 가능하며, 삭제 시 URL에서 해당 이슈 정보가 제거됩니다.

:::
