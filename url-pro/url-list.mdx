## URL 목록 관리

생성된 프로젝트에서 URL을 등록하고 관리할 수 있습니다.

### URL 등록

상단의 **URL 등록하기** 버튼을 클릭하여 URL을 추가합니다.

**필수 입력 항목**

- URL : 분석 대상 URL을 입력합니다.
- 페이지명 :  분석 대상 URL의 페이지명을 입력합니다. 쉽게 구분할 수 있는 이름을 추천합니다.
- 확인 지역 : 접속 테스트를 위한 추가 지역(리전)을 제공합니다. 해당 지역은 URL 체크 에이전트가 설치된 지역을 선택 할 수 있으며, 설치된 지역이 한곳일경우 선택 기능은 비활성화 되며, 해당 기능은 상황에 따라 제공 되지 않을 수 있습니다.
- 모니터링 주기 : 모니터링 주기를 선택합니다. 선택한 주기에 따라 URL 상태를 점검합니다.

**선택 입력 항목**

- 카테고리 : 분석 대상 URL의 카테고리를 설정합니다. 설정하지 않을 경우 미분류로 설정됩니다.
- 이슈 : 분석 대상 URL의 이슈를 설정합니다.
- 커스텀 필드 : 사용자 정의 추가 정보

### URL 수정

URL 목록에서 개별 URL을 수정할 수 있습니다.

1. URL 목록에서 수정할 항목의 **수정** 버튼(연필 아이콘) 클릭
2. 수정 가능 항목:
    - URL
    - 페이지명
    - 확인 지역
    - 모니터링 주기
    - 상태
    - 카테고리
    - 이슈
    - 커스텀 필드
3. **저장** 버튼 클릭하여 변경사항 적용

**일괄 수정**

여러 URL을 한 번에 수정할 수 있습니다.

1. URL 목록에서 수정할 항목들을 체크박스로 선택
2. 상단 옵션바의 **수정** 버튼 클릭
3. 수정 대상 선택
    - 모니터링 주기
    - 카테고리
    - 이슈
    - 상태 변경
4. 수정 내용 입력 후 **적용** 버튼 클릭하여 선택한 URL에 일괄 적용

:::note

**상태 유형**

- **활성화** : 해당 URL에 대한 모니터링이 활성화된 상태
- **중지** : 해당 URL에 대한 모니터링을 사용자 변경 전까지 무기한 정지
- **일시정지** : 해당 URL에 대한 모니터링을 설정된 기간까지 중지하며, 해당 기간 후 활성화 상태로 전환
- **숨김** : 대시보드에서 해당 URL을 나타내거나 집계하지 않음

:::

### URL 삭제

**개별 삭제**

1. URL 목록에서 삭제할 항목의 **삭제** 버튼(휴지통 아이콘) 클릭
2. 삭제 확인 팝업에서 **확인** 클릭

**일괄 삭제**

1. URL 목록에서 삭제할 항목들을 체크박스로 선택
2. 상단 옵션바의 **삭제** 버튼 클릭
3. 삭제 확인 팝업에서 **확인** 클릭

### URL 복사

기존 URL 설정을 복사하여 새 URL을 빠르게 등록할 수 있습니다.

1. URL 목록에서 복사할 항목의 **복사** 버튼(복사 아이콘) 클릭
2. URL 복사 팝업에서:
    - URL 주소와 페이지명을 새로 입력
    - 나머지 설정(카테고리, 이슈, 모니터링 주기 등)은 기존 값 유지
3. 필요 시 설정 수정 후 **저장** 클릭

> 동일한 페이지명을 사용할 수 없습니다.

### Excel 다운로드

현재 URL 목록을 Excel 파일로 다운로드할 수 있습니다.

1. URL 목록 관리 화면 우측 상단의 **Excel 다운로드** 버튼 클릭
2. 파일명 형식: `URL_목록_yyyyMMdd-HH-mm-ss.xlsx`
3. 현재 적용된 필터 조건이 반영된 목록 다운로드

### Excel 업로드

Excel 파일을 통해 URL을 일괄 등록하거나 수정할 수 있습니다.

1. URL 목록 관리 화면 우측 상단의 **Excel 업로드** 버튼 클릭
2. 다운로드한 Excel 파일과 동일한 형식의 파일 선택
3. 업로드 후 유효성 검사 진행
4. 기존 URL과 중복되는 경우 정보 업데이트

### 필터링 및 정렬

**상태별 조회**

왼쪽 상단의 드롭다운 메뉴를 통해 URL 목록을 상태별로 조회

- 전체
- 활성화
- 중지
- 일시정지
- 숨김

**정렬 기능**

URL 목록을 다양한 기준으로 정렬

- 페이지명순
- 생성일자순
- 수정일자순

**검색 및 필터**

필터 추가 버튼을 통해 복합 조건 검색

- URL: 포함 / 미포함
- 페이지명: 포함 / 미포함
- 카테고리: 포함 / 미포함 / 일치 / 불일치
- 이슈: 포함 / 미포함 / 일치 / 불일치

모든 필터 조건은 AND 조건으로 적용됩니다.

## 카테고리 관리

카테고리는 URL을 논리적으로 그룹화하여 분류하고 필터링에 활용할 수 있는 기능입니다. 
카테고리를 등록하면 URL 목록 관리 화면에서 해당 카테고리 기준으로 필터링하거나 대시보드에서 상태를 집계할 수 있습니다.

### 카테고리 추가

1. **카테고리 추가** 버튼 클릭
2. **카테고리명** 입력 (동일 레벨 중복 불가, 최대 50자)
3. **적용할 URL 선택** (선택 사항)
4. **추가** 버튼 클릭

### 카테고리 수정

- 연필 아이콘 클릭 후 카테고리명을 변경
- **수정** 버튼 클릭

### 카테고리 삭제

- 휴지통 아이콘 클릭 또는 다중 선택 후 **삭제**
- 삭제 시 해당 URL의 카테고리는 '미분류'로 변경됨

> ⚠️ 삭제된 카테고리는 복구할 수 없습니다.

## 이슈 관리

이슈는 다수의 URL에 동일하게 적용되는 특이사항(장애, 캠페인 등)을 기록할 수 있는 기능입니다. 
이슈를 설정하면 관련 URL을 필터링하거나 대시보드에서 집중 관리할 수 있습니다.

### 이슈 추가

1. **이슈 추가** 버튼 클릭합니다.
1. 이슈 정보를 입력합니다. 

    - **이슈명** 입력 (중복 불가)
    - **이슈 기간 설정** 여부 선택
    - 시작일/종료일 입력 (필요 시)
    - 적용할 URL 선택 (선택 사항)

6. **추가** 버튼 클릭

### 이슈 수정

- 연필 아이콘 클릭 후 항목 수정

### 이슈 삭제

**개별 삭제**

1. 삭제할 이슈의 **삭제** 버튼(휴지통 아이콘) 클릭
2. 삭제 확인 팝업에서 **확인** 클릭

**일괄 삭제**

1. 이슈 목록에서 삭제할 항목들을 체크박스로 선택
2. 상단 옵션바의 **삭제** 버튼 클릭
3. 삭제 확인 팝업에서 **확인** 클릭

:::note

적용된 URL이 있어도 삭제 가능하며, 삭제 시 URL에서 해당 이슈 정보가 제거됩니다.

:::

## 커스텀 필드 관리

커스텀 필드는 조직 내 특정 정보를 URL에 추가하여 관리할 수 있도록 지원하는 기능입니다.

### 커스텀 필드 추가

1. **커스텀 필드 추가** 버튼 클릭
2. **필드명** 입력
3. **추가** 버튼 클릭

### 커스텀 필드 수정

- 연필 아이콘 클릭 후 필드명 수정
- **수정** 버튼 클릭

### 커스텀 필드 삭제

- 휴지통 아이콘 클릭 또는 다중 선택 후 **삭제**
- 해당 필드를 사용하는 URL의 커스텀 정보는 제거됨
