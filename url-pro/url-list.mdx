## URL 목록 관리

URL 목록 관리 화면을 통해 URL을 추가, 수정, 삭제할 수 있습니다. 각 URL의 페이지명, 상태, 카테고리, 이슈, 모니터링 주기, 커스텀 필드 등을 설정할 수 있습니다.

1. URL 프로젝트를 선택하세요. 
2. **관리 > URL 목록 관리**로 이동하세요.

<ImgLang img='url-pro/urlpro-list.png' desc='url 목록' />

### URL 등록

화면 오른쪽 위의 **+ URL 등록하기** 버튼을 클릭하여 새로운 URL을 등록할 수 있습니다. URL 등록에서 필수 항목을 입력해 URL를 등록하세요. 

<ImgLang img='url-pro/add-url.png' desc='url 등록' />

| 입력 여부 | 항목 | 설명 |
| --- | --- | --- |
| 필수 | URL | 분석 대상 URL을 입력합니다. |
|  | 페이지명 | 모니터링할 URL의 페이지명을 입력합니다.<br />- 쉽게 구분할 수 있는 이름으로 작성하는 것을 권장합니다. |
|  | 확인 지역 | 접속 테스트를 수행할 추가 지역(리전)을 제공합니다.<br />- URL 체크 에이전트가 설치된 지역만 선택할 수 있습니다.<br />- 설치 지역이 1곳뿐이면 선택 기능이 비활성화되며, 상황에 따라 제공되지 않을 수 있습니다. |
|  | 모니터링 주기 | URL 상태 점검 주기를 선택합니다. |
| 선택 | 카테고리 | 분석 대상 URL의 카테고리를 설정합니다.<br />- 미설정 시 ‘미분류’로 지정됩니다. |
|  | 이슈 | 분석 대상 URL의 이슈를 설정합니다. |
|  | 커스텀 필드 | 사용자 정의 추가 정보를 입력합니다. |

### URL 수정

**개별 수정**

개별 URL을 수정할 수 있습니다.

1. **URL 목록**에서 수정할 항목의 ![csv 아이콘](/img/ico-edit.svg) **수정** 아이콘을 클릭하세요. 
2. URL 수정에서 원하는 항목(URL, 페이지명, 모니터링 주기, 상태, 카테고리, 이슈, 커스텀 필드) 을 수정하세요. 
3. **저장** 버튼 클릭하여 변경사항 적용합니다.

**일괄 수정**

여러 URL을 한 번에 수정할 수 있습니다.

**일괄 수정**

여러 URL을 한 번에 수정할 수 있습니다.

1. URL 목록에서 수정할 항목들을 체크박스로 선택하세요.
2. 상단 옵션바의 ![csv 아이콘](/img/ico-edit.svg) **수정** 버튼을 클릭하세요. 
3. **수정 대상**의 드롭다운 항목에서 수정 대상(카테고리, 이슈, 상태, 모니터링 주기)을 선택해 수정하세요.

:::note

**상태 유형**

- **활성화** : 해당 URL에 대한 모니터링이 활성화된 상태
- **중지** : 해당 URL에 대한 모니터링을 사용자 변경 전까지 무기한 정지
- **일시정지** : 해당 URL에 대한 모니터링을 설정된 기간까지 중지하며, 해당 기간 후 활성화 상태로 전환
- **숨김** : 대시보드에서 해당 URL을 나타내거나 집계하지 않음

:::

4. 수정 내용 입력 후 적용 버튼 클릭하여 선택한 URL에 일괄 적용하세요.

### URL 삭제

:::caution

URL 삭제 시 해당 URL에 대한 데이터는 모두 삭제됩니다.

:::


**개별 삭제**

1. URL 목록에서 삭제할 항목의 ![csv 아이콘](/img/ico-trash.svg) **삭제** 아이콘을 클릭하세요. 
2. 삭제 확인 팝업창에서 **삭제** 버튼을 클릭해 삭제합니다. 


**일괄 삭제**

1. URL 목록에서 삭제할 항목들을 체크박스로 선택하세요. 
2. 상단 옵션바의 **삭제** 버튼 클릭하세요. 
3. 삭제 확인 팝업창에서 **삭제** 버튼을 클릭해 삭제합니다.

### URL 복사

기존 URL을 복사해 동일한 설정으로 새 URL을 빠르게 등록할 수 있습니다.

:::caution

동일한 페이지명을 사용할 수 없습니다.

:::

1. URL 목록에서 복사할 항목의 ![csv 아이콘](/img/ico-copy.svg) **복사** 아이콘을 클릭하세요. 
2. **URL 복사**에서 URL 주소와 페이지명을 변경하세요. 
    - 나머지 설정(카테고리, 이슈, 모니터링 주기 등)은 기존 값이 유지됩니다.
3. 필요 시, 나머지 설정 수정 후 **저장** 버튼을 클릭하세요.

### URL 조회 및 정렬

#### 상태별 조회

URL 목록 상단의 드롭다운 메뉴를 통해 URL 목록을 상태별로 조회할 수 있습니다. 

    - **전체**
    - **활성화**: 해당 URL에 대한 모니터링이 활성화된 상태
    - **중지**: 해당 URL에 대한 모니터링을 사용자 변경 전까지 무기한 정지
    - **일시정지**: 해당 URL에 대한 모니터링을 설정된 기간까지 중지하며, 해당 기간 후 활성화 상태로 전환
    - **숨김**: 대시보드에서 해당 URL을 나타내거나 집계하지 않음


#### 정렬 기능 

URL 목록을 페이지명, 생성일자, 수정일자 기준으로 정렬합니다.

#### 검색 및 필터

필터를 추가해 복합 조건으로 검색할 수 있습니다.

1. **필터 추가하기**에서 필터 키와 조건을 선택해, 필터를 입력하세요. 
    - 모든 필터 조건은 AND 조건으로 적용됩니다.
    - 지원 필터 키
        
        | 필터 키 종류 | 조건 |
        | --- | --- |
        | URL | 포함 / 미포함 |
        | 페이지명 | 포함 / 미포함 |
        | 카테고리 | 포함 / 미포함 / 일치 / 불일치 |
        | 이슈 | 포함 / 미포함 / 일치 / 불일치 |

2. **적용** 버튼을 클릭해 필터를 적용합니다.


### Excel 다운로드

현재 URL 목록을 Excel 파일로 다운로드할 수 있습니다.

1. URL 목록 관리 화면 오른쪽 위의 **엑셀 다운로드** 버튼을 클릭하세요.
2. 현재 적용된 필터 조건이 반영된 목록이 다운로드 됩니다.
    * 파일명 형식: `URL_목록_yyyyMMdd-HH-mm-ss.xlsx`

### Excel 업로드

Excel 파일로 URL을 일괄 등록하거나 수정할 수 있습니다.

1. URL 목록 관리 화면 오른쪽 위의 **Excel 업로드** 버튼을 클릭하세요.
2. 다운로드한 Excel 파일과 동일한 형식의 파일 선택해 업로드하세요.
    - 기존 URL과 중복되는 정보는 업데이트됩니다.



## 커스텀 필드 관리

커스텀 필드는 조직 내 특정 정보를 URL에 추가하여 관리할 수 있도록 지원하는 기능입니다.

### 커스텀 필드 추가

1. **커스텀 필드 추가** 버튼 클릭
2. **필드명** 입력
3. **추가** 버튼 클릭

### 커스텀 필드 수정

- 연필 아이콘 클릭 후 필드명 수정
- **수정** 버튼 클릭

### 커스텀 필드 삭제

- 휴지통 아이콘 클릭 또는 다중 선택 후 **삭제**
- 해당 필드를 사용하는 URL의 커스텀 정보는 제거됨
