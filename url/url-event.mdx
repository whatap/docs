홈 화면 &gt; 프로젝트 선택 &gt; <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventSetting" className="uitext" />

URL 모니터링은 이벤트 설정을 통해 이메일 형태의 경고 알림을 제공받을 수 있습니다. 모니터링 중인 프로젝트 화면에서는 실시간으로 발생하는 이벤트에 대해서도 알림을 제공합니다. 등록된 URL을 대상으로 다음과 같이 개별 이벤트 설정을 할 수 있습니다. 

1. 이벤트 목록 화면에서 편집하고자 하는 URL의 ![수정 아이콘](/img/ico-edit.svg) <Cmdname sid="edit" className="uitext" /> 아이콘을 선택하세요.

    <ImgLang img='url-event.png' desc='이벤트 목록 화면' />

1. 해당 URL에 대한 <Cmdname sid="edit_alert_setting" className="uitext" /> 창을 확인하세요. 

    <ImgLang img='url-event-mod.png' desc='url 이벤트 설정 창' />

    <Cmdname sid="status" className="uitext" />, <Cmdname sid="elapsed" className="uitext" /> 등 이벤트 규칙은 모니터링 지표와 조건식을 조합하여 설정합니다. 

    * **상태 이벤트**

      상태 이벤트는 선택한 <Cmdname sid="status" className="uitext" /> 코드 옵션에 따라 이벤트가 발생합니다.
    
      <ImgLang img='alert_status.png' desc='상태 이벤트' />

    * **경과 시간 이벤트**

      경과 시간 이벤트는 실시간 수집 중인 <Cmdname sid="elapsed" className="uitext" /> 값이 사용자가 입력한 임계치와 조건문에 해당하는 경우 이벤트가 발생합니다.
    
      <ImgLang img='alert_elapsed.png' desc='경과 시간 이벤트' />

:::info

**알림 메시지란?**

해당 규칙에서 이벤트가 발생하는 경우 표시할 제목을 말합니다. 개별 이벤트 규칙에 알림 메시지를 설정할 수 있습니다. 설정을 적용하지 않는 경우 기본 표시되는 알림 메시지는 **상태 코드**, **경과 시간**, **URL: 상세 이유**입니다.

경고 알림과 관련해 모니터링 플랫폼별 지원되는 이벤트 종류를 확인하려면 [다음 문서](../support-env#support-notice)를 참조하세요.

:::

:::note

수정한 URL이 목록에 반영되기까지 약 1분의 시간이 소요됩니다.

:::
