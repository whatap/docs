와탭 모니터링 서비스를 사용하기 위해서는 [회원 가입](https://www.whatap.io/ko/signup/) 후 프로젝트를 생성하고 해당 URL을 추가해야 합니다. 회원 가입에 관한 자세한 내용은 [다음 문서](../account/account-manage)를 참조하세요.

## 프로젝트 생성

URL을 추가하기 전에 먼저 프로젝트를 생성하세요.

1. [와탭 모니터링 서비스](https://service.whatap.io)로 이동한 다음 로그인하세요.

1. 프로젝트를 생성하려면 화면 왼쪽 사이드 메뉴에서 <Cmdname sid="BTN07332" className="uitext" /> &gt; ***+*** <Cmdname sid='BTN06696' className='uitext'/> 버튼을 선택하세요.

1. <Cmdname sid="TTL07073" className="uitext" /> 화면에서 URL 모니터링 프로젝트를 생성하세요.

1. <Cmdname sid="project_name" className="uitext" />, <Cmdname sid="data_server_region" className="uitext" />, <Cmdname sid="project_time_zone" className="uitext" /> 항목을 차례로 설정하세요.

    <ImgLang img='create-project-v3-url.png' desc='url 모니터링 프로젝트' /> 
    
    {/* url-project.png */}

1. <Cmdname sid="TTL07727" className="uitext" />에서 경고 알림 메시지의 언어를 선택하세요.

1. 모든 설정을 완료한 다음에는 <Cmdname sid="creating_a_project" className="uitext" /> 버튼을 선택하세요.

1. 프로젝트 목록에서 신규 프로젝트를 확인하세요.

:::note

* <Cmdname sid="data_server_region" className="uitext" />은 리전(클라우드 서비스를 제공하기 위해 설치한 데이터 센터의 묶음)을 의미합니다. 특정 리전을 선택하면 해당 리전에 속한 데이터 센터에 사용자의 데이터를 저장합니다.

* <Cmdname sid="project_time_zone" className="uitext" />은 알림, 보고서를 생성하는 기준 시간입니다.

* 여러 개의 프로젝트를 그룹으로 묶어 관리하려면 <Cmdname sid="project_groups" className="uitext" />에서 그룹을 선택하거나 그룹을 추가하세요. 그룹에 대한 자세한 설명은 [다음 문서](../project/group)를 참조하세요.

:::

:::caution

URL 프로젝트는 최대 10개까지 생성할 수 있습니다. 한 개의 URL 프로젝트에서 등록할 수 있는 URL은 무제한 무료입니다.

:::

## URL 추가하기{#add-url}

1. 생성된 URL 프로젝트를 선택하세요. URL이 설정되지 않은 화면이 표시됩니다.

1. 화면 오른쪽 위에 ***+*** <Cmdname sid="add_url" className="uitext" /> 버튼을 선택하세요. 

1. 입력 필드에 URL 설정을 진행하세요.

    <ImgLang img='url-input-field.png' desc='url 입력 필드' />

    * <Cmdname sid="name" className="uitext" />: URL 설정의 고유 식별 정보로 사용됩니다. 중복 사용할 수 없습니다. 재설정 시 동일한 이름으로 등록할 경우 동일한 이름의 통계 데이터를 같은 URL의 데이터로 간주합니다.

    * <Cmdname sid="url" className="uitext" />: 체크 대상 URL을 등록합니다. 호출 대상 URL의 전체 경로를 입력합니다. http/https 프로토콜을 지원합니다.
    
    * <Cmdname sid="check_location" className="uitext" />: 와탭이 제공하는 에이전트를 선택합니다. Seoul, Tokyo, Singapore, Mumbai, Frankfurt, Califonia 리전을 제공합니다.

      :::note

      <Cmdname sid="TTL08547" className="uitext" />를 선택하면 해당 지역의 IP 주소를 확인할 수 있습니다. 방화벽 규칙을 확인하고 필요한 경우 해당 IP 주소를 허용하세요. 자세한 내용은 <Link to="#select-confirm-region">다음 문서</Link>를 참조하세요.

      :::

    * <Cmdname sid="collection_cycle" className="uitext" />: 모니터링 데이터를 수집하는 간격을 설정합니다. 기본값은 <Cmdname sid="1_min" className="uitext" />(60000 ms)입니다. 최대 5분까지 1분 단위로 선택할 수 있습니다.

1. 설정을 완료하면 <Cmdname sid="save" className="uitext" /> 버튼을 선택하세요.

:::note

* 추가/수정/복사한 URL이 목록에 반영되기까지 약 1분의 시간이 소요됩니다.

* <Cmdname sid="event_setting" className="uitext" />에 대한 자세한 내용은 <Link to='url-event'>다음 문서</Link>를 참조하세요.

:::

### 확인 지역 선택{#select-confirm-region}

향후 접속 테스트를 위한 추가 지역(리전)을 제공할 수 있습니다. 추가 지역에 대해서는 support@whatap.io로 문의하세요.

<ImgLang img="url-check-region-ip.png" desc="리전 IP" />

**폐쇄망 사이트의 접속 테스트를 수행**하고자 하는 경우 에이전트로부터의 호출이 허용되도록 방화벽을 오픈해야 할 수 있습니다. <Cmdname sid="TTL08547" className="uitext" />을 클릭하면 위치 IP 주소가 표시됩니다. 각 에이전트의 **방화벽 오픈 대상 IP Address**는 다음과 같습니다.

* AWS-Frankfurt-Agent - 18.197.22.141

**cloudflare**를 사용하는 경우 cloudflare **방화벽 규칙에 해당하는 리전 IP**를 허용해야 합니다.

### 조건 지정 옵션

홈 화면에서 프로젝트를 선택하고 <Cmdname sid="url_list" className="uitext" /> 메뉴로 이동하세요. ![number-1](/img/number-01.png) 영역 상단 옵션을 통해 보기 형식 및 정렬 방식을 지정하거나 검색을 통해 목록을 조회할 수 있습니다.

* ![리스트 뷰 아이콘](/img/ic-list.svg) <Cmdname sid="list_view" className="uitext" />: <Cmdname sid="url_list" className="uitext" />을 리스트 형식으로 볼 수 있습니다. 기본 설정은 <Cmdname sid="list_view" className="uitext" />입니다. 

* ![카드 뷰 아이콘](/img/ic-project.svg) <Cmdname sid="card_view" className="uitext" />: <Cmdname sid="url_list" className="uitext" />을 카드 형식으로 볼 수 있습니다.

* <Cmdname sid="search" className="uitext" />: 입력창을 통해 URL 혹은 지정한 URL의 이름을 기준으로 검색할 수 있습니다.

![number-2](/img/number-02.png) 영역에서 등록된 URL 목록의 최종 응답 스테이터스 별 건수 정보를 확인하고 필터링 할 수 있습니다. <Cmdname sid="status_total" className="uitext" /> 보기, ***9XX***, <Cmdname sid="status_5xx" className="uitext" />, <Cmdname sid="status_4xx" className="uitext" />, <Cmdname sid="status_3xx" className="uitext" />, <Cmdname sid="status_2xx" className="uitext" /> 등을 기준으로 조회할 수 있습니다. 기본 설정은 <Cmdname sid="status_total" className="uitext" /> 보기 입니다.

### URL 목록 내보내기, 가져오기

![number-3](/img/number-03.png) 영역에서 현재 <Cmdname sid="url_list" className="uitext" />을 엑셀 파일 형식으로 내보내거나 가져올 수 있습니다. 

* ![가져오기 아이콘](/img/ico-import.svg) <Cmdname sid="import" className="uitext" />: 엑셀 파일을 선택해 URL 목록을 불러옵니다. 

* ![내보내기 아이콘](/img/ico-export.svg) <Cmdname sid="export" className="uitext" />: 웹 브라우저에서 현재 URL 목록을 엑셀 파일로 다운로드합니다.

### URL 모니터링 상세

![number-4](/img/number-04.png) 영역 URL 목록 테이블에서 등록된 URL 단위의 모니터링 체크 결과를 확인할 수 있습니다. 각 컬럼 항목은 다음과 같습니다. URL의 등록 및 편집이 필요한 경우 ![편집 아이콘](/img/ic-edit.svg) <Cmdname sid="edit_url" className="uitext" /> 아이콘을 선택해 편집 화면으로 전환하세요.

* <Cmdname sid="current_status" className="uitext" />

  최종 체크 응답 코드(Http Status Code)입니다. 각 상태 코드를 선택하면 HTTP 상태 코드 정리 문서로 이동합니다.

* <Cmdname sid="name_and_url" className="uitext" />

  URL 등록 시에 지정한 고유 식별 정보와 URL 입니다.

* <Cmdname sid="check_region" className="uitext" />

  확인 지역입니다.

* <Cmdname sid="elapsed" className="uitext" />

  20분간의 체크 결과(응답 코드 및 응답 시간) 및 최종 응답 시간입니다. 차트 영역에서 드래그해 시간 범위 조회 시 다음과 같이 해당 시간대의 응답 코드 및 응답 시간을 확인할 수 있습니다.

  <ImgLang img='url-monitoring-dt.png' desc='상세' />

:::note 

**와탭 URL Monitoring 서비스 사용**

URL 모니터링은 별도의 정책 변경이 있기까지 무상 서비스로 제공합니다. 사용자는 별도의 에이전트를 설치할 필요가 없으며 와탭이 제공하는 에이전트 및 서버 저장소를 활용합니다.

:::
