description: Win32 API로 성능 지수 수집하는 방법을 안내합니다.
tags: [ Server, 설정하기 ]
draft: true
---

CPU, MEMORY, DISK, TRAFFIC, PROCESS 성능 정보를 수집할 때 Win32 API를 사용합니다. 윈도우 성능 카운터가 꺼져있거나 사용 불가능한 환경에서 성능 수집 시 사용할 수 있습니다.

와탭 서버 모니터링 설정 파일에 `native_api.enabled=true` 옵션을 추가합니다. 파워셸에서 다음과 같이 적용하세요.

```bash title='Windows Powershell'
Add-Content -Path "C:\Program Files\WhatapInfra\whatap.conf" -Value "native_api.enabled=true"
Restart-Service "Whatap Infra"
```
